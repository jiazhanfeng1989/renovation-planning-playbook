# 01 施工工艺总表

## 1. 先卡死的底线
- 材料品牌、系列、型号、规格、厚度、批次与合同不符：`停工，暂停付款`。
- 隐蔽工程未验收就封槽、封板、回填、贴砖：`停工`。
- 图纸未签字就下单定制、下门、下设备：`暂停`。
- 测试未做或留证不足：`不放行`。
- 整改未闭环：`不付款`。

## 2. 半包合同必须写清
- 半包范围必须拆开写：拆改、水电、防水、瓦工、吊顶、油工、安装收口、成品保护、垃圾清运、现场管理。
- 水电点位、回路、米数、开槽、封槽、找平、防水增刷、海棠角、包管、检修口是否另算必须写清。
- 所有辅材必须写清品牌型号：水管、电线、网线、防水、瓷砖胶、背胶、石膏板、腻子、乳胶漆、玻璃胶。
- 不接受：`同级材料、按现场实际、按公司标准、常规施工、后期再定`。

## 3. 水电工艺底线
### 3.1 水路
- 水管统一 `PPR S3.2 / PN2.0热水管`。
- 管件同品牌，不接受杂牌混用。
- 打压标准：`0.8MPa / 30分钟 / 压降不超过 0.05MPa`。
### 3.2 电路
- 电线用国标单股铜线。
- 照明不低于 `1.5平方`；普通插座不低于 `2.5平方`；厨房、卫浴、阳台大功率位不低于 `4平方`。
- 冰箱独立回路；蒸箱、烤箱、洗碗机、烘干机、洗衣机、空调按功率独立或分区回路。
- 强弱电分管分槽；线管内不得有接头；湿区插座带漏保和防溅。
### 3.3 弱电
- 网线统一 `Cat6 UTP 23AWG 无氧铜六类网线`。
- 书房、儿童房、电视墙必须有网口。
- 弱电箱预留光猫、路由、交换机散热和电源。

## 4. 防水、排水、防臭底线
- 卫生间地面满刷防水；淋浴区墙面不低于 `1.8米`；其他墙面不低于 `30cm`；台盆背墙不低于 `1.2米`。
- 阳台地面满刷防水，墙面上翻不低于 `30cm`。
- 管根、地漏、门口、阴阳角必须加强。
- 闭水 `48小时`，楼下同步检查。
- 卫生间地面必须坡向地漏，不得倒坡、积水。
- 台盆下水、地漏、排水口都要防臭。
- 下水管必须做隔音包管，并留检修口。

## 5. 书房新增一路地暖底线
- 必须单独报价，不接受只写“补一路地暖”。
- 管材：`PE-RT II型` 或 `PEX-a阻氧管`。
- 主选规格 `16x2.0mm`，整根管施工，中间不得有接头。
- 新增一路必须有独立阀门和回路标识，不得混接。
- 固定柜体下方不铺地暖管。
- 管间距 `150-200mm`，冷区可加密到 `150mm`。
- 保温板 `XPS` 厚度不低于 `20mm`，边界保温条不低于 `8mm`。
- 第一次打压 `0.8MPa / 30分钟`，再做 `24小时` 保压观察。
- 回填期间保持 `0.4-0.6MPa` 带压施工。
- 回填完成后二次复压，合格后才能贴砖。

## 6. 中央空调底线
- 每台内机、外机必须写完整型号，不接受只写品牌和系列。
- 铜管必须用 `脱氧无缝紫铜管`。
- `6.35 / 9.52 / 12.7` 管壁厚不低于 `0.8mm`。
- `15.88 / 19.05` 管壁厚不低于 `1.0mm`。
- 保温棉为 `B1级阻燃闭孔橡塑`，厚度不低于 `15mm`，易结露位 `20mm`。
- 冷凝水管为 `UPVC硬管`，单台内机不低于 `DN25`，坡度不低于 `1%`。
- 焊接必须 `氮气保护焊`。
- 每台内机必须留检修口。
- 保压：`4.15MPa / 24小时`。
- 抽真空：真空泵 + 真空表，整套多联机不低于 `60分钟`。
- 每台内机必须做通水测试和不少于 `30分钟` 试运行。

## 7. 封窗底线
- 型材：大阳台 `108/110系列`；普通窗 `75/80系列`。
- 原生铝：`6063-T5`，主受力型材壁厚不低于 `2.0mm`。
- 隔热条：`PA66-GF25`。
- 玻璃：全部钢化并带 `3C` 标识；重点位做 `夹胶中空 Low-E`。
- 五金、纱窗、限位、防坠件必须写清。
- 固定片、膨胀螺栓或连接件必须满足高层外窗要求，不能只靠发泡固定。
- 固定点距框角不大于 `150-200mm`，中间间距不大于 `500mm`。
- 室外耐候胶连续，排水孔不得堵死。
- 开启扇开合不少于 `10次`。
- 淋水不少于 `30分钟`，室内不得渗水。

## 8. 瓦工和瓷砖底线
- 客餐厅、过道、卧室、书房：统一 `750x1500mm` 通铺。
- 厨房墙砖 `400x800mm`；厨房地砖 `300x600mm` 防滑砖。
- 主卫、公卫墙砖 `400x800mm`；地砖 `300x600mm` 防滑砖。
- 卫生间统一 `普通点位地漏`，不做线性地漏。
- 卫生间外阳角做 `海棠角`。
- 卫生间墙地交界做 `墙压地 + 柔性防霉硅酮胶`。
- 铺贴前必须出排版图，显眼位置不能有小窄条。
- 大规格砖和低吸水率砖必须匹配 `瓷砖胶 + 背胶`。
- 厨卫阳台地面先保证坡度和排水，再谈效果。
- 地漏、门洞、窗边、包管、阳角、柜体边、卫生间门口收口必须提前交底。
- 空鼓、平整度、坡度不合格，不进美缝。

## 9. 吊顶和油工底线
- 石膏板接缝必须做防裂处理。
- 阴阳角找直，墙顶面平整。
- 乳胶漆按 `一底两面`。
- 吊顶封板前必须确认：检修口、阀门、风口、灯位、窗帘盒、柜体关系。
- 检修位被封死，不封板。

## 10. 厨房专项底线
- 水槽、灶具、洗碗机、蒸箱、烤箱、微波炉、净水器尺寸和位置必须先定。
- 备菜区必须保留连续台面，优先在水槽和灶具之间，长度按 `90-120cm+` 复核。
- 柜体：`ENF级防潮板 / ENF级多层板`，侧板、层板、门板不低于 `18mm`。
- 封边统一 `PUR封边`。
- 台面统一石英石，厚度 `20mm`，台下盆必须有加固件。
- 高频柜门用阻尼铰链，抽屉用隐藏阻尼滑轨，不用反弹器。
- 洗碗机、蒸箱、烤箱按大功率电器单独回路。
- 水槽柜内插座抬高，避开漏水风险。
- 燃气管不暗埋，不封死在墙体、吊顶、瓷砖或固定柜体内。
- 燃气阀门、燃气表、接头必须可见、可通风、可检修。
- 洗碗机、净水器、下水必须防臭、防倒灌。

## 11. 卫生间专项底线
- 两个卫生间都做干湿分离。
- 马桶统一普通虹吸，不做智能马桶。
- 每个马桶旁预留 `1套增压喷枪`。
- 地砖不做大砖切坡。
- 花洒统一恒温花洒。
- 浴室柜统一 `铝合金柜体 + PET哑光柜门`。
- 卫生间插座带防溅盒和漏电保护。
- 镜柜区预留吹风机、电动牙刷、剃须刀插座。
- 排风或浴霸排气路径通畅，止逆到位。

## 12. 阳台专项底线
- 客厅大阳台为主家政区，书房小阳台不做主家政区。
- 洗烘区必须预留进水、排水、地漏、插座和检修空间。
- 扫地机器人基站必须做自动上下水、防臭、防反水、防溢水。
- 家政柜不挡窗、不挡地漏、不挡检修口。
- 插座高于可能积水位置，并做防溅保护。
- 地面坡向地漏，不积水。

## 13. 全屋定制底线
- 合同必须写：板材品牌、系列、基材、环保等级、厚度、封边、五金型号、结构、颜色、安装验收。
- 干区统一 `ENF级板材 18mm`；厨房阳台柜体 `ENF级多层板 / ENF级防潮板 18mm`；背板 `9mm`。
- 浴室柜统一 `铝合金柜体`。
- 书桌桌板统一 `25mm`。
- 封边统一 `PUR封边`。
- 高频柜门、抽屉统一阻尼方案，不用反弹器。
- 高柜、吊柜、书柜、衣柜必须固定到墙体。
- 潮湿区、水槽柜、洗碗机位、扫地机器人位开孔边必须二次封闭。
- 插座、网口、上下水、阀门、检修口不能被柜体封死。

## 14. 室内门底线
- 卧室、儿童房、书房门扇厚度不低于 `45mm`。
- 必须明确门芯结构，不接受只写“静音门”“实木门”。
- 卫生间门优先 `铝镁合金门 + 钢化磨砂玻璃`。
- 合页、锁体、门吸品牌型号写清。
- 卧室、书房、儿童房优先配静音条和自动门底密封。
- 每樘门必须和地砖完成面、挡水、柜体、卫浴五金一起复核。

## 15. 付款卡点
- 定金不超过合同总价 `20%`。
- 水电材料进场、核对品牌型号后付款一笔。
- 水电、防水、空调、地暖等隐蔽工程验收合格后付款一笔。
- 瓦工、吊顶、油工、定制、门、专项设备分别按验收节点付款。
- 竣工验收、整改完成、资料交付后付主要尾款。
- 保留 `5%-10%` 尾款，入住前复查无渗漏、空鼓、开裂、异响再付。

## 16. 资料交付底线
- 隐蔽照片齐。
- 水路打压、防水闭水、地暖打压、空调保压抽真空、封窗淋水记录齐。
- 设备清单、辅材清单、调试记录、保修卡、说明书齐。
- 整改清单闭环后再结清尾款。
