# 中央空调
## 匹数配置
- 客餐厅：约 `3匹`。
- 主卧：约 `1.5匹`。
- 两个儿童房：各约 `1匹`。
- 书房：约 `1匹`。
- 全屋内机合计约 `7.5-8匹`。
- 外机按商家方案核算，优先保证夏季制冷和冬季辅助制热余量。
## 品牌对比
- 格力：国产品牌，性价比和售后覆盖较好，适合控制预算。
- 海尔：服务体系较完整，价格通常比较稳，适合重视售后。
- 大金/日立：口碑和系统成熟度较好，预算更高，适合追求稳定和舒适。
## 安装避坑
- 必须氮气保护焊。
- 铜管、保温棉、冷凝水管、信号线、电源线、风口、支架、减震垫必须写清品牌和规格。
- 必须氮气保压测试，不接受口头说“测过了”。
- 必须抽真空，不接受只排空气。
- 冷凝水管必须有坡度和通水测试，避免后期吊顶内漏水。
- 每台内机必须保留检修口。
- 出风口不能直吹床头、书桌和长期坐卧位置。
## 业主指定合同标准
### 1. 设备配置
- 采用家用中央空调多联机方案，不用普通挂机、柜机或单独风管机混搭冒充中央空调。
- 商家必须逐项列明内机、外机完整型号，不接受只写品牌和系列。
- 商家必须列明每台内机匹数、安装房间、安装位置、控制方式。
- 客餐厅按约 `3匹` 配置。
- 主卧按约 `1.5匹` 配置。
- 两个儿童房各按约 `1匹` 配置。
- 书房按约 `1匹` 配置。
- 全屋内机合计按约 `7.5-8匹` 配置。
- 外机匹数和型号由商家按系统方案配置，但必须写清完整型号和匹数。
- 不允许用库存机、串货机、旧型号替代合同型号。
- 内机、外机必须提供原厂条码、合格证和保修资料。
### 2. 报价范围
- 报价必须拆分设备费、安装费、辅材费、风口费、检修口费、打孔费、吊装费。
- 标配铜管长度、超出单价必须写清。
- 支架、减震垫、冷媒追加、线控器、排水管、信号线的费用和责任必须写清。
- 所有增项必须先报价，经业主书面确认后施工。
- 合同禁止使用“按实际结算但不写单价”的条款。
- 报价必须列明每台内机到外机的预估管长、允许管长、内外机高度差和追加冷媒单价。
- 分歧管、分歧箱、冷媒追加、外机基础、外机减震、外机检修空间必须单独列明。
- 商家必须提供厂家配置方案或系统选型表，证明内外机组合、管长和高差在厂家允许范围内。
### 3. 冷媒铜管标准
- 冷媒铜管必须使用 `脱氧无缝紫铜管`，不使用普通水管、劣质铜管或回收铜管。
- 铜管品牌在 `海亮 / 飞轮 / 中佳 / 宏泰` 中选择。
- 铜管必须适配 `R410A/R32` 高压冷媒系统。
- `6.35mm` 铜管壁厚不低于 `0.8mm`。
- `9.52mm` 铜管壁厚不低于 `0.8mm`。
- `12.7mm` 铜管壁厚不低于 `0.8mm`。
- `15.88mm` 铜管壁厚不低于 `1.0mm`。
- `19.05mm` 铜管壁厚不低于 `1.0mm`。
- 实际管径按厂家设计确定，但壁厚不得低于以上标准和厂家要求。
- 铜管进场必须整卷或规范盘管进场，端口封堵，避免灰尘和水汽进入。
### 4. 保温和包扎标准
- 保温棉使用 `B1级阻燃闭孔橡塑保温棉`。
- 铜管保温厚度不低于 `15mm`，吊顶内、阳台、温差大和易冷凝位置优先 `20mm`。
- 保温棉接缝必须用专用胶水粘严，再用胶带密封。
- 保温棉不能破损、露铜、断开，否则后期容易结露滴水。
- 冷热管分别独立保温，不允许两根铜管共用一根保温管。
### 5. 冷凝水管标准
- 冷凝水管使用 `UPVC硬管`，主排水不使用软管。
- 单台内机冷凝水管管径不低于 `DN25`。
- 汇总排水或长距离排水位置按现场加大管径。
- 冷凝水管坡度不低于 `1%`，即每米下降不低于 `1cm`。
- 冷凝水管必须做保温，避免吊顶内结露。
- 冷凝水管必须设置固定支架，避免后期下垂形成倒坡。
- 每台内机排水完成后必须做通水试验。
### 6. 线材和电气标准
- 信号线、电源线使用国标铜芯线，不使用非标线、铝线和杂牌线。
- 线材品牌在 `远东 / 上上 / 熊猫 / 正泰 / 德力西` 中选择。
- 室内机控制线、信号线按厂家要求配置，必须穿管保护。
- 强电和弱电分开走线，避免干扰。
- 外机电源必须按设备功率单独配置回路和空开。
### 7. 安装工艺
- 铜管焊接必须使用氮气保护焊，禁止无氮焊接。
- 焊接时铜管内部持续充氮，防止氧化皮进入系统。
- 铜管弯管必须用专业弯管器，不允许硬折、压扁、死弯。
- 铜管穿墙洞、管线穿梁穿墙位置必须做保护和封堵。
- 内机吊装必须牢固、水平，吊杆和减震件安装到位。
- 内机吊装后必须做防尘保护，避免装修灰尘进入机器。
- 分歧管使用厂家原厂或厂家认可配件，不允许用普通三通替代。
- 分歧管安装方向、坡度、前后直管段按厂家要求执行，并拍照留存。
- 外机安装位置必须满足散热、回风、排风和检修要求，不得贴墙、堵风或被后期柜体遮挡。
- 外机必须配置可靠支架或基础、减震垫和冷凝排水路径，运行时不得明显共振扰民。
- 管线穿墙洞必须向室外或排水方向有合理坡度，封堵后不得渗水、漏风、进虫。
### 8. 保压、抽真空和测试条件
- 管路焊接完成后必须做氮气保压测试。
- 保压测试压力按 `4.15MPa` 执行；厂家标准更高时按厂家标准。
- 保压测试时长不低于 `24小时`。
- 保压前后压力表读数必须拍照留存，由业主或监理确认。
- 保压期间不得明显掉压；因温度变化导致的轻微变化需由商家说明。
- 抽真空必须使用真空泵和真空表，不接受只靠冷媒排空气。
- 抽真空必须使用数字真空表；真空度达到厂家标准，不低于文档约定值。
- 如采用表压口径，至少达到 `-0.1MPa`；如采用微米表口径，按厂家安装手册执行并留存读数照片。
- 抽真空时长不低于 `30分钟`；全屋多联机系统按不低于 `60分钟` 执行。
- 抽真空完成后保压观察不少于 `10分钟`，真空表或数字真空表不得明显回升，并留存前后照片或视频。
- 冷凝水排水完成后做通水试验，每台内机倒水测试，排水顺畅、无倒坡、无渗漏。
### 9. 吊顶、风口和检修
- 每台内机必须预留检修口。
- 检修口必须能检修电机、排水、过滤网和管线接头。
- 出风口不得直吹床头、书桌、沙发长期坐卧位置。
- 回风口和检修口不得被柜体、窗帘盒、灯槽遮挡。
- 风口优先选 `ABS风口` 或不易结露变形的优质风口。
- 风口尺寸、颜色、材质必须写入报价。
- 加长风口必须确认出风效果和检修方式，不只为了好看。
- 吊顶高度、风口位置、检修口位置必须在水电前确定。
### 10. 调试验收
- 调试时逐台内机试运行制冷、制热、送风和控制面板。
- 每个房间检查出风、回风、噪音、震动和排水。
- 外机运行时检查支架、减震垫、噪音和排风空间。
- 商家必须提供设备清单、辅材清单、保压照片、抽真空照片、调试记录。
- 业主验收前不得封死检修口和关键管线位置。
- 制冷调试时记录每台内机出风口温度、回风口温度和运行电流。
- 制热调试时记录每台内机出风状态、噪音、震动和化霜/排水情况。
- 调试记录必须包含室内外环境温度、运行模式、运行时长、异常代码和整改结论。
- 每台内机连续运行不少于 `30分钟`，无异常噪音、异味、漏水、报错后才能验收。
- 装修封板前必须完成一次隐蔽验收，拍摄铜管、保温、分歧管、冷凝水管、线材和检修口位置。
### 11. 质保和售后
- 整机质保不低于厂家标准。
- 压缩机质保不低于厂家标准。
- 安装质保不低于 `2年`。
- 冷媒泄漏质保不低于 `2年`。
- 冷凝水漏水质保不低于 `2年`。
- 售后响应时间不超过 `24小时`。
- 因安装导致漏水、冷媒泄漏、噪音异常、制冷制热异常，由商家负责整改。
### 12. 付款节点
- 定金不超过合同总价 `20%`。
- 设备进场、核对内外机型号后支付第二笔。
- 管线安装完成、保压测试合格后支付第三笔。
- 调试运行、验收合格后支付主要尾款。
- 保留 `5%-10%` 质保尾款，运行一段时间无漏水、冷媒泄漏和异常噪音后支付。
## 签约和验收检查
### 签约前检查
- [ ] 内机型号、外机型号、安装房间、风口形式已写清。
- [ ] 铜管品牌、管径、壁厚、保温棉、冷凝水管规格已写清。
- [ ] 氮气保护焊、保压、抽真空、通水测试、检修口要求已写清。
- [ ] 打孔、吊装、超长铜管、冷媒追加、分歧管费用是否另算已写清。
- [ ] 付款节点按验收走，质保尾款保留 `5%-10%`。
- [ ] 不接受“按常规安装、原厂辅材、品牌铜管、包安装、按实际结算不写单价”。
### 验收时检查
- [ ] 内机、外机型号和合同一致。
- [ ] 每个房间内机位置、出风口、回风口、检修口符合图纸。
- [ ] 出风不直吹床头、书桌、沙发长期坐卧位置。
- [ ] 铜管品牌、管径、壁厚与合同一致。
- [ ] 保温棉为 B1 级阻燃闭孔橡塑，厚度不低于 15mm。
- [ ] 冷凝水管为 UPVC 硬管，单台内机不低于 DN25。
- [ ] 冷凝水管坡度不低于 1%，无倒坡。
- [ ] 信号线、电源线为国标铜芯线。
- [ ] 焊接时确认为氮气保护焊，有现场照片或视频。
- [ ] 氮气保压压力 4.15MPa，保压不低于 24 小时，记录留存。
- [ ] 抽真空达到 -0.1MPa，时间不低于 30 分钟，全屋多联机不低于 60 分钟。
- [ ] 分歧管为厂家原厂或认可配件，安装方向和照片留存。
- [ ] 外机散热、排风、减震、检修空间符合厂家要求。
- [ ] 冷凝水排水通畅，无倒坡和漏水。
- [ ] 每台内机完成通水测试。
- [ ] 风口安装平整，检修口可正常打开。
- [ ] 每台内机连续试运行不少于 30 分钟，制冷、制热、噪音和遥控/控制面板正常。
- [ ] 商家交付设备清单、辅材清单、保压照片、抽真空照片、调试记录和保修资料。
