# 阳台与封窗
## 客厅大阳台
- 洗衣机、烘干机：放大阳台一侧，优先上下叠放。
- 扫地机器人基站：放大阳台靠客厅一侧的家政柜底部，选自动上下水款。
- 电动拖把：放大阳台家政柜，先预留位置和插座。
- 拖把、扫把、吸尘器、清洁剂：放大阳台家政高柜。
- 备用纸品、工具、清洁耗材：放大阳台家政柜。
- 自动窗帘：客厅大阳台预留电源和窗帘盒。
- 封窗：按高层系统窗标准执行，重点看安全、隔音、保温、通风、限位和防坠。
- 柜体：只做一侧家政柜，不做满墙压迫式柜体。
## 书房小阳台
- 绿植、少量备用工具、临时晾晒小件。
- 不放洗衣机、烘干机。
- 不放扫地机器人基站。
- 不放高频清洁设备。
- 保持书房安静、干净，不做潮湿家政区。
## 需要买 / 定的设备
- 洗衣机：独立洗衣机。
- 烘干机：独立烘干机。
- 洗烘叠放支架：根据品牌型号确定。
- 扫地机器人：带自动上下水自清洗基站。
- 电动拖把：后期购买，装修时先预留位置和插座。
- 阳台柜：家政高柜、洗衣柜或吊柜。
- 自动窗帘：客厅大阳台预留。
- 封窗：型材、玻璃、五金、纱窗和开启扇方案。
## 必须确认
- 客厅大阳台作为主家政区，洗衣、烘干、扫地机器人、清洁工具和地漏排水必须提前定点。
- 扫地机器人基站必须预留自动上下水、插座、防臭、防反水和检修空间。
- 阳台防水、地漏坡度、封窗安全、柜体防潮和插座防溅必须满足。
- 洗衣机、烘干机叠放必须使用匹配支架，插座和水路不能被机器挡死。
## 客厅大阳台标准
- 大阳台作为唯一主家政区。
- 洗衣机和烘干机上下叠放，使用原厂或匹配叠放支架。
- 洗烘区预留进水、排水、地漏、插座和检修空间。
- 扫地机器人基站放大阳台家政柜底部，必须做自动上下水。
- 扫地机器人基站排水必须做防臭、防反水、防溢水处理。
- 家政高柜集中收纳拖把、扫把、吸尘器、清洁剂、纸品和工具。
- 柜体不挡窗户开启、不挡地漏、不挡检修口。
- 家政柜不是越满越好，洗烘、机器人、拖把、耗材和检修位要留出真实操作空间。
- 洗衣机和烘干机叠放必须确认承重、找平、防震和防倾倒固定，运行时不明显晃动。
- 洗烘区上方柜体和侧板必须防潮，柜门开启不得影响烘干机散热和检修。
- 扫地机器人基站必须能抽出维护，取尘袋、加水、清洁底座不被柜门和窗帘阻挡。
## 书房小阳台标准
- 小阳台不做主晾晒区、不放洗衣机、不放烘干机、不放扫地机器人。
- 小阳台只承担绿植、少量备用工具、临时小件晾晒。
- 小阳台保持书房安静干净，不做潮湿家政区。
## 水电和防水
- 两个阳台地面做防水，墙面上翻不低于 `30cm`。
- 大阳台洗烘、扫地机器人、电动拖把插座必须带防溅保护。
- 插座位置高于可能积水位置，避开水管直接下方。
- 扫地机器人和洗衣机排水接入排水管，避免直接排到地面地漏造成反味。
- 阳台地面坡向地漏，不积水。
- 洗衣机龙头、扫地机器人进水阀、检修阀门必须露出可操作，不得藏在固定柜板后。
- 洗衣机、烘干机、扫地机器人、电动拖把插座分别预留，禁止长期共用拖线板。
- 阳台地漏必须使用防臭地漏，洗衣机排水必须防溢流、防反水。
- 阳台墙地砖和柜体安装完成后，不得破坏防水层；必须打孔时先确认位置并做密封处理。
- 扫地机器人从阳台进入室内的常走路径要控制门槛和收口高差，按目标机型最大越障高度复核。
## 封窗和安全
- 两个阳台封窗按高层封窗标准执行。
- 客厅大阳台必须保留开启扇和纱窗，不能全固定。
- 儿童可接触窗户必须有限位和防坠措施。
- 阳台柜体固定，避免儿童攀爬倾倒。
## 材料与选型
- 阳台封窗型材在 `兴发 / 坚美 / 凤铝` 中选择，客厅大阳台使用 `108/110系列断桥铝系统窗`。
- 阳台封窗玻璃按高层配置，客厅大阳台使用夹胶中空 Low-E 玻璃，所有玻璃带 `3C` 标识。
- 阳台柜体使用 `ENF级防潮板 / ENF级多层板`，封边采用 `PUR封边` 或 `激光封边`。
- 阳台柜门采用门板下延、45度斜切或 G 型拉手槽，不用高频反弹器和自动开合器。
- 洗衣机、烘干机、扫地机器人、电动拖把插座采用 `西门子远景Pro / 施耐德绎尚 / 罗格朗逸远`，并做防溅措施。
- 阳台地面使用防滑砖，客厅大阳台和室内衔接位置按全屋通铺和收口方案处理。
- 阳台防水材料在 `东方雨虹雨虹200 / 德高K11 / 西卡家装柔性防水` 中选择，墙面上翻不低于 `30cm`。
- 阳台地漏采用 `潜水艇 / 九牧 / 科勒` 防臭地漏，洗衣机排水做防溢流、防反水。
## 需要提前定好尺寸和点位
- 洗衣机、烘干机宽深高。
- 洗烘叠放后总高度，以及上方吊柜空间。
- 扫地机器人基站宽深高、开盖方向、取尘袋空间。
- 扫地机器人基站的进水、排水、防臭、防溢水、插座和检修空间。
- 电动拖把位置、插座、水源/地漏距离。
- 阳台柜深度、高度和开门方式。
- 地漏、排水管、进水口和插座位置。
- 封窗开启扇位置、窗台高度、护栏和纱窗方式。
- 客厅窗帘盒宽度、电机位置和电源点位。
## 复尺时重点确认
- 两个阳台的净宽、净深、梁柱和窗台高度。
- 大阳台家政柜、水源、排水、地漏和插座的具体位置。
- 洗烘叠放后不影响开窗、开柜门和日常通行。
- 扫地机器人基站方便全屋进出，不被门槛、柜门或窗帘影响。
- 阳台到室内门口收口高度和轨道高差，满足扫地机器人长期稳定通行。
- 扫地机器人和电动拖把排水接入排水管，并做好防臭。
- 家政柜不挡住检修口、地漏或窗户开启。

## 封窗专项

## 推荐参数
- 型材：断桥铝系统窗，客厅大阳台优先 108/110 系列，卧室优先 75/80 系列。
- 玻璃：高层优先夹胶中空或钢化中空 Low-E；大阳台、低窗台、儿童活动位置优先夹胶中空。
- 开启：优先内开内倒，不选高层外开窗。
- 五金：写清品牌和数量，重点看合页、执手、锁点、滑撑、防坠件。
- 胶条：优先三元乙丙胶条，合同写清。
- 密封胶：室外耐候胶、室内密封胶写清品牌和施工范围。
- 纱窗：采用金刚网或高透纱窗，开启方式和窗型匹配。

## 业主指定合同标准
### 1. 报价和图纸
- 商家必须提供全屋窗型图，标清每一樘窗的固定扇、开启扇、纱窗、限位器位置。
- 报价必须按每一樘窗分别列面积、单价、总价，不接受只报总价。
- 报价必须包含拆旧、垃圾清运、基层处理、收口、防水、发泡、打胶、成品保护。
- 任何吊装、高空作业、超大玻璃、异形窗、转角窗等费用必须在签约前一次性列明。
- 合同禁止使用模糊说法，型材、玻璃、五金、胶条、密封胶都必须写成明确参数。
- 每樘窗必须标注洞口尺寸、成品尺寸、开启方向、玻璃分格、排水孔位置和窗台收口做法。
- 复尺后商家必须出最终生产图，业主书面确认后才能下单生产。
- 生产图确认后因商家复尺、下单、生产错误造成的返工和延期，由商家承担。
### 2. 型材标准
- 客厅大阳台优先按 `108/110系列断桥铝系统窗` 报价。
- 卧室和普通窗优先按 `75/80系列断桥铝系统窗` 报价。
- 型材品牌优先 `兴发铝材 / 坚美铝材 / 凤铝铝材`。
- 铝材材质要求 `6063-T5原生铝`。
- 主受力型材壁厚不低于 `2.0mm`。
- 非主受力型材壁厚不低于 `1.8mm`。
- 隔热条使用 `PA66-GF25尼龙隔热条`，不接受 PVC 隔热条。
- 客厅大阳台和迎风面重点看抗风压性能，商家必须按高层外窗报价。
- 型材品牌、系列、壁厚、颜色、隔热条材质必须写进合同。
### 3. 玻璃标准
- 所有玻璃必须钢化，并带 `3C` 标识。
- 客厅大阳台优先配置 `6mm钢化 Low-E + 16A/18A中空 + 5mm钢化 + 0.76PVB + 5mm钢化夹胶`。
- 普通卧室窗优先配置 `5mm钢化 Low-E + 20A中空 + 5mm钢化`。
- 临街、噪音较大、低窗台、儿童容易接触的位置，升级为 `夹胶中空玻璃`。
- 单块玻璃面积不追求过大，超过 `1.5平方米` 或底边离地较低的位置，按更高安全标准配置夹胶中空。
- 不采用普通单层玻璃、非钢化玻璃和无 3C 标识玻璃。
- 玻璃品牌、厚度、中空层厚度、Low-E、夹胶层厚度必须写进合同。
### 4. 五金、胶条、密封胶标准
- 五金在 `好博 HOPO / 坚朗 KINLONG / 丝吉利娅 SIEGENIA / 诺托 ROTO` 中选择。
- 开启扇必须写清执手、合页、滑撑、锁点、防坠件数量。
- 胶条使用 `三元乙丙 EPDM 胶条`。
- 室外密封胶使用中性耐候胶，在 `之江 JS-2000 / 白云 SS611 / 道康宁 791 / 西卡 Sikasil` 中选择。
- 室内密封胶使用中性环保密封胶。
- 外墙和窗框交接处必须连续打胶，不允许漏打、断胶、薄打。
- 静音和密封优先选多腔体型材、多道密封结构，不选普通推拉窗作为主要外窗。
### 5. 性能要求
- 商家必须提供所选系列门窗的检测报告，至少包含抗风压性能、水密性能、气密性能。
- 按高层外窗要求，抗风压性能不低于 `5级`，迎风面和大阳台优先更高等级。
- 气密性能不低于 `6级`。
- 水密性能不低于 `4级`。
- 隔音重点靠 `夹胶中空玻璃 + 平开/内开内倒结构 + 多道密封`，不要只听商家说“静音玻璃”。
- 商家报价时必须说明实际使用的玻璃结构，并证明检测报告对应同系列产品。
### 6. 开启、通风和安全
- 每个主要房间至少保留一个有效开启扇。
- 客厅大阳台不能只做全固定玻璃，必须保留对流通风位置。
- 客厅大阳台至少设置 `2个开启扇`，分别布置在利于对流的位置。
- 开启方式优先 `内开内倒`，日常通风用内倒，清洁和大通风时再内开。
- 不选高层外开窗；确因物业原窗型限制必须外开时，必须配置防坠绳、防坠链和限位器。
- 有儿童活动区域的开启扇必须安装限位器。
- 高层外开窗必须有防坠措施。
- 纱窗优先金刚网或高透纱窗，数量与开启扇匹配。
### 7. 施工工艺
- 安装前必须复尺，窗框尺寸以现场复尺为准。
- 窗框安装必须横平竖直，固定点间距合理。
- 窗框与墙体缝隙必须发泡填充饱满。
- 室外侧必须做耐候密封，窗台和外口必须考虑排水坡度。
- 室内侧收口平整，和墙面、瓷砖、柜体交接责任写清。
- 施工过程保护地面、墙面、瓷砖、栏杆和已安装成品。
- 窗框固定片、膨胀螺栓或连接件规格必须满足高层外窗要求，不允许只靠发泡胶固定。
- 固定点距框角不大于 `150-200mm`，中间固定点间距不大于 `500mm`，现场受结构限制时必须加固并拍照说明。
- 窗框下口必须有排水通道，排水孔不得被发泡、密封胶、收口材料堵死。
- 室外胶缝必须打在干燥、洁净、牢固基层上，雨天、基层潮湿或明显起砂时不得打胶。
- 拆除旧窗后发现洞口空鼓、开裂、渗水，必须先基层修补并验收后再安装新窗。
### 8. 测试和验收条件
- 安装完成后检查所有开启扇，连续开合不少于 `10次`，不得卡顿、异响、下垂。
- 锁闭后窗扇不得明显晃动，执手、锁点、合页、滑撑工作正常。
- 室外具备条件时做淋水测试，重点测试窗框四周、拼接处、转角处，连续淋水不少于 `30分钟`，室内不得渗水。
- 淋水条件不具备时，至少约定第一次大雨后复检，发现渗漏由商家免费整改。
- 验收时核对型材壁厚、玻璃 3C、Low-E、夹胶层、五金品牌、胶条和密封胶品牌。
- 淋水测试必须从窗外上方和两侧连续淋水，不能只在室内少量泼水代替。
- 验收必须逐樘核对玻璃 3C 编码、五金品牌标识、胶条材质、密封胶批次和检测报告。
- 发现漏水、排水孔堵塞、窗扇下垂、明显晃动、玻璃划伤、密封胶断胶，必须整改后重新验收。
### 9. 施工责任
- 高空作业、吊装、搬运安全责任由封窗商家承担。
- 施工造成外墙、室内墙面、地面、瓷砖、柜体损坏，由封窗商家修复或赔偿。
- 窗框周边防水、发泡、密封、收口由封窗商家负责。
- 因安装原因导致漏水、渗水、起雾、开关不顺，由封窗商家负责整改。
### 10. 质保和售后
- 型材质保不低于 `10年`。
- 五金质保不低于 `5年`。
- 玻璃中空起雾质保不低于 `5年`。
- 漏水渗水质保不低于 `5年`。
- 胶条和密封胶质保不低于 `3年`。
- 漏水问题 `24小时内响应`，约定时间内上门处理。
- 五金故障 `72小时内响应`，约定时间内维修或更换。
### 11. 付款节点
- 定金不超过合同总价 `20%`。
- 材料进场、核对品牌型号和玻璃 3C 标识后支付第二笔。
- 安装完成、验收合格后再支付主要尾款。
- 保留 `5%-10%` 质保尾款，验收后一段时间无漏水、起雾、开关故障再支付。
- 第一次有效大雨或淋水复验前，质保尾款不支付。
- 私自替换型材、玻璃、五金、胶条、密封胶的，业主有权拒收并要求按合同重做。
## 签约和验收检查
### 签约前检查
- [ ] 型材系列、壁厚、原生铝、隔热条、胶条已写清。
