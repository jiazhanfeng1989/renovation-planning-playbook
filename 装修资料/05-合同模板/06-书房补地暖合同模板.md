# 书房补地暖合同模板

打印和签约时，正文及连续装订页逐页签字或盖章，空白处划线作废。本合同仅适用于 `书房新增一路地暖`，不适用于全屋重做地暖。

## 签约信息
- 甲方：本房屋业主
- 乙方：地暖施工商家工商全称
- 项目地址：本项目房屋
- 联系人：甲方本人；乙方销售、施工负责人、售后负责人
- 联系电话：签约时填写
- 预估签约报价：`合理报价 0.9万`；`高配报价 1.2-1.8万`
- 合同总价：按本合同报价汇总填写大写和小写金额
- 收款账户：乙方对公账户全称、开户行、账号
- 开工条件：物业或供暖管理方允许接入；接入点位确认；书房完成面高度已复核
- 签约日期：签约时填写

## 合同模板

### 1. 适用范围
- 本合同仅适用于 `书房新增一路地暖`。
- 乙方负责现场勘查、方案确认、材料供应、施工安装、打压测试、调试、资料交付和售后。
- 乙方负责和半包施工、瓷砖、全屋定制、水电、门套完成面做工序配合。

### 2. 报价范围
- 报价必须写清：铺设区域、预计面积、管材米数、接入位置、分集水器改造、阀门、过滤器、保温板、反射膜、边界保温条、固定方式、温控件、打压、回填配合、调试和售后。
- 报价必须写清原系统接入方式、原分集水器或原接口改动范围，以及由谁负责原系统改动后的渗漏和失效责任。
- 不接受只写 `书房补一路地暖 / 地暖改造一项 / 按现场实际发生`。
- 打孔、开槽、接管、阀门、过滤器、回填、带压施工、调试、资料交付必须写入总价或逐项清单。
- 除甲方主动变更外，结算增幅不得超过签约总价 `5%`。

### 3. 材料标准
- 地暖管采用 `PE-RT II型` 或 `PEX-a阻氧管`，规格不低于 `16x2.0mm`。
- 地暖管品牌在 `瑞好 / 乔治费歇尔 / 伟星 / 日丰 / 保利` 中选定并写入合同。
- 分集水器、阀门、过滤器、排气件必须写清品牌、型号、路数和接口规格。
- 保温板采用 `XPS挤塑板`，厚度不低于 `20mm`，边界保温条厚度不低于 `8mm`。
- 温控件、执行器或控制配件的品牌、型号、安装位置和供电方式必须写清。

### 4. 施工与测试条款
- 地暖管中间不得有接头；回路编号必须清晰。
- 固定柜体下方不铺地暖管。
- 管间距 `150-200mm`，冷区可加密到 `150mm`。
- 第一次打压按 `0.8MPa / 30分钟` 执行，再做 `24小时` 保压观察。
- 回填期间保持 `0.4-0.6MPa` 带压施工。
- 回填完成后二次复压，合格后才能贴砖。

### 5. 配合、资料与售后
- 乙方必须和半包方确认书房完成面高度、门套高度、瓷砖通铺标高和柜体安装关系。
- 地暖未验收通过前，不得进入贴砖或其他不可逆工序。
- 原系统接口改动前后，乙方必须分别留存接口位置、阀门、接头和压力表照片。
- 施工全过程必须留存管线走向照片，照片应能看清墙体参照物和回路编号。
- 乙方必须交付：铺设图、回路编号图、材料合格证、打压记录、调试记录、质保书、售后联系人。
- 乙方必须交付：原系统接口改动前后照片、接口位置说明和责任边界说明。
- 售后响应时间应在 `48小时` 内。
