# 主卧设计

## 这份文档管什么
- 这份只讨论 `设计`，不讨论板材、五金、门芯、窗型和开关品牌。
- 重点是：`功能划分`、`空间利用`、`空间布局`、`颜色配比`、`家具搭配`、`睡眠舒适度`、`软装和灯光氛围`。
- 主卧不是杂物间，也不是临时办公间，它首先应该是一个让人稳定休息和放松的房间。

## 本项目主卧定位
- 主卧主要承担：`睡眠`、`衣物收纳`、`简单梳妆`、`夜间放松`。
- 设计重点不是做满功能，而是让这个房间有 `安静感`、`秩序感`、`包裹感`。
- 主卧不建议承担长期办公、儿童杂物堆放和全屋临时收纳补位。

## 设计总原则
- 先定 `床`，再定衣柜、床头柜、灯光和窗帘。
- 先保证 `睡眠舒适度`，再谈装饰效果。
- 先保证 `通道和开门关系`，再追求满墙储物。
- 先保证 `视觉安静`，再做复杂背景和色彩层次。
- 先保证 `晚上好住`，再考虑白天拍照好看。

## 功能划分

### 1. 睡眠区
- 床是主卧核心，不是背景墙。
- 睡眠区要围绕 `安静`、`柔和`、`不压迫` 来设计。
- 所有视觉中心都应该服务床和休息感，而不是抢床的存在感。

### 2. 收纳区
- 收纳以衣柜为核心。
- 目标不是柜子越多越好，而是衣服、被褥、换季物和小件都有稳定归位。
- 封闭收纳优先，减少视觉杂乱和落灰。

### 3. 梳妆区
- 主卧可以保留小型梳妆位。
- 梳妆区更适合轻量存在，不建议发展成大体量多功能桌面。
- 重点是顺手、好看、和整体氛围协调。

## 空间利用

### 1. 床优先于柜体
- 先保证床两侧通行和床头使用感，再决定柜体做到多大。
- 主卧最怕为了收纳把床挤成一个被柜体夹住的睡眠夹缝。

### 2. 柜体克制
- 衣柜可以做足，但不应让房间显得沉重。
- 主卧收纳需要的是 `够用 + 稳定 + 看起来不乱`，不是体量上的压制。
- 柜门一关全都藏起来，比做开放式衣帽架更适合这个房间。

### 3. 台面和小件
- 床头柜台面、梳妆台台面要能放下真实高频物品，但不应该鼓励长期堆积。
- 好的卧室不是“什么都放得下”，而是“东西不会不断溢出来”。

## 空间布局

### 1. 床位
- 床应成为房间中最稳定、最安心的位置。
- 进门第一眼最好先感受到整洁和平衡，而不是大柜压顶或复杂背景。
- 床靠墙和床头背景的关系要优先考虑睡眠感和对称感，但不要为了绝对对称牺牲使用。

### 2. 衣柜关系
- 衣柜位置应与床、门、窗帘、床头柜一起看。
- 柜门开启、通道、床头活动范围都要提前复核。
- 衣柜体量越大，越要控制门型和颜色的存在感。

### 3. 梳妆位关系
- 梳妆台优先靠近自然光更友好。
- 但梳妆台不能挡主通道，也不能和衣柜开门打架。
- 梳妆位更适合做成轻巧、融入式，而不是再增加一个很重的家具中心。

## 家具搭配

### 1. 床
- `1.8米床` 是主卧的主角，床体和床头应优先给人稳、柔和、放松的感受。
- 带一定存在感的床头可以增加包裹感，但不建议太厚重、太复杂。

### 2. 床头柜
- 两侧床头柜更适合完整卧室秩序。
- 床头柜不需要太大，但应能满足手机、灯、水杯、纸巾和少量夜间物品。
- 如果空间紧，床头柜可以轻量化，但不建议彻底取消双侧使用逻辑。

### 3. 衣柜
- 主卧衣柜更适合整体感强、分缝简洁、门型克制的设计。
- 柜体不建议做过多开放展示，不然卧室会失去安静感。

### 4. 梳妆台
- 梳妆台更适合小体量、低干扰、和衣柜或床头体系协调。
- 它可以是主卧精致感来源之一，但不应压过床和休息感。

## 颜色配比

### 1. 主卧基调
- 主卧更适合 `低饱和、偏柔和、偏安静` 的颜色结构。
- 相比客餐厅，主卧应该更收一点、更静一点。
- 重点不是做出强烈风格，而是让人一进来就松下来。

### 2. 推荐方向
- 暖白、米白、浅灰、灰米、低饱和奶咖、柔和木色都更适合主卧。
- 窗帘、床品、地毯、靠包可以比墙面和柜体多一点层次，但不宜跳。

### 3. 不建议的方向
- 不建议主卧用很重的大面积深色背景。
- 不建议大面积高对比拼色。
- 不建议太冷、太硬的灰黑白组合把卧室做得像展示空间。

## 软装和织物

### 1. 床品
- 主卧氛围很大程度上来自床品。
- 床品更适合柔和、克制、有层次但不花哨的搭配。
- 主卧不是儿童房，不需要靠图案建立活力。

### 2. 窗帘
- 窗帘对睡眠感影响很大。
- 更适合 `遮光帘 + 纱帘` 的组合，既满足睡眠，也保持白天层次。
- 窗帘颜色最好和墙面、床品有递进关系，不要突然变重或跳色。

### 3. 地毯与靠包
- 如果要加地毯，作用更多是提升柔软感和层次，而不是制造强视觉中心。
- 靠包和床尾毯可以增加卧室完成度，但数量和颜色要克制。

## 灯光和氛围

### 1. 主灯不是全部
- 主卧不适合只靠一盏顶灯解决所有场景。
- 更好的做法是：主灯负责基础，床头灯或局部灯负责夜间和氛围。

### 2. 床头照明
- 床头阅读、夜间起身、伴侣不同作息，都需要更温和的局部光。
- 床头灯具和床头柜、床头背景要一起看，不要后期临时补。

### 3. 梳妆照明
- 梳妆位的灯光要服务使用，不只做装饰。
- 如果自然光条件一般，局部照明更要提前考虑。

### 4. 氛围感
- 主卧灯光色温不宜太冷。
- 好的主卧照明重点是让晚上舒服，而不是让白天拍照更亮。

## 睡眠舒适度

### 1. 空调和风感
- 主卧要重点避免空调直吹床头和枕头区。
- 夜间体感稳定比“风口看起来整齐”更重要。

### 2. 声音
- 主卧应优先减少来自客餐厅、走廊和窗外的干扰感。
- 这意味着门、窗帘、床头背景、软装和空间布置都应向“安静”靠拢。

### 3. 视觉安静
- 主卧的平静很大程度来自“不乱”。
- 收纳稳定、颜色收束、表面干净，睡眠感会比复杂设计更强。

## 和其他系统怎么配合

### 和中央空调
- 风口不要直吹床位。
- 吊顶和床头背景的关系要以睡眠感优先，不追求硬对称。

### 和定制柜
- 衣柜体量和门型要服务卧室安静感，不要做成客厅展示柜逻辑。
- 柜体再重要，也不能压缩主卧的呼吸感。

### 和门窗
- 窗帘、窗、门都会影响主卧的完整度和舒适度。
- 门口进入视角、窗边采光感和床头背景应一起看，而不是分开处理。

## 这个家更适合的主卧方向
- 更适合：`柔和安静 + 低饱和配色 + 封闭收纳 + 轻量梳妆 + 多层次灯光`。
- 不建议：满墙复杂背景、过重深色、开放式衣架、过多装饰线条、强办公感家具。
- 主卧更应该像一个真正休息的空间，而不是“把很多功能塞进来的大房间”。

## 设计定稿清单
- [ ] `1.8米床` 已作为主卧第一核心确定下来。
- [ ] 床两侧通道和床头柜关系已复核。
- [ ] 衣柜体量没有压迫主卧，柜门开启关系顺畅。
- [ ] 梳妆位轻量、顺手，不挡主通道。
- [ ] 主卧整体颜色基调安静、柔和、低饱和。
- [ ] 床品、窗帘、柜体和墙面之间的颜色关系已统一。
- [ ] 主灯、床头灯和局部灯光层次已考虑。
- [ ] 空调出风不会直吹床位。
- [ ] 主卧整体没有被过度功能化，仍然以睡眠和放松为主。
- [ ] 这个房间在白天整洁，在晚上也真正舒服。
