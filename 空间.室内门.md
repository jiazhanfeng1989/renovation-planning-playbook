# 室内门

## 业主指定合同标准
## 1. 报价范围
- 报价必须拆分：门扇、门套、套线、五金、安装、运输、上楼、收口、门吸、门锁。
- 每樘门分别写清洞口尺寸、成品尺寸、开启方向、门扇厚度、门套厚度、门芯结构、五金型号。
- 垭口、门套加宽、单包套/双包套、玻璃、百叶、隐形门吸、静音升级、自动门底密封必须单独列价。
- 合同禁止只写“全屋几樘门套餐价”。
- 卫生间门如采用玻璃门或铝合金门，必须单独写清玻璃、型材、五金和地轨/吊轨方式。
## 2. 房间建议配置
### 卧室、儿童房、书房
- 优先 `实木复合门 / 静音门`。
- 门扇厚度优先 `45mm`。
- 要求静音条、合页、锁体、门底密封齐全。
- 儿童房门锁避免误反锁和夹手风险。
### 卫生间
- 优先 `铝镁合金门 + 钢化磨砂玻璃` 或同等级耐潮方案。
- 如果坚持木门，必须把 `门套基层防潮 + 下口防水 + 门扇防潮结构` 写清。
- 卫生间门下口和挡水、门槛高差要一起复核。
## 3. 材料和结构标准
### 3.1 门扇标准
- 卧室、儿童房、书房门扇厚度不低于 `45mm`。
- 门扇表面优先耐磨、耐污、低反光饰面。
- 门扇平整，无明显起鼓、翘曲、色差和波浪。
### 3.2 门芯结构标准
- 必须明确门芯结构，不接受只写“静音门”“实木门”。
- 可接受方向：`桥洞力学板 / 实木复合填充 / 同等级稳定结构`。
- 不接受低配空心蜂窝纸门冒充高配静音门。
### 3.3 门套和基层标准
- 门套材质、厚度、基层做法必须写清。
- 门洞两侧垂直度、顶部平整度、墙厚偏差必须在复尺前处理。
- 门套不能大面积离墙、收口爆缝、压砖关系混乱。
- 卫生间门套基层必须考虑防潮，不接受泡水风险高的低配基层。
### 3.4 五金标准
- 合页优先 `DTC / 顶固 / 海蒂诗 / 百隆` 同等级耐用五金。
- 锁体、把手、门吸必须写品牌、型号、颜色。
- 儿童房避免尖锐把手和高夹手风险五金。
- 高频房门五金不接受杂牌散件。
### 3.5 静音和密封标准
- 卧室、书房、儿童房门优先配静音条。
- 门底优先 `自动门底密封`；如不做自动门底密封，必须写清门底缝控制范围。
- 门扇关闭后不得明显晃动、撞框、异响。
## 4. 卫生间门专项标准
- 卫生间门优先防潮、通风和耐用，不只看颜值。
- 如采用铝镁合金门，玻璃必须钢化并带 `3C` 标识。
- 如采用极窄边玻璃门，型材壁厚、玻璃厚度、合页或滑轨五金必须写清。
- 卫生间门开启不得撞马桶、浴室柜、毛巾架、喷枪和淋浴玻璃。
- 卫生间门口地砖、挡水和门下口关系要提前定，不接受安装时临时切改。
## 5. 设计和复尺标准
- 每樘门必须记录：洞口宽、高、墙厚、地面完成面、门口收口方式、开启方向、是否靠墙、是否撞柜。
- 门洞复尺时间必须在 `地面完成面明确、墙面基层稳定、瓷砖和门口收口逻辑确定` 后。
- 卧室、书房、儿童房门与柜门、床头柜、书桌、窗帘、插座、开关避让必须复核。
- 卫生间门与马桶、浴室柜、喷枪、淋浴房、挡水条避让必须复核。
## 6. 与其他工种配合
- 室内门必须和地砖通铺、地暖完成面、卫生间挡水、踢脚线、定制柜体同步复核。
- 门厂复尺前，地砖铺贴完成面和门洞修补必须稳定。
- 墙面厚度、套线宽度、踢脚线高度、门套压边关系必须统一。
- 门洞修补和套口找平责任必须在半包合同中写清。
## 7. 安装工艺
- 门套安装必须横平竖直、固定牢靠。
- 门扇安装后上下缝、侧缝均匀，开关顺畅。
- 门锁、合页、门吸安装位置准确，不松动、不歪斜。
- 门套与墙面收口平顺，无大缝、无爆边、无明显打胶糊弄。
- 卫生间门下口、防水和门套接地位置处理完整。
## 8. 验收标准
- 每樘门开合不少于 `10次`，无卡顿、异响、回弹异常。
- 门扇关闭后不明显晃动，锁舌顺畅进入锁扣。
- 门套和墙面收口平整，无明显缝隙和爆边。
- 门扇表面无明显磕碰、划伤、色差、起鼓和翘曲。
- 合页、锁体、门吸型号与合同一致。
- 卫生间门防潮收口、门口高差、挡水和开门避让符合方案。
## 9. 付款节点
- 定金不超过合同总价 `20%`。
- 复尺完成、图纸签字后支付第二笔。
- 成品到场核对型号、数量后支付第三笔。
- 安装完成、验收合格后支付主要尾款。
- 保留 `5%-10%` 质保尾款，整改闭环后支付。
## 容易被坑的点
- 只写“静音门”“实木门”，不写门芯结构和厚度。
- 复尺太早，后期地面完成面变化导致门下口和门套出问题。
- 卫生间门只看款式，不看防潮和门口高差。
- 门套收口责任不清，后期和瓦工、油工、门厂互相甩锅。
- 合页锁体写品牌不写型号，最后给低配五金。
- 只看门扇，不看门吸、锁体、门底密封和关闭手感。
## 签约和验收检查
### 签约前检查
- [ ] 全屋门洞尺寸表完成。
- [ ] 每樘门开启方向和家具、柜体、窗帘、卫浴五金避让确认。
- [ ] 门扇厚度、门芯结构、静音条、门底密封写入合同。
- [ ] 门套材质、基层做法和收口责任写入合同。
- [ ] 合页、锁体、门吸品牌型号写入合同。
- [ ] 主卧、书房、儿童房静音配置确认。
- [ ] 卫生间门防潮和门口高差确认。
- [ ] 付款节点按验收走，质保尾款保留 `5%-10%`。
- [ ] 不接受“静音门、实木门、套餐门、品牌五金、常规安装、现场再收口”。
### 验收时检查
- [ ] 每樘门开合 `10次` 无异常。
- [ ] 门套收口平整，无明显离墙大缝和爆边。
- [ ] 卫生间门与挡水、浴室柜、马桶、淋浴房不打架。
