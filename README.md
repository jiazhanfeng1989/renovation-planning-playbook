# 业主装修预算与施工管理手册

这是一套面向业主的装修管理资料，重点用于预算控制、材料约束、施工工艺约束、现场检查和合同落地。

它不是装修灵感合集，也不是单纯的知识整理。更适合在真实装修过程中直接拿来用：开工前先定预算和边界，施工中按工艺和清单逐项对照，采购和签约时留痕，验收时减少漏项、返工和临时加价。

## 这套资料主要解决什么问题

- 预算失控：把装修费用拆到具体项目，提前看清大项、小项和预留空间
- 标准模糊：把材料、工艺和施工要求写清楚，减少口头约定
- 现场被动：按清单和专项逐项检查，辅助和施工方、材料商、定制商沟通
- 采购踩坑：把该买什么、怎么买、什么时候确认，尽量前置
- 合同空泛：把关键约定落到文本，降低后期扯皮成本

## 怎么使用

- 装修前：先看预算、开工、材料总表，把范围和优先级定下来
- 施工中：结合工艺总表、检查清单和空间专项，做现场对照
- 采购时：按材料清单和对应专项文档核对规格、品牌、数量和节点
- 签约时：把口头约定尽量落进对应合同模板
- 验收时：按检查项逐条确认，避免凭感觉收口
