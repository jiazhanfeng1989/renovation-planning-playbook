# 厨房设计

## 这份文档管什么
- 这份只讨论 `设计`，不讨论品牌、板材、五金、止逆阀、回路和施工工艺。
- 重点是：`功能分区`、`空间利用`、`空间布局`、`颜色配比`、`家具与柜体搭配`、`功能性`。
- 厨房设计的重点不是“塞多少设备”，而是让做饭这件事顺手、清爽、长期不乱。

## 本项目厨房定位
- 这是一个以 `日常中式烹饪` 为基础的家庭厨房。
- 要满足：做饭、洗菜、备菜、洗碗、蒸烤、小家电、基础囤货。
- 设计上优先级应当是：`动线顺`、`备菜区够用`、`收纳不乱`、`台面清爽`、`视觉干净`。

## 设计总原则
- 先定 `做饭流程`，再定柜体和设备。
- 先保 `连续台面`，再谈开放格和展示感。
- 先保 `使用效率`，再谈形式对称。
- 先保 `收纳边界清楚`，再谈所有东西都藏起来。
- 先保 `长期耐看`，再追求一次性很满的高级感。

## 功能分区

### 1. 洗
- 包括水槽、沥水、净水、垃圾初处理。
- 洗区不是孤立点，它应该和洗碗机、清洁收纳、常用餐具回收逻辑连在一起。

### 2. 切
- 切配区是厨房最容易被挤掉的核心区。
- 这一区要真正留下连续台面，不能被微波炉、刀架、调料、咖啡机长期侵占。

### 3. 炒
- 炒区围绕灶具、锅具、调料、油和常用铲勺组织。
- 炒区重点是取用顺手和清洁逻辑清楚，不是堆满挂件。

### 4. 洗碗
- 洗碗机如果有，逻辑上应靠近水槽区。
- 这样洗、放、收能形成一条顺手路径，而不是来回绕。

### 5. 电器
- 蒸箱、烤箱、微波炉、高柜电器区要集中，而不是把台面割裂成很多小块。
- 电器区既要好用，也要避免把厨房做成“家电陈列墙”。

### 6. 收纳
- 收纳应服务流程，不是平均分配给每个柜子。
- 厨房最常用的东西必须放在最顺手的位置，低频物品才往上、往里、往角落走。

## 空间利用

### 1. 台面利用
- 厨房最宝贵的是 `连续备菜台面`。
- 设备很多时，更要克制开放格和台面外摆，不然看起来功能齐全，用起来却没有操作面。
- 水槽和灶具之间优先保留主备菜区，这是最关键的位置。

### 2. 垂直空间
- 吊柜适合存轻物和低频物，不适合承担所有收纳压力。
- 高柜可以提高容量，但不能把厨房压得太重。
- 小厨房里，过多上柜会让空间显紧；但完全没有上柜，又容易台面外溢。

### 3. 小厨房策略
- 小厨房设计关键不是“做满”，而是“每一段都真的有用”。
- 与其做很多零碎功能，不如保留核心流程：洗、切、炒、收。
- 如果空间有限，宁可减少展示感，也不要牺牲工作流。

## 空间布局

### 1. 工作流程
- 更适合围绕 `水槽 - 备菜 - 灶具` 来组织。
- 冰箱、洗碗机、蒸烤箱、高柜和小家电都应围绕这个核心流程辅助布置。
- 做饭时最常走的路应该最短、最顺，不要频繁绕门、绕岛台、绕高柜。

### 2. 洗切炒关系
- 洗和切要相邻。
- 切和炒之间要顺接。
- 洗碗机最好靠近洗区，而不是远离水槽独立存在。
- 小家电区如果有，最好从主备菜区分开，不要长期挤占切配区。

### 3. 高柜关系
- 电器高柜适合集中放蒸箱、烤箱、微波炉等设备。
- 但它的存在不能破坏主台面连续性，也不能让厨房显得像走廊尽头一堵厚墙。
- 高柜越重，旁边越需要留白和轻一点的柜体关系去平衡。

## 收纳逻辑

### 1. 最顺手的东西放最近
- 锅具靠近灶具。
- 调料、油、锅铲靠近炒区。
- 餐具、杯具靠近洗碗机或清洗区回收路径。
- 清洁用品放洗区下方，不和食材混放。

### 2. 抽屉优先
- 厨房地柜更适合以抽屉为主，而不是全做平开门层板柜。
- 抽屉更符合看见、拿取、归位的日常节奏。
- 这会直接影响厨房是否容易维持整洁。

### 3. 开放与封闭
- 厨房更适合 `封闭为主，开放为辅`。
- 少量开放可以展示杯子、常用器具或轻小物。
- 但开放越多，视觉噪音和积灰压力越大。

## 家具与柜体搭配

### 1. 厨房不是客厅
- 厨房柜体要更强调效率、清洁和秩序。
- 不要为了做“好看展示”牺牲操作逻辑。

### 2. 柜体体块关系
- 地柜提供稳定感。
- 吊柜和高柜提供容量。
- 轻重关系要平衡，不要一进厨房就满眼都是厚重高柜。

### 3. 开放格怎么用
- 开放格只留给真正高频、真正好看的东西。
- 微波炉、咖啡机、杯子、少量托盘可以考虑。
- 不建议把大量杂物、调料瓶、塑料包装暴露在视觉中心。

## 颜色配比

### 1. 厨房整体基调
- 厨房更适合 `低饱和、干净、耐看` 的基调。
- 相比客厅，厨房可以略清爽一点；相比卫生间，厨房可以多一点温度。
- 重点不是做出“网红厨房”，而是每天进来都不觉得乱和压。

### 2. 柜体颜色
- 大面积柜体颜色应和全屋主砖、客餐厅柜体保持相容关系。
- 如果全屋已经有不少柜体，厨房不建议再用特别跳的颜色。
- 更稳的方向通常是：暖白、奶白、浅灰、浅米、低饱和灰绿或柔和木色点缀。

### 3. 台面颜色
- 台面最好比柜体略稳、略克制，不建议纹理过强。
- 厨房台面是高频看到和使用的面，越耐看越重要。
- 如果柜门颜色已经有存在感，台面就更应退后。

### 4. 墙地砖关系
- 厨房墙砖和地砖不需要抢戏，更适合作为干净背景。
- 墙面可偏明亮整洁，地面可略耐脏稳重。
- 只要与全屋主砖冷暖一致，厨房就不会从整体风格里跳出去。

## 比例和尺度

### 1. 台面比例
- 厨房视觉上最重要的是台面、柜门分缝和高柜比例。
- 台面被切得太碎，厨房会显乱。
- 电器和开孔太多集中在一个面，也会让视觉压力很大。

### 2. 高柜比例
- 高柜数量、厚度和位置要克制。
- 小厨房里高柜过多，会直接压缩呼吸感。
- 高柜更适合集中布置，而不是到处插入。

### 3. 吊柜比例
- 吊柜太厚会压。
- 吊柜太零碎会乱。
- 吊柜分段、门型和地柜的对应关系，比单独看某一扇门更重要。

## 视觉层次
- 柜体、台面、墙砖、地砖、五金、灯光需要清晰层次。
- 厨房如果每个面都很有存在感，就会显得很满。
- 好的厨房通常是：柜门整洁，台面克制，墙面干净，地面稳住。

## 灯光和氛围
- 厨房不是只有“亮”就够了。
- 要有基础照明，也要考虑操作面照明。
- 吊柜下、备菜区、洗区如果照明不足，会直接影响日常使用体验。
- 灯光色温不宜太冷，不然厨房会显得发硬；也不宜太黄，不然食物颜色和清洁感都打折。

## 和其他空间怎么协调

### 和客餐厅
- 如果餐厅紧邻厨房，厨房的颜色和体块不要比餐厅更重。
- 厨房是功能区，但视觉上仍然属于整个公共空间体系的一部分。

### 和瓷砖
- 厨房地墙砖应延续全屋的综合色调，不要突然换成完全不同的冷暖和纹理语言。
- 地砖更偏功能，墙砖更偏整洁背景。

### 和门
- 厨房推拉门、玻璃分格、门套收口会直接影响厨房的现代感和轻重感。
- 门做得太厚、太碎，会让厨房视觉更紧。

## 这个家更适合的厨房方向
- 更适合：`现代简洁 + 温和中性色 + 台面克制 + 收纳清楚 + 电器整合`。
- 不建议：大面积重色柜门、过多开放格、太强的装饰性护墙、很多小而碎的功能分区。
- 更适合做成一个“安静、顺手、长期能保持整洁”的厨房。

## 设计定稿清单
- [ ] 厨房已经明确洗、切、炒、洗碗、电器、收纳六个分区。
- [ ] 水槽、备菜区、灶具的关系顺手，不绕路。
- [ ] 连续备菜区已经保留下来，没有被小家电和开放格长期侵占。
- [ ] 电器高柜没有破坏主台面连续性，也没有把厨房压得太重。
- [ ] 地柜、吊柜、高柜和开放格的比例已经平衡。
- [ ] 抽屉和封闭收纳逻辑已经优先于展示逻辑。
- [ ] 柜体、台面、墙砖、地砖的颜色关系已统一。
- [ ] 厨房虽然是功能区，但没有从全屋风格里跳出去。
- [ ] 小家电、餐具、锅具、调料、清洁用品都有明确归位逻辑。
- [ ] 这个厨房看起来整洁，实际也能顺手做饭。
