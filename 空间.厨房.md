# 厨房
## 厨房内部分工
- 水槽区：水槽、普通冷热水龙头、独立净水小龙头、厨下净水器。
- 洗碗区：13套或15套嵌入式洗碗机，靠近水槽。
- 烹饪区：烟机、灶具、调料、锅具和常用油。
- 电器高柜：独立蒸箱 + 独立烤箱上下叠放。
- 微波炉位：独立微波炉，放厨房高柜中部开放格或台面外的固定开放格。
- 备菜区：保留连续台面，不能被小家电长期占用。
## 收纳分配
- 厨房收纳偏紧，只放高频烹饪用品和少量日常囤货。
- 水槽下方：净水器、管线、清洁剂和少量备用耗材，不放食品。
- 灶台下方：锅具、锅盖、炒锅、蒸锅、常用大件厨具。
- 灶台旁抽屉：调料、油盐酱醋、刀具、保鲜膜、厨房剪刀等高频物品。
- 备菜区下方：米桶、面粉、小米、杂粮、干货，优先用抽屉或拉篮。
- 吊柜：轻、不常用的餐具、烘焙工具、备用调料和轻质干货。
- 常温蔬菜：不放密闭柜，放厨房角落透气菜篮或透气拉篮。
- 食用油：常用小瓶放灶台旁抽屉或窄拉篮；大桶备用油放厨房下柜低处。
## 需要买的电器
- 洗碗机：独立嵌入式，选 `13套或15套`。
- 蒸箱：独立嵌入式，放厨房电器高柜。
- 烤箱：独立嵌入式，放厨房电器高柜。
- 微波炉：独立微波炉，放厨房固定开放格。
- 净水器：厨下RO反渗透，选 `800G-1000G` 无桶机。
- 前置过滤器：装入户位置，保护全屋用水设备。
- 烟机灶具：中餐为主，烟机吸力和排烟路径要优先。

## 必须确认
- 水槽、灶具、洗碗机、蒸箱、烤箱、微波炉、净水器位置和尺寸必须在水电前定稿或预留清楚。
- 橱柜板材、防潮、封边、台面、铰链、滑轨、拉篮和台面开孔必须写清。
- 厨房大功率电器必须有独立或分区回路，插座位置不能被柜体和电器挡住。
- 水槽柜、净水器、洗碗机、地漏、排水、防臭和检修口必须满足后期维修。
- 台面连续备菜区、油烟机排烟、燃气安全和散热空间必须满足。
- 电器都塞进厨房后，仍必须保留足够连续备菜区，不能只有纸面功能齐全。
- 油烟机止逆阀必须提前购买并在吊顶封闭前安装，品牌、材质、口径、密封和检修方式必须在对应下单文件中锁定。
- 油烟机灶具后方墙砖必须耐油污、易擦洗，不能使用表面粗糙、深纹理、难清洁墙砖。
- 燃气管不做私自暗埋，不封死在墙体、瓷砖、吊顶或固定柜体内；燃气阀门、燃气表和接头必须可见、可通风、可检修。
- 水管、下水、净水器管线可在水槽柜内整理隐藏，但必须保留检修空间，不得用固定板材封死。
## 橱柜和台面
- 橱柜按洗、切、炒、装盘顺序组织，保留连续备菜台面。
- 连续备菜区优先留在水槽和灶具之间，理想长度按 `90-120cm+` 复核；空间受限时也不要被小家电和开放格长期侵占。
- 台面优先石英石，厚度、品牌、颜色、挡水条和开孔做法必须在对应下单文件中锁定。
- 地柜多做抽屉和拉篮，少做普通层板柜。
- 高柜集中放蒸箱、烤箱、微波炉，散热孔和插座位置按设备说明书预留。
- 吊柜不到顶位置必须处理积灰问题，优先做到顶或做封板。
- 石英石台面厚度、垫条、前挡水、后挡水、台下盆加固、开孔和接缝位置必须写清。
- 石英石台面不能只看花色，必须写清品牌、系列、厚度、台下盆加固做法、灶具和水槽开孔加固、接缝位置和开裂质保。
- 台面接缝不得放在水槽、灶具开孔边缘和长期受力位置。
- 水槽台下安装必须有加固件和防霉密封，不只靠胶粘。
## 电器标准
- 洗碗机选 `13套或15套嵌入式`，靠近水槽，独立进水、排水、插座。
- 蒸箱、烤箱选独立嵌入式，不选蒸烤一体。
- 微波炉选独立微波炉，放固定开放格，不长期占用备菜台面。
- 净水器选厨下 `RO反渗透 800G-1000G无桶机`。
- 烟机优先大吸力、易清洁款，排烟管路径短、弯头少。
- 蒸箱、烤箱、微波炉必须按厂家说明书预留散热空间，背板不得完全封死。
- 洗碗机、净水器、垃圾处理器预留位必须能单独检修进水阀、排水管和电源。
- 油烟机止逆阀提前购买，品牌在 `潜水艇 / 壹品印象 / 老板原厂止逆阀 / 方太原厂止逆阀` 中选择。
- 止逆阀采用加厚阻燃 ABS 或不锈钢材质，带硅胶密封圈，口径与烟机烟管匹配，常用为 `180mm / 200mm`。
- 止逆阀位置必须保留检修条件，石膏板吊顶必须预留检修口，铝扣板吊顶必须能拆板检修。
## 水电和安全
- 水槽区必须有普通冷热水龙头和独立净水小龙头。
- 水槽柜内插座抬高，避开漏水风险。
- 蒸箱、烤箱、洗碗机按大功率设备配置专用插座，蒸箱和烤箱不共用一个普通插排。
- 燃气灶、烟机、燃气报警器位置提前确定。
- 厨房推拉门使用钢化玻璃，轨道、门套和地面收口提前确定。
- 厨房台面上方插座不少于 `6个`，分布在备菜区、小家电区和水吧区，不集中挤在一个角落。
- 蒸箱、烤箱、洗碗机、净水器、烟机、燃气报警器分别预留独立插座，插座不得位于设备正后方无法拔插的位置。
- 燃气管、燃气阀门、燃气表不得被柜体封死，必须留可开启检修门和通风孔。
- 燃气管不做墙体暗埋、地面暗埋、吊顶暗封和瓷砖封死处理。
- 燃气管需要弱化外观时，采用可拆卸百叶门、可开启柜门、通风孔或外露同色装饰遮挡，检修时能完整打开。
- 燃气软管、阀门、表具、接头位置必须外露可检修，不能穿过封闭柜体和密闭墙体。
- 水管可走顶、走墙或水槽柜内整理，水槽柜内冷热水阀、净水器、洗碗机进水阀和排水接口必须可检修。
- 下水管、洗碗机排水、净水器排水不得被固定背板封死，柜体开孔边必须封边或防潮处理。
- 洗碗机排水接入水槽下水或专用排水时必须做防臭、防倒灌处理。
- 厨房地漏、下水、止逆阀、防臭芯和检修口必须在贴砖和定制前确认。
## 热水器和燃气专项
- 厨房内涉及燃气表、燃气阀门、燃气管、热水器、排烟和橱柜避让的内容，必须作为厨房专项一部分统一确认，不放到施工中途再决定。
- 热水器方案必须在水电前锁定：容量、安装位置、是否零冷水、回水方案、排烟路径、检修空间和与橱柜、吊顶的避让。
- 两个卫生间 + 厨房存在同时用热水场景，热水器升数和最远端等待时间必须按实际使用场景评估，不只看促销型号。
- 采用零冷水时，必须确认回水管、回水泵噪音、耗气量、启停逻辑和后期维护，不默认零冷水一定更优。
- 热水器安装位置必须满足通风、排烟、检修和后期更换条件，不得被固定柜体、固定吊顶或燃气管遮挡。
- 燃气表、阀门、波纹管、接头、排烟接口位置必须可见、可检修、可拆卸，不得封死在吊顶、瓷砖或固定柜体内。
- 热水器、燃气报警器、止回阀、检修门、百叶门和橱柜开孔边界必须提前写清，避免柜体下单后才发现避让不足。
## 收纳标准
- 米、面、杂粮放备菜区下方抽屉或拉篮。
- 常用食用油放灶台旁窄拉篮，大桶备用油放地柜低处。
- 常温蔬菜放透气菜篮或透气拉篮，不放密闭柜。
- 清洁剂只放水槽下方，不和食品混放。
## 材料与选型
- 橱柜柜体使用 `ENF级防潮板 / ENF级多层板`，柜体侧板、层板、门板不低于 `18mm`。
- 橱柜封边采用 `PUR封边` 或 `激光封边`，水槽柜、洗碗机位、蒸烤箱位开孔边必须做防潮处理。
- 高频柜门使用 `百隆 CLIP top BLUMOTION / 海蒂诗 Sensys 8645i / DTC三段力阻尼铰链`。
- 抽屉使用 `百隆 TANDEM/MOVENTO / 海蒂诗 Quadro V6/Actro 5D / DTC 40kg/60kg隐藏阻尼滑轨`。
- 橱柜免拉手采用门板下延、45度斜切或 G 型拉手槽，不用高频反弹器和自动开合器。
- 台面使用 `赛凯隆 / 戈兰迪 / 中迅` 石英石台面，厚度 `20mm`，水槽和灶具开孔周边加固。
- 水槽使用 `摩恩 / 科勒 / 普乐美 / 欧琳` 的 `SUS304不锈钢手工单槽`，厚度 `0.8-1.2mm`，深度不低于 `20cm`。
- 厨房龙头使用 `摩恩 / 科勒 / 九牧` 抽拉式厨房龙头，采用陶瓷阀芯。
- 厨房推拉门和固定玻璃必须使用钢化安全玻璃，玻璃厚度、五金、轨道和防撞缓冲必须在对应下单文件中锁定。
- 厨房水管使用 `PPR S3.2 / PN2.0` 热水管和 `S4 / PN1.6` 冷水管。
- 开关插座采用 `西门子远景Pro / 施耐德绎尚 / 罗格朗逸远`，蒸箱、烤箱、洗碗机按大功率设备配置专用插座。
- 厨房地面固定使用 `300x600mm` 防滑砖；厨房墙面固定使用 `400x800mm` 易清洁釉面墙砖。
- 厨房墙砖固定 `400x800mm`，油烟机和灶具后方使用亮面或柔光釉面砖，不使用凹凸深纹、仿古粗面、小马赛克。
- 油烟机背后、灶台两侧和备菜区墙面美缝使用防霉耐油污材料，颜色接近墙砖，减少油污显脏。
- 厨房地面和有水区域防水材料在 `东方雨虹雨虹200 / 德高K11 / 西卡家装柔性防水` 中选择。
## 需要提前定好尺寸
- 洗碗机：宽、高、深，决定橱柜、水电、进排水位置。
- 蒸箱和烤箱：具体型号、开孔尺寸、散热要求、插座位置和功率。
- 微波炉：尺寸、散热空间、开放格宽深高。
- 水槽：单槽/双槽、台上/台中/台下、龙头孔数量。
- 净水器：主机尺寸、滤芯更换空间、净水龙头开孔位置。
- 烟机灶具：烟机尺寸、灶具开孔、烟道位置和排烟管路径。
- 油烟机止逆阀：品牌、口径、安装位置、检修口和密封方式。
- 油烟机后方墙砖：规格、铺贴高度、烟机挂板位置和易清洁收口。
- 燃气管：燃气表、阀门、接头、管线走向、可拆卸遮挡和通风检修方式。
- 热水器：容量、宽深高、排烟管径、回水方案、检修空间和安装高度。
- 厨房推拉门：门洞宽度、高度、轨道方式和玻璃参数。
## 水电预留
- 水槽区保留两个出水口：普通冷热水龙头 + 独立净水小龙头。
- 水槽柜内预留净水器插座、净水器进出水、洗碗机进排水。
- 洗碗机、蒸箱、烤箱、微波炉都要单独预留插座；蒸箱和烤箱按大功率电器单独配置回路。
- 厨房电器高柜要预留散热通道，不能完全封死。
- 插座不要放在容易漏水的位置，水槽柜内插座要抬高。
- 燃气报警器预留电源，位置按燃气类型和厂家要求确定。
- 燃气阀门附近不布置遮挡检修的固定层板和固定背板。
- 热水器电源、燃气、冷热水、回水管条件和排烟孔位必须在水电和吊顶前统一确认。
## 复尺时重点确认
- 厨房净宽、净深，确认洗碗机、电器高柜和备菜台能否同时保证。
- 厨房门洞能否扩大、扩大后的净宽、墙体性质、烟道和燃气表是否受影响。
- 厨房门洞扩大条件和可拆改墙体范围。
- 烟道、油烟机止逆阀、燃气管、下水、地漏和窗户位置。
- 热水器安装位、排烟孔位、燃气表阀门、百叶检修门和橱柜避让关系。
- 燃气管外露、可拆卸遮挡、通风孔和检修门位置。
- 厨房内放蒸箱、烤箱、微波炉后，剩余台面和收纳满足日常使用，不能只做到“机器都塞进去”。
- 蒸箱、烤箱高度不能太高，端热盘要安全。
