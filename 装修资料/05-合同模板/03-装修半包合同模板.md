# 装修半包合同模板

打印和签约时，正文及连续装订页逐页签字或盖章，空白处划线作废。

## 签约信息
- 甲方：本房屋业主
- 乙方：半包施工单位工商全称
- 项目地址：本项目房屋
- 联系人：甲方本人；乙方项目负责人、施工负责人、现场工长、售后负责人
- 联系电话：签约时填写
- 预估签约报价：`9.5万` 合理口径；高配口径约 `12.1-17万`
- 合同总价：按半包报价汇总填写大写和小写金额
- 收款账户：乙方对公账户全称、开户行、账号
- 开竣工日期：签约时填写
- 备注：除上述签约信息、总价和开竣工日期外，其余范围边界、材料、工艺、验收和付款条款按本合同固定执行。

## 适用范围
- 适用于本项目半包施工，即乙方负责人工和合同约定辅材，甲方负责主材及专项设备采购。

## 本项目签约底线
- 半包必须把人工、辅材、工艺、工程量、单价、总价全部拆开写清。
- 甲方自购主材，乙方负责提前提资、现场配合、尺寸复核、成品保护和安装衔接。
- 辅材品牌型号必须锁定，不接受 `同级材料 / 品牌同档次 / 按公司标准`。
- 隐蔽工程未验收前，不得封槽、封板、回填、贴砖。
- 在无甲方主动变更的情况下，竣工结算增幅不超过签约预算总价的 `5%`，含隐蔽工程和按实结算项目。
- 付款必须按节点验收走，尾款保留 `5%-10%`。

## 合同模板

### 1. 合同主体与承包模式
- 甲方为本房屋业主，乙方为半包施工单位。
- 乙方必须提供营业执照、收款账户、项目负责人、施工负责人和实际施工队长信息；合同、报价单、补充协议和收款账户主体必须一致。
- 本合同承包模式为半包，即乙方负责人工和合同约定辅材，甲方负责主材及专项设备采购。
- 合同总价以《半包报价清单》《施工项目清单》《甲供乙供边界表》《施工图纸和点位图》《专项交底与提资清单》为准。
- 未写入合同清单的必要施工项，若属于乙方正常施工范围，不得在施工中再追加为“漏项增项”。
- 甲方据此签约的口头承诺、赠送项目和免费升级项目，必须同步写入合同或附件，否则视为未承诺。

### 2. 乙方施工范围
- 乙方负责拆除、砌筑修补、水路、电路、弱电预埋、防水、闭水、地面找平、瓷砖铺贴人工及辅材、吊顶、石膏板、油工、局部安装收口、成品保护、垃圾清运和现场管理。
- 乙方负责与封窗、中央空调、地暖、瓷砖、全屋定制、室内门、洁具电器等专项商家做工序配合。
- 乙方负责提供施工进度计划、材料进场计划和甲方需提前采购的主材清单。
- 乙方施工范围默认包含本项目正常落地所必需的基层处理、洞口修补、封槽回填、局部找平、包管隔音、防水加强、地漏坡度处理、检修口预留、成品保护和收口，不得以“报价习惯未含”为由后补收费。
- 乙方在开工前应完成对厨房燃气热水器、止逆阀、浴霸、洗烘机、扫地机器人基站、冰箱、蒸箱、烤箱、洗碗机、净水器、中央空调、地暖和全屋定制的接口条件复核；未书面提出冲突意见的，视为乙方已审核并接受当前方案。

### 3. 甲供乙供边界
- 甲方自购主材包括但不限于：瓷砖主材、美缝材料、封窗、中央空调、室内门、卫浴洁具、厨房电器、开关面板、灯具、地漏、角阀、软管、全屋定制、五金成品件。
- 乙方自带辅材包括但不限于：水管、电线、网线、线管、底盒、防水、瓷砖胶、背胶、水泥、沙子、石膏板、龙骨、腻子、乳胶漆、玻璃胶、发泡胶、收口辅材。
- 乙方必须在开工前明确哪些甲供材料需要在水电前、贴砖前、吊顶前、木作前提前到场。
- 乙方未提前书面提资，导致返工、延期或现场临时加价的，由乙方承担责任。
- 甲方材料到场后，乙方必须按品牌、型号、规格、数量和外观进行共同验收并书面签收；签收后因乙方保管、搬运、施工不当造成的损坏、丢失或污染，由乙方负责赔偿或更换。
- 专项商家进场前，乙方必须提前完成基层、点位、洞口、标高、检修口、排水、电源和收口条件；因乙方基础条件不到位导致专项商家空跑、返工或重复上门费用的，由乙方承担。

### 4. 报价清单要求
- 报价必须按阶段拆分：拆改砌筑、水路、电路、弱电、防水、瓦工、吊顶、油工、安装收口、成品保护、垃圾清运。
- 每一项必须列工程量、单位、单价、总价和计价规则。
- 水电不得只写“按实际结算”而不写点位单价、米数单价、封槽单价、回路单价和增减规则。
- 找平、挂网、界面剂、阴阳角找直、防水增刷、海棠角、包管隔音、检修口、窗帘盒、灯槽、垃圾清运必须在签约前写清是否包含。
- 水电按实结算必须同时写清：点位定义、米数起止口径、回路定义、强弱电箱是否含扩容和整理、空开漏保品牌型号、底盒数量口径、开槽封槽是否重复计费、横平竖直是否单独加价、厨房卫浴阳台大功率位是否另算，以及最终结算上限。
- 乙方在签约前已现场复尺、看房并完成预算，后续因乙方漏量、漏报、漏工序导致的必要费用，不得转嫁给甲方。
- 未明确单价的增项，默认不予执行。

### 5. 材料品牌型号条款
- 水管在 `伟星 / 日丰 / 金牛 / 保利 / 联塑` 中锁定品牌和型号，主水路按 `PPR S3.2 / PN2.0热水管` 执行。
- 电线在 `远东 / 上上 / 熊猫` 中锁定品牌和规格，必须为国标铜芯线。
- 网线按 `六类无氧铜 Cat6` 执行，不得使用铜包铝。
- 线管、底盒、强弱电箱、空开、漏保、接线端子等电气辅材必须锁定品牌和规格，不接受工人自带散料。
- 防水在 `东方雨虹 / 德高 / 西卡 / 立邦 / 三棵树` 中锁定品牌和型号。
- 瓷砖胶、背胶在 `德高 / 马贝 / 西卡 / 东方雨虹` 中锁定品牌和型号。
- 水泥、界面剂、嵌缝材料、网格布、玻璃胶、发泡胶均应写清品牌、型号和适用位置。
- 石膏板、龙骨在 `龙牌 / 泰山 / 可耐福` 中锁定品牌和系列。
- 腻子在 `美巢 / 立邦 / 圣戈班` 中锁定，乳胶漆在 `立邦 / 多乐士 / 三棵树 / 芬琳` 中锁定具体系列。
- 厨卫收口玻璃胶必须使用防霉中性胶，不得使用工人自带杂牌胶。
- 材料进场时乙方必须配合甲方核对外包装标签、型号、批次、防伪和合格资料；未经验收的材料不得拆包上墙上地。

### 6. 施工工艺条款
- 水路完工后必须打压，压力按 `0.8MPa` 执行，稳压不少于 `30分钟`，压降不超过 `0.05MPa`。
- 电路完工后必须做通断测试、绝缘测试、漏保测试和插座极性测试，测试结果拍照或视频留档；配电箱回路标签必须和实际房间、设备一一对应。
- 强弱电分槽分管，线管内不得有接头；厨房、卫生间、阳台大功率回路单独配置；冰箱、蒸箱、烤箱、洗碗机、洗衣机、烘干机、中央空调外机按功率独立或分区回路配置。
- 卫生间地面防水满刷，淋浴区墙面不低于 `1.8m`，其他墙面不低于 `30cm`，台盆背墙不低于 `1.2m`。
- 防水施工前基层必须清理，阴阳角、管根、门口、地漏先做加强；卫生间闭水不少于 `48小时`，阳台闭水或淋水不少于 `24小时`，楼下同步查验无渗漏后方可贴砖。
- 瓷砖铺贴前必须排版，显眼位置不得出现小窄条；大规格砖和低吸水率砖必须使用匹配等级的 `瓷砖胶 + 背胶`；卫生间和阳台地面必须做坡向地漏，不得倒坡积水；卫生间墙地交界按本项目 `墙压地` 做法执行。
- 卫生间外阳角、墙面可视阳角按本项目要求执行 `海棠角`；倒角必须顺直、不崩边、不割手。
- 中央空调、地暖、封窗等专项未完成对应隐蔽验收前，乙方不得擅自封板或进入不可逆工序。
- 吊顶封板前必须确认空调检修口、阀门、风口、灯位、窗帘盒、柜体关系；重型灯具、电器和风口不得直接固定在石膏板上，必须有独立加固。
- 石膏板接缝留缝、嵌缝、贴防裂带；阴阳角找直；乳胶漆按 `一底两面` 执行。
- 燃气管、燃气表、燃气阀门不得暗埋，不得封死在墙体、吊顶、瓷砖或固定柜体内，必须保持可见、可通风、可检修。

### 7. 工期、验收与资料留档
- 乙方开工前必须提供阶段工期表，写清每一阶段预计开始和完成时间。
- 因乙方擅自停工、消极施工、跨工地调人、材料不合格返工、专项配合失误造成工期延误的，工期不顺延。
- 乙方逾期完工的，每延误 `1日` 按合同总价的 `0.03%` 向甲方支付违约金，累计不超过合同总价的 `5%`。
- 每个关键节点必须做验收，未验收不得进入下一工序。
- 关键节点至少包括：水电隐蔽、防水闭水、中央空调隐蔽配合、地暖配合、瓦工铺贴、吊顶封板前、油工基层、竣工交付。
- 乙方应至少提前 `3日` 书面通知甲方或监理进行材料验收、隐蔽验收和阶段验收；未经通知擅自进入下道工序的，甲方有权要求拆除或返工。
- 乙方必须交付隐蔽工程照片、测试照片或视频、材料包装与防伪照片、回路表、材料验收单、增项单、材料替换单和售后联系人。
- 竣工时乙方必须交付水电定位图或红线图、主要隐蔽走向图、配电回路表、闭水和打压记录、整改闭环清单。

### 8. 付款、停工与违约
- 所有付款均支付至合同约定的乙方对公账户。
- 签约定金或首期预付款累计不超过半包合同总价 `10%`。
- 水电材料进场并核对品牌型号后支付第一阶段款，累计付款不超过合同总价 `30%`。
- 水电、防水、专项隐蔽配合验收合格后支付第二阶段款，累计付款不超过合同总价 `55%`。
- 瓦工验收合格后支付第三阶段款，累计付款不超过合同总价 `75%`。
- 吊顶、油工验收合格后支付第四阶段款，累计付款不超过合同总价 `90%`。
- 竣工验收、整改完成、资料交付后支付主要尾款，累计付款不超过合同总价 `95%`。
- 保留合同总价 `5%-10%` 作为质保尾款。
- 出现材料品牌型号与合同不符、隐蔽工程未验收即封闭、测试未做、图纸未签字即进入不可逆工序、整改逾期未完成等情形时，甲方有权要求停工并暂停对应节点付款。
- 增项必须在施工前确认；未经甲方书面确认，乙方不得擅自变更材料、工艺或工程量。
