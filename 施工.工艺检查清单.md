# 02 施工工艺现场打勾版

## 1. 开工前先卡死
- [ ] 半包范围已拆开写：拆改、水电、防水、瓦工、吊顶、油工、收口、清运、现场管理
- [ ] 所有辅材都写了 `品牌 + 系列 + 型号 + 规格`
- [ ] 付款节点按验收走，尾款保留 `5%-10%`
- [ ] 没有出现 `同级材料 / 按现场实际 / 按公司标准 / 常规施工 / 后期再定`
- [ ] 图纸未签字不下单已明确

## 2. 水电阶段
- [ ] 水管为 `PPR S3.2 / PN2.0热水管`
- [ ] 电线平方数符合方案：照明 `1.5`、普通插座 `2.5`、大功率 `4` 起
- [ ] 冰箱、蒸箱、烤箱、洗碗机、洗烘、空调回路已划分
- [ ] 强弱电分管分槽
- [ ] 线管内无接头、无死弯
- [ ] 书房、儿童房、电视墙网口已预留
- [ ] 厨卫阳台湿区插座带漏保和防溅
- [ ] 水路打压 `0.8MPa / 30分钟 / 压降不超过 0.05MPa`
- [ ] 电路通断、绝缘、漏保、极性测试已做
- [ ] 隐蔽照片已拍齐

## 3. 防水和排水
- [ ] 卫生间地面满刷
- [ ] 淋浴区墙面防水 `1.8米+`
- [ ] 其他墙面防水 `30cm+`
- [ ] 台盆背墙防水 `1.2米+`
- [ ] 阳台地面满刷，墙面上翻 `30cm+`
- [ ] 管根、地漏、门口、阴阳角加强已做
- [ ] 闭水 `48小时` 已做
- [ ] 楼下检查无渗漏
- [ ] 卫生间、阳台坡向地漏，无倒坡、积水
- [ ] 下水包管隔音和检修口已做

## 4. 书房新增一路地暖

- [ ] 地暖管规格为 `16x2.0mm`
- [ ] 中间无接头

- [ ] 固定柜体下方未铺管
- [ ] 保温板 `20mm+`，边界保温条 `8mm+`
- [ ] 第一次打压 `0.8MPa / 30分钟`
- [ ] `24小时` 保压已做
- [ ] 回填期间 `0.4-0.6MPa` 带压施工

## 5. 中央空调
- [ ] 内机、外机完整型号与合同一致
- [ ] 条码、合格证、厂家报备口径与合同一致
- [ ] 铜管品牌、管径、壁厚与合同一致
- [ ] `6.35 / 9.52 / 12.7` 铜管壁厚不低于 `0.8mm`
- [ ] `15.88 / 19.05` 铜管壁厚不低于 `1.0mm`
- [ ] 保温棉 `B1级闭孔橡塑`，厚度 `15mm+`
- [ ] 易结露位保温厚度 `20mm`
- [ ] 冷凝水管为 `UPVC硬管`，单台内机 `DN25+`
- [ ] 冷凝水坡度 `1/100+`，无倒坡
- [ ] 冷凝水固定支撑间距 `1.2m` 内
- [ ] 冷凝水末端排水、防臭、防返水做法已按方案执行
- [ ] 焊接有氮气保护焊
- [ ] 每台内机有检修口，独立检修口净尺寸 `400x400mm+`
- [ ] 回风口兼检修口时净尺寸满足方案
- [ ] 回风不被柜体、窗帘盒、灯槽挡死
- [ ] 保压 `4.15MPa / 24小时` 合格
- [ ] 抽真空 `60分钟+`
- [ ] 外机散热、排风、回风、顶距和检修空间合格
- [ ] 每台内机通水测试已做
- [ ] 每台内机试运行 `30分钟+`
