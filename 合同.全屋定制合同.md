## 本项目签约底线
- 合同必须逐项写清 `品牌 + 系列 + 基材 + 环保等级 + 厚度 + 饰面 + 颜色 + 封边工艺 + 五金品牌型号 + 数量`。
- 干区柜体、厨房柜体、阳台柜体、浴室柜柜体、柜门、背板、书桌板、台面必须分开写，不能只写“ENF板材”或“同级板材”。
- 柜体侧板、层板、门板不低于 `18mm`，背板不低于 `9mm`；长跨层板、悬空书桌、高门板、超高柜必须单独写结构加固方案。
- 书桌桌板默认不低于 `25mm`；如做 `18mm` 桌板，必须单列加固方式、支撑结构和承重口径。
- 封边锁定 `豪迈PUR封边`，开孔边、切割边和潮湿区裸露边必须做封闭防潮处理。
- 五金锁定 `百隆铰链 + 百隆滑轨 + 悍高拉篮 + 欧普灯带`，不接受 `品牌五金 / 套餐五金 / 同档次五金`。
- 厨房、阳台柜体锁定 `兔宝宝 ENF级桉芯多层板`；普通干区柜体锁定 `兔宝宝 ENF级双饰面实木颗粒板`。
- 浴室柜锁定 `蜂窝铝柜体 + PET哑光柜门`；如改用板式浴室柜，必须单列基材、厚度、封边和开孔边防潮做法。
- 厨房和卫生间涉及台面、水槽、台盆、上下水、电器和阀门避让的，必须在下单图纸里标清，不允许安装时现场随意改。

## 关键分项必须填写的字段

### 1. 厨房柜体和电器高柜
- 常规报价口径先按 `地柜深度600mm / 吊柜深度350mm / 高柜深度600mm` 询价，超深、超高、异形转角单独列价。
- 必须逐项写清：`地柜长度/深度/高度`、`吊柜长度/深度/高度`、`高柜长度/深度/高度`、`台面品牌/系列/厚度/颜色`、`挡水形式`、`台下盆或台中盆做法`、`开孔尺寸`、`燃气表/阀门/烟管/止逆阀避让`、`洗碗机/蒸箱/烤箱/微波炉/冰箱位净尺寸`、`见光板/封板/收口条`。
- 柜体必须分开写：`柜体板`、`门板`、`背板`、`层板`、`抽屉板`、`拉篮`、`铰链`、`滑轨`、`拉手`、`踢脚板`。
- 水槽柜、洗碗机位、垃圾处理器位、净水器位必须写清：`防潮处理`、`检修空间`、`上下水位置`、`电源位置`。

### 2. 浴室柜和镜柜

## 合同模板

### 1. 合同主体与项目概况

### 2. 报价与计价条款

- 每个空间必须列明柜体尺寸、计价方式、单价、数量、总价和计价规则。
- 柜体板、门板、背板、书桌板、台面、抽屉、拉篮、玻璃门、见光板、封板、收口、灯带、变压器、安装费、搬运费、上楼费必须分项写清。
- 投影面积和展开面积不得混用；同一柜体采用哪一种计价方式必须写死，封板、见光板、顶封板、侧封板、收口条、拉直器、加固件不得重复计费。
- 异形柜、加厚板、超高柜、特殊开孔、台下盆、拉篮、抽屉、玻璃门、灯带、感应器、拉直器、加固件、台面异形切割必须提前列单价。

### 3. 材料与五金条款
- 附件A《默认材料与报价口径》与报价清单、签字图纸具有同等效力；除甲方书面同意外，乙方不得以“同级替代”方式更换品牌、基材、厚度、封边或五金型号。
- 合同必须逐项写清板材品牌、系列、基材类型、环保等级、厚度、颜色、饰面和封边工艺。
- 柜门必须写清材质、厚度、颜色、封边方式，以及是否需要拉直器或加强筋。
- 背板必须写清厚度和材质，不接受“标配背板”“同级背板”。
- 书桌板必须写清环保等级、厚度、跨度、支撑方式和收边方式；悬空书桌必须写清固定方式和承重方案。
- 板材进场必须能对应到合同清单、包装标识、批次信息和检测资料；甲方在安装前有权按包装、标签和清单核验，不符可拒装。

- 铰链、滑轨、拉篮、反弹器、上翻支撑、抽屉侧板、挂衣杆、裤架、感应灯配套件必须写清品牌、系列、型号、数量。

- 滑轨必须锁到具体系列型号和承重等级，不接受 `品牌同级 / 工程滑轨 / 套餐滑轨 / 品牌代工款` 这类替代口径。
- 重物抽屉、锅具抽屉、宽抽屉、深抽屉必须单独写清滑轨承重等级、抽屉净宽和是否需要加强件。
- 滑轨到货验收以钢印、包装型号、承重标识和数量为准；未写清型号和承重的抽屉，默认不得下单。

### 5. 柜体结构与专项做法条款
- 衣柜必须标明长衣区、短衣区、抽屉区、被褥区、行李箱区的内部净尺寸，不能只画外轮廓。

- 电视柜设备区必须可散热、可走线、可检修，不能做成封死柜。
- 书桌、悬空台面、长跨层板、高门板、薄门板、深色门板、肤感门板等易变形部位，乙方必须在图纸和报价中明确防变形方案；未写明的，视为乙方已包含满足正常使用要求的加固方案。

- 厨房、阳台、卫生间柜体开孔边、切割边、靠墙收口边必须做防潮封闭，水槽柜、洗碗机位、台盆柜重点加强。

### 6. 安装、验收与付款
- 安装前核对板材、门板、背板、五金、台面、拉篮、灯带、变压器与签字图纸一致。
