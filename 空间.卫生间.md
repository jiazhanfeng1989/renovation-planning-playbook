# 卫生间
## 主卫设计
- 淋浴区：做独立淋浴区，优先玻璃隔断或淋浴房。
- 马桶区：确认坑距和排水方式，采用普通虹吸马桶，不配置智能马桶。
- 浴室柜：带镜柜，增加洗漱和护肤品收纳。
- 收纳：洗漱用品、纸巾、清洁用品分区放置。
- 灯光：镜前灯 + 顶灯，避免只靠顶灯导致脸部阴影。
## 公卫设计
- 干湿分离：优先保证淋浴区独立，减少地面积水。
- 儿童安全：地面防滑，转角少，玻璃必须钢化并贴防爆膜。
- 马桶：确认坑距，采用普通虹吸马桶，不配置智能马桶。
- 浴室柜：高度和台面兼顾孩子使用，配置脚凳。
- 夜间照明：预留低位感应灯或柔和夜灯。
## 防水和排水
- 淋浴区墙面防水高度要足够。
- 地漏位置、排水方向和坡度要提前定清，不能积水。
- 门口要处理好挡水和门槛，避免水外溢。
- 下水、地漏、马桶移位严格控制，减少堵塞和返味风险。
## 材料和五金
- 地砖优先防滑、耐脏、易清洁。
- 墙砖优先较大规格、简洁耐看，不用复杂花砖。
- 淋浴房五金、合页、拉手要选耐用款。
- 地漏、角阀、软管、止逆、防臭要买好一点。
- 玻璃隔断必须钢化玻璃，并贴防爆膜。
## 瓷砖尺寸结论
- 两个卫生间墙面更建议走大砖方向，效果更完整、缝更少、后期更好打理。
- 但卫生间地面不要只为了少缝盲目上超大砖。
- 本项目最终按 `普通点位地漏` 执行，不做线性长条地漏。
- 因为普通点位地漏需要四周找坡回水，地面太大不利于找坡，后期更容易积水、倒坡和局部空鼓。
- 所以这套房卫生间最终不走“超大砖地面效果优先”路线，而走“墙面大砖 + 地面中小规格防滑砖”的稳妥路线。
- `漏不漏水` 主要看防水层、闭水、地漏和收口，不是看砖缝多少。
- 墙面外阳角固定做 `海棠角`。
- 墙地交界固定按 `墙压地 + 柔性胶缝` 做法执行。
## 墙压地和地压墙做法
- 本项目最终选 `墙压地`，不选 `地压墙`。
- 观感、打理和普通点位地漏的排水逻辑更适合这套方案。
## 海棠角最终范围
- 本项目卫生间默认 `显眼外阳角尽量都做海棠角`。
- 但不是所有位置都适合硬做海棠角，功能位和高碰撞位要让位给耐久性。
- 卫生间必须做海棠角的位置：
- 墙面外阳角
- 门洞包边显眼外角
- 包管外阳角
- 壁龛外口
- 不建议强行做海棠角的位置：
- 地漏四周
- 地面低位尖角
- 门后、脚边、儿童高频碰撞位
- 后期要拆装的检修口边
## 必须确认
- 防水、闭水、排水坡度、防臭、地漏、角阀、软管、淋浴房五金和浴霸安全必须满足。
- 两个卫生间必须完成干湿分离、防滑、通风、照明和漏电保护。
- 马桶采用普通虹吸马桶，坑距、法兰、冲水、防晃、防渗漏和防返味要求必须提前锁定。
- 两个马桶旁各预留一个增压喷枪，供清洗马桶和地面边角使用。
- 卫生间下水管必须做隔音包管，减少楼上冲水和排水噪音。
- 淋浴房玻璃必须钢化并带 `3C` 标识，五金固定必须牢固。
- 卫生间所有插座必须有漏电保护，防溅盒安装牢固。
## 防水标准
- 两个卫生间地面满刷防水。
- 淋浴区墙面防水高度不低于 `1.8米`。
- 非淋浴墙面防水高度不低于 `30cm`。
- 洗手盆背墙防水高度不低于 `1.2米`。
- 管根、地漏、门口、阴阳角必须做加强防水。
- 闭水试验按 `48小时` 预留。
## 排水和防臭标准
- 地漏优先选择防臭地漏，淋浴区排水快、不积水。
- 地面坡向地漏，不能倒坡。
- 马桶坑距复尺确认后再定马桶。
- 不随意大距离移位马桶，减少堵塞和返味风险。
- 台盆下水、地漏、洗衣相关排水必须做防臭处理。
- 台盆下水使用带存水弯或防臭结构。
- 干区和湿区地面高差、挡水条或淋浴房挡水必须能阻止洗澡水外溢。
- 马桶喷枪使用独立角阀或一进二出角阀，不和马桶进水软管硬挤共用一个无检修三通。
- 喷枪挂架位置不影响如厕和开门。
## 安全和材料标准
- 地砖必须选防滑砖，公卫防滑优先级最高。
- 淋浴房或玻璃隔断必须使用钢化玻璃，并贴防爆膜。
- 淋浴房五金、合页、拉手选择耐腐蚀耐用款。
- 角阀、软管、地漏、止逆阀、防臭芯选品牌产品，不能用装修公司杂牌赠品。
- 马桶喷枪选择增压手持喷枪，材质采用 `304不锈钢 / 全铜镀铬 / 优质ABS电镀`，喷头带开关，软管采用 `304不锈钢编织软管`。
- 马桶喷枪品牌在 `潜水艇 / 九牧 / 箭牌 / 科勒 / 摩恩` 中选择。
- 马桶采用普通虹吸马桶或旋涡虹吸马桶，品牌在 `TOTO / 科勒 / 恒洁 / 九牧` 中选择。
- 马桶重点看釉面、冲水、静音、防溅、防臭和法兰密封，不买智能款。
- 高层和水压不稳定场景选择带水箱马桶，不选无水箱智能马桶。
- 马桶下单前必须按现场坑距选择 `305mm / 350mm / 400mm` 对应型号。
- 公卫预留夜灯或低位感应灯，方便孩子夜间使用。
- 淋浴房玻璃必须有 `3C` 标识，五金固定牢固，开门不得撞马桶、浴室柜或墙面。
- 公卫淋浴区和马桶区预留可加装扶手的墙面条件，方便儿童和老人临时使用。
## 电气和通风标准
- 马桶区不按智能马桶配置。
- 镜柜内或旁边预留电动牙刷、剃须刀、吹风机插座。
- 浴霸、排风、照明、镜前灯分开控制。
- 卫生间插座避开淋浴直溅区。
- 排风或浴霸排气路径必须通畅，止逆措施到位。
- 卫生间所有插座必须有漏电保护，防溅盒安装牢固。
- 镜柜电源、灯线、浴霸线路必须留检修空间，不得封死在瓷砖或柜体后。
## 材料与选型
- 防水材料在 `东方雨虹雨虹200 / 德高K11 / 西卡家装柔性防水` 中选择。
- 卫生间地砖使用防滑砖，淋浴区不使用亮面砖。
- 墙砖优先使用 `400x800mm / 600x1200mm` 易清洁大砖，不使用复杂花砖。
- 本项目采用普通点位地漏，卫生间地面优先 `300x300mm / 300x600mm` 防滑砖。
- 如果希望视觉更完整，优先选 `300x600mm`，不要上 `750x1500mm` 强行切坡。
- 淋浴区地面必须以排水、防滑、脚感和施工稳定性优先，不以少缝为第一目标。
- 墙面外阳角采用海棠角工艺，要求顺直、细腻、不崩边、不割手。
- 墙地阴角采用 `墙压地 + 防霉中性硅酮胶柔性收口`，不做硬性美缝顶死。
- 普通虹吸马桶在 `TOTO / 科勒 / 恒洁 / 九牧` 中选择，按现场坑距选 `305mm / 350mm / 400mm`。
- 花洒在 `汉斯格雅 Raindance Select E 120 / 高仪恒温花洒系列 / 摩恩恒温花洒系列` 中选择。
- 地漏在 `潜水艇 / 九牧 / 科勒` 防臭地漏中选择。
- 角阀在 `潜水艇 / 九牧 / 摩恩` 中选择。
- 软管采用 `304不锈钢编织软管`。
- 每个卫生间马桶旁配置 `1套增压喷枪 + 1个独立角阀或一进二出角阀 + 1根304不锈钢编织软管`。
- 卫生间包管采用 `阻尼隔音毡 / 橡塑隔音棉 + 砖砌包管 + 抹灰贴砖` 做法，检修位置必须留检修口。
- 浴室柜使用 `ENF级防潮板 / ENF级多层板 / 铝合金柜体`，封边和开孔边必须防潮。
- 浴室柜铰链和抽屉滑轨使用阻尼五金，不用高频反弹器和自动开合器。
- 开关插座采用 `西门子远景Pro / 施耐德绎尚 / 罗格朗逸远`，卫生间插座必须带防溅盒和漏电保护。
- 淋浴房玻璃必须钢化并带 `3C` 标识，五金使用 `304不锈钢` 或高品质铜件。
## 水电预留
- 镜柜电源，给电动牙刷、剃须刀、吹风机使用。
- 浴室柜下方冷热水和排水。
- 马桶旁增压喷枪冷水点位或一进二出角阀安装空间。
- 排风或浴霸电源。
- 夜灯或感应灯电源。
## 需要提前定好尺寸
- 马桶坑距和马桶尺寸。
- 浴室柜长度、深度、高度。
- 淋浴房尺寸和开门方式。
- 地漏位置、排水方向和坡度。
- 镜柜尺寸、插座位置和灯光位置。
- 马桶喷枪角阀、挂架和软管活动范围。
- 包管尺寸、检修口位置和贴砖收口方式。
