# 施工监管
## 总体判断
- 这套房更建议请第三方监理。
- 原因不是“装修公司一定不行”，而是你家有 `中央空调 + 书房补地暖 + 全屋封窗 + 全屋瓷砖 + 全屋定制 + 双卫 + 厨房燃气热水`，交叉工种多，隐蔽工程多，只靠装修公司自检风险偏高。
- 但如果最终不请第三方监理，也不是不能做，只是你自己要承担更多节点管理、验收、拍照留档和暂停付款的责任。

## A. 如果找第三方监理，需要做什么
### 1. 监理要管什么
- 报价审核
- 合同陪签
- 水电定位复核
- 材料进场核对
- 隐蔽工程验收
- 阶段验收
- 整改复验
- 付款放行建议

### 2. 必须覆盖的关键节点
- 开工交底
- 水电定位
- 水电验收
- 防水闭水验收
- 书房新增一路地暖铺设和打压
- 中央空调管线封板前检查
- 封窗安装和淋水复验
- 瓦工验收
- 吊顶/木作验收
- 油工验收
- 定制复尺或下单前图纸复核
- 定制安装验收
- 竣工验收

### 3. 监理服务最低标准
- 每次上门必须出书面或电子报告。
- 报告必须包含：现场照片、问题清单、整改要求、责任方、整改期限、复验结论。
- 报告必须区分：`必须整改 / 优化项 / 现场确认`。
- 发现材料型号、施工工艺、测试条件与合同不一致时，必须明确写“暂停下一道工序”或“暂停付款”。
- 水路打压、防水闭水、空调保压抽真空、地暖打压、封窗淋水这些测试，要么监理到场，要么视频见证并留档。

### 4. 监理合同怎么签
- 写清总服务次数和每次节点。
- 写清是否包含报价审核和合同陪签。
- 写清每次必须出报告和照片。
- 写清整改复验次数和费用。
- 写清监理不得和装修公司、材料商、空调、封窗、定制商存在利益关系。
- 写清监理失职责任：明显问题漏看、报告严重漏项，要退回对应服务费或承担约定责任。
- 写清监理服务费按节点付，不一次性付清。

### 5. 监理参与下的业主仍然要做什么
- 重大决定你自己拍板，监理不能替你决定买什么、怎么花钱。
- 所有图纸、排版图、下单单、替换单、增项单仍要你确认。
- 监理说能放行，也不代表你可以不看合同和预算。
- 付款前仍要你自己看一遍：是否验收合格、资料是否齐、整改是否闭环。

### 6. 请监理时最容易踩坑
- 只买 `1-2次` 验收，节点明显不够。
- 监理不审合同只看现场，这样前端漏项和付款陷阱没人帮你挡。
- 上门不出报告，只微信群口头说。
- 监理和装修公司有合作关系，问题写得很轻。

## B. 如果不找第三方监理，自己监理需要做什么
### 1. 角色和底线
- 你自己就是“业主监理”，重点抓 `关键节点 + 关键资料 + 关键付款`。
- 原则只有一句：`没验收、没记录、没签字，就不放行、不付款。`
- 底线只有五条：`材料不符不用、隐蔽未验不封、图纸未签不下单、测试未做不放行、整改未闭环不付款。`

### 2. 开工前准备
#### 2.1 资料
- 总预算表
- 半包合同和报价清单
- 主材采购清单
- 全屋定制合同和图纸
- 中央空调、封窗、地暖、瓷砖等专项合同
- 各专项验收标准
- 所有增项单、替换单、整改单

#### 2.2 工具
- 卷尺
- 空鼓锤
- 记号笔和美纹纸
- 插座测试器
- 充电宝

#### 2.3 固定执行规则
- 每次进场先拍整屋全景，再拍本次节点细节。
- 关键照片必须能看出 `房间 + 位置 + 参照物`。
- 仪表测试必须拍到 `读数 + 系统 + 时间`。
- 发现问题先标记，再拍照，再发群留痕。
- 所有变更必须走文字确认，不接受口头“先做着”。

### 3. 关键节点时间表
- 开工交底
- 水电定位
- 水电材料进场
- 水电验收
- 防水和闭水验收
- 中央空调隐蔽验收、保压、抽真空、调试
- 书房新增一路地暖铺设、打压、回填复验
- 封窗安装和淋水复验
- 瓦工排版确认和铺贴验收
- 吊顶/木作封板前验收
- 油工基层验收和面漆验收
- 定制复尺、下单前图纸确认、安装验收
- 卫浴、五金、灯具、开关插座、设备联调
- 竣工验收

### 4. 节点手册
#### 节点1 开工交底
- 到场：开工第一天，预留 `1-2小时`。
- 核对：半包范围、主材自购范围、各专项责任边界、施工顺序、付款节点、验收节点、需要你提前采购的材料和设备。
- 留证：原始全景、门窗裂缝空鼓渗水、下水烟道、配电箱、现场纪要。
- 不放行：没有书面或群内纪要；主材边界没说清；付款和验收节点没对齐。

#### 节点2 水电定位
- 到场：开槽前必须到场；厨房、双卫、餐边柜、电视柜、书桌、玄关柜、扫地机位逐点确认。
- 带资料：家电尺寸、家具尺寸、橱柜方案、浴室柜方案、全屋定制初步方案、空调风口和内机位置、网络和 AP 点位方案。
- 核对：冰箱、蒸烤箱、洗碗机、热水器、净水器、垃圾处理器、浴霸、洗烘机、扫地机的电源和接口；床头、书桌、餐边柜、电视柜、玄关柜、阳台家政柜的插座网口；厨卫台面小家电位、镜柜位、喷枪角阀位；空调回风口、检修口、冷凝水排向是否和吊顶柜体冲突。
- 留证：每面墙定位照、厨房和卫生间近景、电视墙和书桌位、配电箱和回路规划。
- 不开槽：大家电尺寸未定；关键柜体尺寸未定；厨卫和电视柜、书桌、扫地机位未确认。

#### 节点3 水电材料进场
- 到场：第一批水管、电线、线管、底盒、防水、网线进场时。
- 核对：品牌、型号、规格、批次必须与合同一致；电线为国标铜线；网线为六类无氧铜；防水不是同品牌低配款。
- 留证：每种材料整包外包装、关键标签近照、同批次堆放全景。
- 不开工：现场材料无法对应到合同型号；包装和标签信息不全；口头说“这个也一样用”但无书面确认。

#### 节点4 水电施工中途巡检
- 到场：开槽完成后一次，布管布线完成封槽前一次。
- 核对：强弱电分管分槽；线管无死弯；线盒横平竖直；冷热水走向清楚；湿区漏保和防溅措施到位；冰箱和厨房大功率回路合理；空调、热水器、地暖配合点位合理；关键点位不和定制冲突。
- 留证：全屋布线全景、厨房双卫阳台重点位、配电箱回路标识和走向。
- 不封槽：回路未标识；走向照片不全；关键点位与定位不一致。

#### 节点5 水电验收
- 到场：封槽前必须到场，这是最重要的隐蔽节点之一。
- 必测：`水路打压 0.8MPa / 30分钟 / 压降不超过 0.05MPa`；电路通断测试；绝缘测试；漏保测试；插座极性测试；回路核对。
- 材料和工艺下限：水管按合同型号执行，主水路优先 `伟星PPR / 日丰PPR / 金牛PPR` 同级；冷热水主水路按 `S3.2 / PN2.0热水管` 控制；电线为国标单股铜芯线，照明不低于 `1.5平方`，普通插座不低于 `2.5平方`，厨房卫浴阳台大功率不低于 `4平方`；网线为 `六类无氧铜 Cat6`；强弱电分管分槽；线管内不得有接头；湿区插座必须配漏保和防溅措施。
- 分位置重点：配电箱标签清晰并预留备用位；厨房重点看冰箱独立回路、蒸烤洗碗净水热水器、水槽下方接口和检修空间；卫生间重点看镜柜、吹风机、浴霸、喷枪、马桶、浴室柜和湿区防溅；阳台重点看洗烘、晾衣架、扫地机、家政柜；客厅餐边柜重点看电视墙设备区、网口、电源、50 管出口；卧室书房重点看床头、书桌、网口、AP、窗帘电机。
- 留证：打压表带时间读数、每个空间隐蔽全景、配电箱内部、回路标签、漏保和电气测试照片视频。
- 不放行：水路未打压或记录不完整；电气测试无记录；关键点位有偏差未整改。

#### 节点6 防水施工和闭水验收
- 到场：防水刷完一次，闭水结束再验一次。
- 核对：卫生间地面满刷，淋浴区墙面不低于 `1.8米`，其他墙面不低于 `30cm`，台盆背墙不低于 `1.2米`；阳台地面满刷，墙面上翻不低于 `30cm`；阴阳角、管根、门口、地漏加强处理；至少 `2遍` 交叉涂刷；卫生间闭水不低于 `48小时`，阳台闭水或淋水不低于 `24小时`。
- 材料下限：防水品牌和型号按合同执行，优先 `东方雨虹 / 德高 / 西卡 / 立邦 / 三棵树` 同级正规产品。
- 留证：防水全景、加强部位近照、闭水水位线、楼下检查照片或视频。
- 不贴砖：闭水时间不够；楼下有渗漏；门口、地漏、管根未加强。

#### 节点7 中央空调隐蔽验收
- 到场：管线安装完成封板前一次，抽真空和调试时再一次。
- 必验：内外机和控制器型号；铜管、保温、冷凝水管、信号线、电源线、分歧管；焊接是否为 `氮气保护焊`；`4.15MPa / 24小时` 保压；抽真空是否达到约定标准且观察不少于 `10分钟`；逐台通水；逐台试运行不少于 `30分钟`。
- 材料和工艺下限：铜管为合同约定品牌 `脱氧无缝紫铜管`；`6.35 / 9.52 / 12.7` 常规不低于 `0.8mm`，`15.88 / 19.05` 常规不低于 `1.0mm`；保温为 `B1级阻燃闭孔橡塑保温棉`，厚度不低于 `15mm`，易结露位置优先 `20mm`；冷凝水主管采用 `UPVC硬管`，单台内机常规不低于 `DN25`；坡度不低于 `1%`；每台内机必须留检修口。
- 分位置重点：客餐厅风口不直吹沙发和餐桌，不和灯槽窗帘盒打架；卧室不直吹床头；书房不直吹书桌；外机位散热、检修、支架、减震、排水到位；冷凝水无倒坡、下垂、积水点。
- 留证：铜管、保温、分歧管、冷凝水管、检修口全景；压力表和真空表；各房间内机和检修口位置。
- 不封吊顶：保压或抽真空没做或留证不完整；冷凝水未通水；检修口缺失。

#### 节点8 书房新增一路地暖验收
- 到场：铺管完成后一次，回填前一次，回填后复验一次。
- 核对：确认为书房新增一路，不是乱接原系统；管材品牌、型号、喷码、规格一致；有独立阀门和回路标识；保温板、反射膜、边界保温条到位；柜体下方不乱铺。
- 测试：首次打压 `0.8MPa / 30分钟`，并做 `24小时` 保压观察；回填带压 `0.4-0.6MPa`；回填后复压。
- 留证：书房铺管全景、分集水器和新增回路特写、压力表、回路标识、铺设走向。
- 不回填：打压未完成；新增回路无标识；铺设图和现场不一致。

#### 节点9 封窗安装和淋水验收
- 到场：窗框安装完成一次，淋水测试一次。
- 必验：型材、玻璃、五金、胶条、密封胶是否与合同一致；开启扇数量、方向、纱窗、限位器、防坠件是否到位；固定点、连接件、发泡、胶缝、排水孔是否合理；开合是否顺滑；淋水不少于 `30分钟`。
- 材料和工艺下限：客厅大阳台优先 `108/110系列`，卧室普通窗优先 `75/80系列`；主受力型材壁厚不低于 `2.0mm`；隔热条采用 `PA66-GF25`；玻璃必须钢化并带 `3C` 标识，重点位置采用 `Low-E中空` 或 `夹胶中空`；胶条采用 `EPDM三元乙丙`；室外胶采用中性耐候胶；窗框必须有固定片或膨胀连接件，不能只靠发泡固定。
- 分位置重点：客厅大阳台不能偷换成全固定低配结构；卧室窗开启方向顺手且不和窗帘盒打架；厨卫窗不和吊柜、浴室柜、热水器、五金冲突。
- 留证：每樘窗全景；玻璃 3C、五金品牌、胶缝、排水孔、限位器、防坠件；淋水过程和淋水后室内检查。
- 不付尾款：没做淋水或复检；玻璃五金型材与合同不符；开启扇卡顿、晃动或漏水。

#### 节点10 瓦工排版确认和铺贴验收
- 到场：铺砖前确认排版图一次，铺贴中期一次，完工验收一次。
- 核对：全屋通铺按图执行，显眼面无小窄条；厨卫阳台优先排水和防滑；瓷砖品牌、型号、色号、批号与采购单一致；辅材和工艺按约定执行；地漏坡度、卫生间排水做法正确。
- 工艺下限：大规格砖、低吸水率砖必须按约定使用匹配的 `瓷砖胶 + 背胶` 或厂家认可工艺；瓷砖胶和背胶优先 `德高 / 马贝 / 西卡 / 东方雨虹` 同级正规产品；卫浴和阳台地面现场泼水验坡；空鼓率按专项文档控制。
- 现场验法：空鼓锤抽检门口、地漏、墙角、开孔周边；`2米靠尺` 看平整度和高低差；卫生间和阳台现场泼水；看砖缝、阳角、地漏、门口、柜边收口。
- 留证：客餐厅主视角通铺效果；各卫生间地漏坡度和排水视频；空鼓标记；门口收口、阳角、开孔、窗边、橱柜后重点细节。
- 不放行：坡度不过关；空鼓超标；排版和显眼区效果不符合确认图。

#### 节点11 吊顶/木作封板前验收
- 到场：龙骨和基层完成、封石膏板前。
- 核对：空调检修口、风口、回风口保留；灯位、筒灯、射灯、窗帘盒、风口不冲突；厨房、卫生间、阳台设备检修位保留；封板后不会挡住阀门、接头、设备、线盒。
- 留证：天花龙骨全景、空调内机和检修口、风口和筒灯位综合照片、重要检修点近照。
- 不封板：没留检修口；关键检修位会被封死；风口灯位柜体关系未确认。

#### 节点12 油工基层和面漆验收
- 到场：基层找平完成一次，面漆完成后一次。
- 核对：阴阳角顺直；墙顶面平整度和波浪感；石膏板接缝、网格带、阴阳角条；面漆颜色统一，无透底、流坠、刷痕、污染。
- 现场验法：白天自然光和晚上开灯都看；站侧光位看墙面起伏；重点看门套边、柜边、窗边、灯带边、吊顶边。
- 留证：客厅大面墙、走廊、卧室主墙面、顶面；问题位贴美纹纸后拍照。
- 不放行：大面平整度差；阴阳角歪；面漆有明显透底、流坠、污染。

#### 节点13 定制复尺和下单前确认
- 到场：瓷砖、吊顶、墙面基本定型后复尺，商家出最终图纸时逐空间核对。
- 核对：板材环保等级、柜体板、门板、背板、五金符合合同；尺寸和内部结构落地；厨房、阳台、浴室柜明确防潮要求；插座、网口、上下水、阀门、检修口不会被封死；电视柜散热孔、背板开孔、PVC 50 管出口、电源网口标清。
- 留证：每个空间最终签字图纸、重点柜体局部、与现场点位对照照片。
- 不下单：图纸未签字；柜体与现场点位冲突；材料和五金清单不完整。

#### 节点14 定制安装验收
- 到场：柜体安装基本完成后一次，五金调试和整改后复验一次。
- 核对：柜体横平竖直、门缝均匀；板材、门板、五金品牌型号与合同一致；封边无开胶翘边割手；高频柜门开合顺畅；高柜、书柜、衣柜、阳台柜固定到墙；厨房、阳台、浴室柜开孔边做防潮处理。
- 材料和工艺下限：柜体、门板、背板、五金按合同清单执行；卧室、书房、儿童房主要柜体优先按 `ENF级` 环保板材控制；柜体侧板、层板、门板厚度不低于 `18mm`，背板不低于 `9mm`；潮湿区域采用更适合防潮的板材和开孔防潮处理；封边优先 `PUR封边` 或 `激光封边`；高频柜门采用阻尼铰链；高柜、书柜、衣柜、阳台柜必须固定防倾倒。
- 现场验法：柜门和抽屉逐个开关 `5-10次`；抽屉拉满和回关；晃高柜、书柜看固定；检查插座、阀门、地漏、检修口能否正常使用。
- 留证：每个空间柜体整体效果、五金标识、封边、开孔、固定件、检修口、问题位门缝和磕碰。
- 不付主要尾款：五金与合同不符；高柜未固定；开孔边未做防潮；功能性问题未整改。

#### 节点15 安装收尾和设备联调
- 到场：灯具、开关插座、卫浴、花洒、马桶、喷枪、角阀、地漏、五金、家电安装完成后。
- 核对：所有插座、开关、灯具通电正常；浴霸、排风、镜前灯、夜灯正常；马桶冲水顺、不漏、不晃、不返味；喷枪、角阀、软管、台盆下水、水槽下水不漏水；热水器、燃气、烟道、止逆阀、排烟符合要求；中央空调整机试运行正常。
- 留证：卫生间冲水和试水视频；厨房水槽、净水器、洗碗机、热水器周边接口；开关插座、配电箱、灯具点亮照片。
- 不进入最终付款：有漏水、漏电、异响、返味；空调、浴霸、排风、灯具、插座未联调；热水、燃气、排烟未确认。

#### 节点16 竣工验收
- 到场：全屋清场后，安排半天完整验收。
- 核对：地面、墙面、顶面、柜体、门窗、五金、设备整体完成度；整改项全部闭环；水电、防水、空调、地暖、封窗、定制、卫浴测试资料齐全；备用砖、板材余料、保修卡、说明书、遥控器、色号批次资料交付齐。
- 现场验法：按房间逐一检查，顺序按 `门窗 -> 墙顶地 -> 柜体 -> 开关插座 -> 给排水 -> 设备`。
- 不结清尾款：资料不齐；问题清单未闭环；关键功能未实测。

#### 厨房

- 角阀和插座不能被橱柜封死；热水器和燃气必须留检修空间；排烟和止逆阀要确认。

#### 主卫和公卫
- 镜柜、吹风机、浴霸、喷枪、马桶、花洒、地漏、下水防臭是否齐。
- 防水高度、闭水时间、地漏坡度、马桶返味、浴室柜封阀门是高频问题。

#### 阳台
- 洗烘、晾衣架、扫地机、家政柜、水电排水是否齐。
- 常见问题是尺寸不够、排水不顺、插座无防溅、柜体和设备打架。

#### 客厅和餐厅
- 电视墙、路由器、音响、餐边柜、饮水机、灯带、扫地机电源和网口是否齐。
- 设备区散热、空调风口、回风、检修口和吊顶柜体关系要提前看。

#### 卧室和书房
- 床头、书桌、网口、AP、窗帘电机、空调控制位是否顺手。
- 常见问题是床头插座不顺手、网口被柜体挡住、风口直吹人。

### 8. 付款前总检查
- 本节点验收是否合格。
- 上一轮整改是否全部完成。
- 本节点资料是否齐全。
- 现场实物是否与合同一致。
- 只要有 `材料不符 / 隐蔽未验收 / 测试未做 / 整改未闭环 / 图纸未签字` 任一项，就暂停付款。

### 9. 自己监理最容易漏的地方
- 只看表面效果，不看测试记录。
- 只拍完工照，不拍隐蔽照。
- 相信“这个我们一直这么做”，不去对合同和专项文档。
- 为了赶工默认放行，后期返工代价更大。
- 付款太快，整改没有抓手。

### 10. 自己监理的最低执行标准
- 每个关键节点都拍照和留记录。
- 每个问题都写成清单并编号。
- 每次付款前都核对验收结论、整改状态、资料状态。
- 拿不准的节点，至少找专项商家复核或临时买一次节点验收。
- 再忙也不能跳过 `水电 / 防水 / 空调 / 地暖 / 封窗 / 定制下单 / 竣工` 这几个节点。

## 共同适用的停工和暂停付款规则
- 材料品牌型号、规格、批次与合同不符：暂停使用，暂停付款。
- 隐蔽工程未验收就要封槽、封板、回填、贴砖：立即停工。
- 水路打压、防水闭水、空调保压抽真空、地暖打压、封窗淋水未做或留证不足：不得放行。
