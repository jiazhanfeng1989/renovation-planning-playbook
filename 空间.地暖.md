# 书房与地暖
## 功能分工
- 书房：两个孩子长期学习、打印、资料收纳、安静阅读。
- 儿童房：保留临时学习、阅读、画画和手工位置，不作为长期主学习区。
- 书房是两个孩子主要学习区，儿童房学习位作为备用。
## 书房内放什么
- 书桌：放两张儿童升降桌，满足两个孩子同时长期学习。
- 书柜：做封闭柜为主，局部开放格放常用书。
- 学习椅：预留两把椅子的活动空间和后退通道。
- 打印机：预留固定位置，不放地上临时摆。
- 路由/网络设备：预留网口、电源和弱电位置。
- 文件资料：用封闭柜或抽屉收纳，减少视觉杂乱。

## 必须确认
- 书房固定为两个孩子长期学习区，两张升降桌、网口、插座、打印机和资料收纳必须提前定点。
- 书房新增一路地暖必须和书桌、柜体、水电、瓷砖完成面统一复核。
- 有线网口、稳定 WiFi、足够插座和舒适照明必须满足。
- 书柜和高柜必须固定到墙体，柜体板材、封边和五金必须符合全屋定制要求。

## 布局标准
- 书房按纯书房设计，固定为两个孩子长期学习、临时办公、阅读、打印和资料收纳。
- 书房放两张儿童升降桌，每张桌面宽度按 `120-140cm` 预留，深度不低于 `60cm`。
- 两张升降桌并排或 L 型布置，必须保证椅子后方活动空间，不让柜门、房门和窗帘冲突。
- 书柜深度控制在 `30-35cm`，避免压迫小房间。
- 开放格只放常用书和少量展示，主要收纳使用封闭柜。
- 打印机必须有固定位置，不落地临时摆放。

## 网络和电源
- 书房至少预留 `1个有线网口`。
- 书房预留 `1个86面板AP网口 + 2个书桌有线网口`。
- 两张升降桌分别预留电脑、显示器、护眼台灯、平板充电和升降桌电机插座。
- 书桌下方插座集中但不外露，避免拖线板横跨地面。
- 路由、交换机、打印机等设备位置预留散热和检修空间。
- 两张升降桌区域强电插座不少于 `8个`，左右分布，避免全部集中在一侧。
- 打印机位置单独预留强电插座，旁边预留网络或数据线走线空间。
- 网口和电源不能被固定书桌、背板或柜体封死，后期能检修更换。
## 灯光和舒适
- 灯光按主灯 + 桌面工作灯配置。
- 书桌位置避免强眩光和背光。
- 中央空调出风口不能直吹座位。
- 窗户按高层封窗标准执行，重点隔音、密封、限位。
- 如重视安静，书房门需同步确认门芯结构、门扇厚度、静音条和门底密封，不只看窗户。
## 柜体和安全
- 书柜、吊柜必须固定到墙体。
- 板材环保等级、封边、五金品牌在全屋定制下单文件中锁定。
- 柜门优先封闭门，少用玻璃门和大面积开放格。
- 儿童可进入书房，柜体边角和拉手避免尖锐。
- 书柜层板跨度、承重和可调孔位必须写入图纸，避免后期放书下弯。
- 文件抽屉、打印纸和书籍重物放低位，减少高处承重和倾倒风险。
## 材料与选型
- 地面使用全屋 `750x1500mm` 柔光或哑光主砖通铺，不做过门石。
- 书柜和资料柜使用 `ENF级板材`，柜体侧板、层板、门板不低于 `18mm`，背板不低于 `9mm`。
- 柜体封边采用 `PUR封边` 或 `激光封边`。
- 书柜柜门使用 `百隆 CLIP top BLUMOTION / 海蒂诗 Sensys 8645i / DTC三段力阻尼铰链`。
- 抽屉使用隐藏阻尼滑轨，重物抽屉明确承重。
- 书房柜门采用门板下延、45度斜切或 G 型拉手槽，不用高频反弹器和自动开合器。
- 开关插座采用 `西门子远景Pro / 施耐德绎尚 / 罗格朗逸远`。
- 网线使用 `Cat6 UTP 23AWG 无氧铜六类网线`，型号在 `普天天诚 UTP-13-6-4P / 安普康 AMC655305 / 秋叶原 QS2668` 中选择。
- 墙面使用耐水腻子和乳胶漆，乳胶漆在 `立邦竹炭抗甲醛 / 多乐士森呼吸 / 三棵树鲜呼吸` 中选择。
- 窗户按高层封窗标准执行，普通窗采用 `75/80系列断桥铝系统窗`。
## 需要提前定好尺寸
- 两张升降桌长度、深度、高度和升降范围。
- 书柜深度、开放格比例和柜门方式。
- 打印机尺寸、开盖方向和插座位置。
- 两个孩子同时学习时椅子后方通道尺寸。
- 电脑、显示器、台灯、充电器数量。
## 水电和网络预留
- 两张升降桌下方分别预留足够插座，避免后期拖线板外露。
- 桌面附近预留手机、台灯、电脑、显示器、升降桌电机电源。
- 至少预留两个桌面有线网口和一个 86 面板 AP 网口，书房网络要稳定。
- 打印机位置单独预留插座。
- 灯光分层：主灯 + 桌面工作灯，避免只靠顶灯。
## 复尺时重点确认
- 书房净宽、净深。
- 门洞和开门方向不影响书桌和柜门。
- 窗户位置、窗台高度和窗帘方式。
- 两张升降桌放下后，坐下、起身、开门和开柜门通道满足使用。
- 书柜尺寸控制压迫感。

## 地暖专项
## 业主指定合同标准
### 1. 报价范围
- 报价必须按“书房新增一路地暖”单独报价。
- 报价必须拆分现场勘查、材料费、人工费、接口改造、阀门、过滤、温控、辅材、回填、打压、调试、售后。
- 报价必须写清新增一路的铺设面积、管材米数、接入位置、供回水连接方式和控制方式。
- 不接受只写“补一路地暖”“地暖改造一项”的报价。
- 报价必须写清管材品牌、型号、规格、壁厚、产地、批次、质保年限。
- 报价必须写清保温板厚度、密度、反射膜材质、边界保温条规格、卡钉或模块固定方式。
- 报价必须写清回填材料、回填厚度、养护时间、带压回填责任。
- 打孔、开槽、接口、阀门、过滤器、温控器、电源线、控制线、找平和成品保护费用必须列入报价。
- 商家漏报、错报、少报造成的补项，不得转嫁给业主。
### 2. 管材标准
- 地暖管采用 `PE-RT II型` 或 `PEX-a阻氧管`。
- 管材品牌在 `瑞好 REHAU / 乔治费歇尔 GF / 伟星 / 日丰 / 保利` 中选择。
- 主选规格为 `16x2.0mm`，大面积区域按设计采用 `20x2.0mm`。
- 管材必须为整盘新管进场，外壁喷码清晰，喷码包含品牌、型号、规格、生产批次。
- 每个回路必须使用整根管，中间不得有接头。
- 穿墙、穿门洞、分集水器出入口必须加保护套管。
- 管材弯曲半径必须符合厂家要求，不允许硬折、烤弯、压扁。
- 管材进场必须核对外包装、喷码、合格证和质保卡。
### 3. 分集水器和控制标准
- 先检查开发商原有分集水器路数、接口、阀门、过滤器、排气阀、排污阀和回路标识。
- 原有分集水器有空余路数时，新增书房一路接入原分集水器。
- 原有分集水器无空余路数时，商家必须提供独立接入方案，写清新增阀门、过滤、排气和检修方式。
- 新增一路必须有独立阀门和回路标识，不允许和其他房间混接成无法控制的回路。
- 分集水器箱位置必须便于检修，不得被柜体、墙板、固定家具封死。
- 集中供暖接入必须配置过滤器、阀门和温度控制措施，供水温度不得直接高温冲击地暖管。
- 温控器品牌在 `曼瑞德 / 丹佛斯 / 海林 / 绿米 / 西门子` 中选择。
- 书房新增一路温控按现有系统兼容方式配置，不破坏开发商原有地暖控制。
### 4. 铺设设计标准
- 施工前必须提供书房新增一路铺设图，标明接入位置、覆盖区域、管间距、管长、回路编号和避让区域。
- 书房按舒适供暖铺设。
- 固定柜体、橱柜、衣柜、玄关柜、大型落地柜下方不铺地暖管。
- 管间距控制在 `150-200mm`，靠窗、阳台门、外墙冷区加密到 `150mm`。
- 单路管长控制在 `80米以内`，最长不超过 `100米`。
- 新增一路和原有回路必须完成水力平衡调试，避免书房不热或影响其他房间。
- 管道距墙边完成面 `100-150mm`，距固定家具边界留出检修和打孔安全距离。
- 地暖图必须和水电图、中央空调图、全屋定制图、瓷砖排版图统一复核后签字。
### 5. 保温和反射层标准
- 地面基层必须清理干净，无尖锐石子、砂浆块、钢钉和明显起砂。
- 保温板采用 `XPS挤塑板`，厚度不低于 `20mm`。
- 保温板表观密度不低于 `30kg/m³`，压缩强度不低于 `150kPa`。
- 反射层采用铝膜反射膜或地暖专用反射膜，拼缝严密。
- 墙边必须铺设边界保温条，厚度不低于 `8mm`。
- 房间门口、长走廊、大面积区域必须设置伸缩缝或边界缓冲。
- 管道固定采用地暖卡钉、模块板或钢丝网固定，固定间距均匀，管道不得翘起。
- 保温板、反射膜、边界保温条进场必须核对品牌、厚度、密度和数量。
### 6. 打压测试
- 地暖铺设完成后必须做第一次打压测试。
- 打压压力按 `0.8MPa` 执行。
- 稳压时间不少于 `30分钟`，压力下降不超过 `0.05MPa`。
- 打压合格后进入 `24小时保压观察`，压力稳定且各回路无渗漏。
- 回填施工期间必须带压施工，压力保持在 `0.4-0.6MPa`。
- 回填完成后必须做第二次打压复验，压力、时间和留档要求与第一次一致。
- 打压测试必须通知业主或监理到场，照片或视频必须包含压力表读数、时间、房间位置和回路编号。
- 未完成打压验收，不允许回填、找平、贴砖或封闭分集水器。
### 7. 回填和找平标准
- 回填材料采用地暖专用回填砂浆或细石混凝土，不得使用带尖锐石子的粗骨料。
- 回填层必须覆盖地暖管，管顶以上保护层厚度控制在 `30-40mm`。
- 回填前必须检查管道固定、边界保温、伸缩缝和压力表。
- 回填施工不得踩踏、拖拽、砸压地暖管。
- 回填完成后必须进行养护，不得提前重载堆料、打孔、开槽。
- 地暖回填、找平、瓷砖铺贴后的完成面高度必须和入户门、室内门、卫生间、阳台、柜体踢脚线统一。
- 地暖区域后期打孔必须对照地暖管线照片和图纸，不得盲打。
### 8. 瓷砖配合标准
- 全屋瓷砖通铺前必须完成书房新增一路地暖打压和回填验收。
- 地暖区域瓷砖铺贴必须使用匹配地暖环境的瓷砖胶、背胶或水泥砂浆方案。
- 大规格砖铺贴必须控制空鼓、平整度和热胀冷缩风险。
- 门口、过道、客餐厅通铺区域必须提前做瓷砖排版，避免地暖伸缩缝和瓷砖显眼位置冲突。
- 美缝材料必须耐水、耐污、适配地暖热胀冷缩。
### 9. 升温调试
- 回填和瓷砖铺贴完成后必须达到材料养护周期，再进行升温调试。
- 初次升温从 `25℃` 供水温度开始。
- 每天升温不超过 `5℃`。
- 地暖供水温度控制在 `35-45℃` 范围内。
- 室内地表温度控制在 `26-28℃`，儿童房、卧室不得长期过热。
- 调试期间检查分集水器、阀门、接头、过滤器、回路流量和各房间升温均匀性。
- 调试记录必须写清日期、供回水温度、压力、各回路流量、异常问题和整改结果。
### 10. 售后和质保
- 地暖管质保不低于 `50年`，施工质保不低于 `2年`。
- 分集水器、阀门、温控器按厂家质保执行，质保书必须交付业主。
- 渗漏、阀门失灵、温控失灵、局部不热、分集水器漏水必须在 `24小时` 内响应，`48小时` 内到场。
- 商家必须交付地暖铺设图、回路编号图、材料合格证、打压记录、调试记录、质保书和售后联系人。
- 地暖管线照片必须按房间归档，照片能看清墙体参照物和管道走向。
## 11. 付款节点
- 定金不超过合同总价 `20%`。
- 现场勘查、接入方案、铺设图和材料清单确认后支付第二笔。
- 管材和辅材进场核对品牌、型号、规格后支付第三笔。
- 第一次打压、`24小时` 保压观察、带压回填和第二次复验合格后支付主要阶段款。
- 升温调试、资料交付、验收合格后支付主要尾款。
- 保留 `5%-10%` 质保尾款，运行一段时间无渗漏、局部不热和控制异常后支付。
## 容易被坑的点
- 管材只写品牌不写型号、规格、壁厚，实际使用低端工程管。
- 分集水器写进口或高端，实际使用同品牌低端款或无流量计款。
- 保温板厚度和密度缩水，后期热量向楼板散失。
- 边界保温条、伸缩缝省略，后期地面开裂、瓷砖鼓起。
- 回路过长或各回路长度差过大，导致部分房间不热。
- 固定柜体下方铺管，后期柜体受热变形、无效耗能。
- 打压只拍压力表，不拍房间、时间和回路，无法证明真实测试。
- 回填不带压施工，压坏管道后无法及时发现。
- 回填层使用尖锐石子，存在划伤管材风险。
- 分集水器被柜体封死，后期无法排气、清洗和维修。
## 签约和验收检查
### 签约前检查
- [ ] 物业和供暖管理方允许书房新增一路地暖，有书面要求或许可记录。
- [ ] 地暖报价写清书房新增一路的管材、接入位置、阀门、过滤、保温板、反射膜、边界保温条、温控、回填和调试。
- [ ] 地暖铺设图标明书房新增一路的管长、管间距、避让区域、接入位置和回路编号。
- [ ] 付款节点按验收走，质保尾款保留 `5%-10%`。
- [ ] 不接受“补一路地暖、接原系统、按常规施工、地暖管同级、现场再看接口、按实际发生不写单价”。
### 验收时检查
- [ ] 管材喷码、品牌、型号、规格、批次和合同一致。
- [ ] 原有分集水器和新增一路阀门、过滤器、回路标识与合同一致。
- [ ] 保温板厚度、密度、反射膜、边界保温条和合同一致。
- [ ] 书房新增一路管长、管间距和避让区域符合合同。
- [ ] 第一次打压 `0.8MPa / 30分钟` 合格，完成 `24小时保压观察`。
