# 全屋定制设计

## 这份文档管什么
- 这份只讨论 `设计`，不讨论板材品牌、封边工艺、五金型号、报价拆项和合同条款。
- 重点是：`功能划分`、`空间利用`、`空间布局`、`立面比例`、`颜色配比`、`家具搭配`、`功能性`。
- 全屋定制不是把柜子做满，而是用更清楚的收纳和更干净的立面，把家变得耐看、顺手、不乱。

## 本项目全屋定制定位
- 这是一个以 `长期自住` 为前提的家。
- 更适合做成 `收纳清楚 + 立面克制 + 颜色统一 + 使用轻松` 的方向。
- 设计目标不是“哪里都做柜”，而是让柜体真正服务玄关、客餐厅、厨房、卧室、儿童房、书房和阳台的日常生活。

## 设计总原则
- 先定 `每个空间放什么、怎么用`，再定柜体体块和门型。
- 先保 `空间轻重平衡`，再谈大面积满墙定制。
- 先保 `真正高频的收纳效率`，再追求表面整齐。
- 先保 `柜门可开、抽屉可拉、家电可用`，再追求无缝一体感。
- 先保 `全屋颜色和线条统一`，再做局部存在感。

## 设计时要考虑什么

### 1. 功能划分
- 玄关柜解决鞋、包、外套、雨伞、临时置物和回家落手区。
- 餐边柜解决饮水、小家电、杯具、零食和餐厨缓冲收纳。
- 客厅收纳解决设备、杂物、玩具、书和视觉留白。
- 书房柜解决书籍、资料、打印区和安静工作的秩序。
- 主卧衣柜解决当季衣物、换季收纳、被褥和行李箱。
- 儿童房柜体解决衣物、书本、玩具和成长变化。
- 阳台柜解决清洁工具、洗烘用品和家政杂物。

### 2. 空间利用
- 优先把柜体留在 `真的能提升效率` 的位置，而不是见墙就做。
- 小空间比起“做满”，更重要的是 `动线不堵`、`开门不打架`、`台面和留白还在`。
- 满墙柜不是越多越好，柜体越大，越要控制压迫感和厚重感。
- 柜体和空调回风口、检修口、窗帘盒、插座、门洞的关系必须提前一起看。

### 3. 空间布局
- 柜体应服务家具布置，不应反过来逼着沙发、餐桌、床和书桌迁就。
- 高柜适合集中，不适合零碎插在各个墙面里打断空间。
- 开门方向、抽屉位置、过道宽度要按日常动作推演，而不是只看立面好不好看。
- 电视柜、餐边柜、书柜和衣柜都要预留真实可用的开启与站立空间。

## 收纳策略怎么定

### 1. 封闭和开放的比例
- 这个家更适合 `封闭为主，开放为辅`。
- 开放格只留给少量真正高频、真正好看的物品。
- 开放越多，越容易乱，也越依赖长期整理习惯。

### 2. 常用和低频的层级
- 高频物品放在最顺手的一层，不靠弯腰和踩凳子。
- 低频物品再往高处、深处和角落走。
- 行李箱、被褥、大件杂物要提前留专门区域，不要等入住后再四处塞。

### 3. 抽屉和平开门的关系
- 高频收纳更适合抽屉。
- 大体积、低频或需要竖向利用的收纳可用平开门。
- 不要为了立面统一，把所有柜体都做成同一种开合方式。

## 柜体体块怎么控制

### 1. 轻重关系
- 地柜和低柜负责稳定感。
- 高柜和满墙柜负责容量，但要克制数量。
- 柜体越高、越宽、越整面，旁边越需要留白、矮柜或开放空间来平衡。

### 2. 满墙定制怎么避免压抑
- 不要每个房间都做满墙通顶。
- 客餐厅更需要考虑视觉呼吸感，不适合柜子、背景墙、线条同时都很重。
- 卧室和儿童房要优先保证安静感，不适合做过于复杂的分色和切割。

### 3. 柜门分缝和立面秩序
- 立面好不好看，关键不是门做得多贵，而是 `分缝是否整齐`、`比例是否顺`。
- 门缝、抽屉缝、开放格和墙面线条最好有统一逻辑。
- 柜门尺寸不要忽大忽小，不要为了迁就局部尺寸把整面柜体切得很碎。

## 各空间设计建议

### 玄关
- 重点是 `进出顺手 + 不显乱 + 有落手区`。
- 玄关柜不宜过厚、过满，避免一进门就压。
- 换鞋凳、挂衣区、常穿鞋区和临时置物区要形成连续动作。

### 客餐厅
- 重点是 `收纳够用 + 立面干净 + 不抢公共空间的轻松感`。
- 餐边柜更适合承担水吧和日常收纳，不适合堆太多高大家电。
- 电视柜和收纳柜要给设备、电源、网口、散热和检修留出设计位置，不要只顾封板整齐。

### 书房
- 重点是 `资料有序 + 工作安静 + 不显压`。
- 书柜更适合封闭和半开放混合，而不是整墙全部开放。
- 书桌、书柜、空调回风和插座位置要一起看，避免后期互相遮挡。

### 主卧
- 重点是 `安静、克制、顺手`。
- 衣柜应围绕起居和换衣路径布置，不要让开门与床、床头柜打架。
- 主卧柜体更适合简洁门型和低存在感配色。

### 儿童房
- 重点是 `成长变化 + 安全 + 留余地`。
- 柜体不要把房间一次性定死，后续床位、书桌位和收纳重点都可能变化。
- 立面宜更轻，避免大面积沉重柜体压低儿童房氛围。

### 阳台
- 重点是 `家政效率 + 杂物收住 + 视觉不乱`。
- 洗烘区、清洁工具区、纸品区和杂物区边界要清楚。
- 阳台柜更适合作为辅助收纳，不宜做成视觉中心。

## 颜色配比

### 1. 全屋基调
- 更适合 `低饱和、中性色、长期耐看` 的方向。
- 柜体颜色应与墙面、瓷砖、地面和室内门保持冷暖一致。
- 如果全屋已经有较多木色，定制柜的木色就要控制数量和分布，避免越做越重。

### 2. 柜体颜色关系
- 大面积柜体更适合浅色、灰米色、柔和木色或低对比搭配。
- 公共空间的柜体颜色宜更安静，不宜突然跳色。
- 小房间柜体越大，颜色越应克制，这样空间更轻。

### 3. 深浅比例
- 浅色柜门可以扩大空间感。
- 深色更适合小面积点缀、局部台面或少量开放层板。
- 不建议全屋同时出现太多强纹理、重木色和深柜门。

## 和家具怎么配
- 柜体不是独立设计，必须和 `沙发`、`餐桌`、`床`、`书桌`、`儿童家具` 一起看。
- 客餐厅柜体如果已经存在感较强，家具就应更简洁，避免全都抢视觉中心。
- 卧室柜体宜退后，让床和软装成为主要氛围来源。
- 书房定制应优先服务书桌和工作位，不要让柜体挤压坐姿和采光。

## 细节上最容易忽视的设计问题
- 只看立面，不看柜门打开后与家具、门扇、过道是否冲突。
- 柜子做满了，却没有给扫地机、路由器、空气净化器或小推车留位置。
- 只顾一门到顶的整齐感，忽略设备散热、插座使用和后期检修。
- 颜色单看一块板好看，放到全屋后却偏黄、偏灰或偏重。
- 开放格做得很多，但实际住进去后全是视觉噪音。

## 这个家更适合的全屋定制方向
- 更适合：`浅色主柜体 + 少量木色点缀 + 封闭收纳为主 + 立面简洁 + 分缝克制`。
- 不建议：满屋高柜、满墙展示、复杂拼色、过多开放格、每个房间都做强存在感定制。
- 目标不是做出样板间，而是让家在入住后依然整洁、轻松、长期耐看。

## 设计定稿清单
- [ ] 每个空间的柜体已经先按生活功能划分，而不是先按墙面做满。
- [ ] 高频物品、低频物品和大件杂物都有明确收纳位置。
- [ ] 高柜、低柜、开放格和留白的比例已经平衡。
- [ ] 柜门开启、抽屉拉出和家具摆放之间没有冲突。
- [ ] 柜体没有挡住空调回风、检修口、插座、网口和必要设备位。
- [ ] 客餐厅柜体没有把公共空间压得过重。
- [ ] 主卧和儿童房柜体颜色、体块和分缝都保持克制。
- [ ] 全屋柜体与墙面、瓷砖、地面、室内门的冷暖和明度关系已统一。
- [ ] 电视柜、餐边柜、书柜、阳台柜都兼顾了设备使用和后期维护。
- [ ] 这套定制方案看起来整洁，住进去也能长期保持顺手和不乱。
