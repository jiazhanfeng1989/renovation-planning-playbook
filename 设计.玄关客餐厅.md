# 玄关客餐厅设计

## 这份文档管什么
- 这份只讨论 `设计`，不讨论板材、五金、乳胶漆型号和施工工艺。
- 重点是：`功能划分`、`空间利用`、`空间布局`、`颜色配比`、`家具搭配`、`功能性`。
- 玄关、客厅、餐厅要作为一个连续空间一起看，不单独拆开做漂亮但互相打架的局部。

## 这个空间的核心任务
- 玄关负责 `进出门秩序`、`收纳`、`换鞋`、`临时落物`。
- 客厅负责 `全家停留`、`会客`、`陪伴`、`放松`、`影音`。
- 餐厅负责 `日常就餐`、`家庭互动`、`轻度水吧功能`、`偶尔扩展待客`。
- 三个区域共享的目标是：`动线顺`、`视觉统一`、`储物够用`、`不过度压迫`。

## 设计总原则
- 先定 `生活场景`，再定柜体、桌椅和配色。
- 先保证 `通行` 和 `使用舒适`，再做满墙造型。
- 先保证 `收纳边界清楚`，再谈展示感。
- 先保证 `大件家具比例`，再决定软装和装饰。
- 先建立 `统一基调`，再局部加颜色和层次。

## 功能划分

### 玄关
- 必须承担：鞋、外套、书包、雨伞、钥匙、快递、临时包袋的落位。
- 玄关不是纯展示区，要能真的接住一家四口高频出入的杂物。
- 儿童家庭要给孩子单独留一层或一段可自行拿放的区域，减少全靠大人收拾。

### 客厅
- 客厅核心是 `沙发区`，不是电视墙本身。
- 电视墙和对面柜体都应为“日常停留”服务，而不是为了做满而做满。
- 客厅最好能同时容纳：大人坐着聊天、孩子活动、偶尔看电视、偶尔来客。

### 餐厅
- 餐厅核心是 `餐桌 + 餐边柜 + 通行`。
- 餐边柜更适合承担饮水、咖啡、杯具、零食、部分餐具和日常杂物缓冲。
- 餐厅不建议过度厨房化，不把它变成蒸烤箱和大电器展示区。

## 空间利用

### 玄关
- 鞋柜可以做足，但不要压窄入户第一段动线。
- 换鞋凳、柜门和入户门的开启关系要先复核，避免实际使用互相打架。
- 玄关底部留一段常穿鞋开放区，比全部封死更符合日常体验。

### 客厅
- 客厅不要为了收纳做成“满墙柜 + 满墙电视墙”的对冲式压迫布局。
- 柜子可以做，但应控制 `封闭储物` 和 `留白` 的比例。
- 如果沙发背后有空间，可以考虑低存在感的边几、落地灯、移动边桌，而不是再塞一组厚柜。

### 餐厅
- 餐桌周边要留出椅子拉出和人绕行的空间。
- 小空间优先用 `可扩展餐桌` 或尺度克制的四人桌，而不是一开始就用过大的六人桌。
- 餐边柜深度和开门方式要服务日常，不要为了视觉厚重感做太深。

## 空间布局

### 玄关布局
- 玄关柜建议包含：`鞋区`、`换鞋位`、`挂衣区`、`临时置物区`。
- 常用区放在最顺手的位置，低频区再放上层或角落。
- 行李箱位、儿童书包位、雨伞位这种需求最好提前留，不要靠后期“找地方塞”。

### 客厅布局
- 沙发、茶几、电视之间的关系，应优先围绕 `看电视舒适`、`聊天方便` 和 `孩子活动留白`。
- L 型沙发不一定必须靠墙，条件允许时也可以用来划定客厅区域，但前提是不会堵动线。
- 茶几不一定必须大，边几、可移动小桌、储物边桌有时更适合家庭生活。
- 电视墙保持简洁通常比复杂背景更耐看，也更适合后续调整家具。

### 餐厅布局
- 餐桌灯必须围绕 `最终餐桌中心` 定位，不围绕原始顶面中心定位。
- 餐边柜、水吧区、餐桌、通道要在一张图里一起看，避免餐椅一拉出就撞柜。
- 如果餐厅兼部分学习、做手工或轻办公需求，优先用 `灵活家具` 解决，而不是永久做很重的功能岛。

## 家具搭配

### 玄关
- 玄关家具要偏轻巧、耐用、低负担，不要一进门就是厚重压顶的大体量。
- 换鞋凳可与鞋柜一体，也可做独立小件，重点是坐下和起身顺手。

### 客厅
- 沙发是客厅第一大件，先定沙发，再反推茶几、边几、地毯和客厅柜。
- 如果想让空间显得轻，优先选 `低重心`、`线条简洁`、`体块克制` 的家具。
- 茶几可以弱化成组合边几或小体量茶几，把活动空间让出来。
- 客厅柜以封闭为主、少量展示为辅，更适合家庭长期维持整洁。

### 餐厅
- 餐桌和餐椅不一定要成套，但必须 `比例协调`、`颜色相容`、`风格不冲突`。
- 四口之家日常更重要的是舒适和通行，不是摆一张看起来气派但难用的大桌。
- 如果以后偶尔待客，优先考虑 `伸缩餐桌`，比常年占地的大桌更合理。

## 颜色配比

### 整体基调
- 客餐厅建议以 `低饱和中性色` 做底，保证耐看和整体统一。
- 主基底优先用在：墙面、顶面、地面、大柜体。
- 第二层颜色可放在：沙发、餐椅、窗帘、地毯、挂画。
- 点缀色只放少量，不要每样都跳。

### 玄关
- 玄关适合比客厅略稳一点，给人收束和归家的感觉。
- 鞋柜颜色若太深，容易让入户第一眼变重；若太浅，又容易显脏，要结合地面和墙面一起平衡。

### 客厅
- 客厅主色建议安静、克制，避免几年后快速审美疲劳。
- 大件家具颜色不要彼此抢戏，沙发和客厅柜至少要有一方退后。
- 如果有儿童活动需求，色彩可以活泼一些，但最好通过可替换软装实现，而不是靠固定硬装上很多强色。

### 餐厅
- 餐厅可以比客厅略暖，增强日常用餐氛围。
- 餐边柜、水吧区和餐桌椅的颜色关系应更偏“协调”，不建议每件都强调存在感。

## 比例和尺度

### 玄关
- 入户第一段动线要留出自然通过感，不能开门就撞凳、撞柜、撞人。
- 柜体深度要结合鞋码、门扇、换鞋动作和通道一起定，不只看单独数值。

### 客厅
- 沙发、茶几、电视柜、地毯的尺度要成组看，不能单件都对却组合失衡。
- 大沙发配小地毯会显得局促，小沙发配太大茶几会显得笨重。
- 如果电视尺寸较大，电视柜和墙面留白也要跟上，不然画面很重、底座很轻，会失衡。

### 餐厅
- 餐桌尺寸要按日常人数定，不按“偶尔来客最多能坐几个”定。
- 餐桌周边至少要确保：坐下、拉椅、端菜、孩子绕行都不难受。
- 如果铺地毯，地毯边缘最好大于餐桌外轮廓，保证拉椅后仍然协调。

## 视觉层次
- 地面、墙面、柜体、沙发、窗帘、灯具要有清晰层次，不要全都一个重量。
- 客厅如果电视墙简洁，对面柜体就也要克制；不要一边极简、一边极满。
- 玄关、客厅、餐厅虽然可以分区，但风格语言要统一，不然会像三个拼起来的空间。

## 灯光和氛围
- 玄关优先保证进门即亮、取物方便、夜间不刺眼。
- 客厅灯光不要只靠一盏主灯，应考虑基础照明、沙发区氛围照明和局部辅助照明。
- 餐厅灯光要围绕餐桌，重点是把桌面照好、氛围舒服。
- 灯具形式要和空调风口、吊顶线条、窗帘盒一起看，不单独决定。

## 和其他系统怎么配合
- 和 `中央空调` 配合：避免风口直吹沙发、餐桌和主坐位；避免回风口被柜体和灯槽挡住。
- 和 `窗帘` 配合：窗帘盒、窗帘堆叠和客厅柜、空调吊顶关系要提前复核。
- 和 `定制柜` 配合：高柜、吊柜不要把插座、弱电箱、检修口和路由设备封死。
- 和 `瓷砖` 配合：地面通铺的尺寸、颜色和光泽，会直接影响柜体颜色和沙发选择。

## 这个家更适合的方向
- 整体方向更适合：`现代简洁 + 温和耐看 + 轻收纳压迫感`。
- 玄关客餐厅不建议做太重的岩板背景、复杂护墙、满墙开放格和过度装饰性灯槽。
- 更适合用：干净墙面、简洁柜体、少量木色或暖灰、少量点缀色、可替换软装。

## 设计定稿清单
- [ ] 玄关已经明确鞋区、换鞋位、挂衣区、临时置物区的边界。
- [ ] 入户门、柜门、换鞋凳和通行动线已复核。
- [ ] 客厅已经先定沙发区，再定电视墙和客厅柜。
- [ ] 客厅柜没有做成满墙压迫式收纳。
- [ ] 沙发、茶几、地毯、电视柜的比例已成组复核。
- [ ] 餐桌尺寸按日常四口之家使用优先，不按极少数聚餐场景优先。
- [ ] 餐桌、餐边柜、通道、餐桌灯中心关系已定稿。
- [ ] 玄关、客厅、餐厅的主色、辅助色和点缀色关系已明确。
- [ ] 大件家具颜色和柜体颜色没有互相抢戏。
- [ ] 空调风口、回风口、窗帘盒、灯具和柜体关系已一起复核。
