# 封窗合同模板

打印和签约时，正文及连续装订页逐页签字或盖章，空白处划线作废。

## 签约信息
- 甲方：本房屋业主
- 乙方：签约封窗商家工商全称
- 项目地址：本项目房屋
- 联系人：甲方本人；乙方销售、复尺、安装、售后负责人
- 联系电话：签约时填写
- 预估签约报价：`5.0万` 合理口径；高配口径约 `6.5-9万`
- 合同总价：按逐樘报价合计填写大写和小写金额
- 收款账户：乙方对公账户全称、开户行、账号
- 签约日期：签约时填写

## 适用范围
- 适用于本项目客厅大阳台、卧室窗、小阳台窗的封窗签约。
- 本项目按高层住宅标准控制，不按低配普通推拉窗执行。

## 本项目签约底线
- 客厅大阳台优先 `108/110系列断桥铝系统窗`。
- 卧室和普通窗优先 `75/80系列断桥铝系统窗`。
- 型材材质为 `6063-T5原生铝`，主受力壁厚不低于 `2.0mm`，非主受力壁厚不低于 `1.8mm`。
- 隔热条为 `PA66-GF25`，不得使用 `PVC` 隔热条。
- 玻璃全部为钢化玻璃并带 `3C` 标识。
- 客厅大阳台玻璃优先 `Low-E + 中空 + 夹胶` 组合。
- 五金在 `好博 HOPO / 坚朗 KINLONG / 丝吉利娅 SIEGENIA / 诺托 ROTO` 中锁定具体型号。
- 胶条为 `EPDM三元乙丙胶条`。
- 室外耐候密封胶在 `之江 / 白云 / 道康宁 / 西卡` 中锁定具体型号。
- 客厅大阳台至少保留 `2个` 有效开启扇，且不能与家政柜、洗烘区、电动晾衣架、窗帘盒打架。
- 不接受 `同级替代 / 品牌同档次 / 常规系统窗 / 家装系列 / 工程系列 / 按现场调整 / 主材正品辅材同级` 这类模糊说法。

## 合同模板

### 1. 合同主体与项目概况
- 甲方为本房屋业主，乙方为封窗供货及安装单位。
- 本合同总价、逐樘报价、复尺尺寸、窗型生产图、材料与五金型号，必须直接写在本合同正文内，或作为与本合同连续装订、逐页签字或盖章的合同页一并生效。
- 合同正文、连续装订的报价页、生产图页、材料型号页、书面变更页具有同等效力；口头承诺无效，微信或短信承诺必须转成合同页或盖章补充页。
- 未写入正文、连续装订页和盖章补充页的项目，视为乙方未报价，不得在安装阶段再以“常规增项”方式追加。

### 2. 供货与施工范围
- 乙方负责拆旧、复尺、深化图、生产、运输、吊装、安装、收口、发泡、打胶、五金安装、纱窗安装、成品保护、垃圾清运和验收整改。
- 乙方负责窗框与墙体交接部位的固定连接、发泡填充、内外侧密封、防水和排水通道处理，不得只供货不对安装质量负责。
- 乙方负责拆旧后洞口的复核、找方、找平和与新窗安装直接相关的必要修补；如存在超出常规安装范围的结构性空鼓、开裂、渗水等问题，乙方须在拆旧后 `24小时内` 书面提交现场照片、处理方案、费用和责任边界，未经甲方书面确认不得形成增项。
- 乙方负责第一次淋水测试；现场条件不允许时，负责第一次有效大雨后的复检和整改。
- 乙方负责在复尺图阶段解决开启扇与阳台柜、窗帘盒、电动晾衣架等位置冲突，不得安装后再以“现场条件如此”为由甩锅。

### 3. 图纸与报价要求
- 每一樘窗必须单独列明洞口尺寸、成品尺寸、系列、型材颜色、固定扇、开启扇、开启方向、纱窗、限位器、防坠件、玻璃结构、五金型号、单价和总价。
- 窗型生产图必须标注每樘窗的分格、执手方向、排水孔、拼接位置、转角立柱、安装定位、内外侧收口边界，不得只给简单立面草图。
- 报价必须包含拆旧、运输、吊装、安装、固定件、发泡、打胶、收口、成品保护、垃圾清运、淋水测试和售后整改。
- 任何高空作业费、超大玻璃费、转角窗费、异形窗费、夜间施工费、外墙修补费、二次上门费，必须签约前一次性写清；未写清视为已含。
- 乙方复尺后必须提交最终《窗型生产图》，甲方书面确认后方可下单。
- 乙方复尺后必须同步提交《封窗材料与五金清单》，逐樘写清：`型材品牌、系列、颜色、壁厚、玻璃结构、五金品牌系列型号、胶条、室外耐候胶、室内密封胶、发泡材料、固定件、纱窗、限位器、防坠件、收口边界`。
- 因乙方复尺错误、下单错误、生产错误产生的返工、延期和材料损失，由乙方承担。

### 4. 材料配置条款
- 客厅大阳台按 `108/110系列断桥铝系统窗` 执行；卧室窗按 `75/80系列断桥铝系统窗` 执行。
- 型材品牌在 `兴发 / 坚美 / 凤铝` 中锁定唯一品牌和系列，不接受“同级替代”。
- 玻璃必须写清品牌、厚度、中空层厚度、Low-E 面、夹胶层厚度和 `3C` 编码要求。
- 客厅大阳台玻璃优先 `6钢化 Low-E + 16A/18A中空 + 5钢化 + 0.76PVB + 5钢化夹胶`。
- 五金必须写清品牌、系列、型号、数量；开启扇执手、合页、滑撑、锁点、防坠件不得笼统写“品牌五金”。
- 胶条必须写清 `EPDM`；密封胶必须写清品牌和型号，不得写“中性胶”“耐候胶”这类模糊词。
- 室外耐候胶和室内密封胶必须分开写清品牌、型号、颜色和施工位置，不接受只写“打胶处理”。
- 发泡胶、固定片、膨胀螺栓、连接件、防坠件虽不单独指定品牌，也必须写清规格、数量口径和使用位置，并纳入安装隐蔽拍照留档。

### 5. 性能与安全条款
- 乙方应提供所选系列门窗的检测资料，至少包含抗风压、水密、气密性能。
- 本项目按高层外窗控制，抗风压性能不低于 `5级`，气密性能不低于 `6级`，水密性能不低于 `4级`。
- 高层优先 `内开内倒`，确需外开时必须配置限位、防坠绳或防坠链。
- 儿童可触达位置的开启扇必须配置限位器。

### 6. 安装工艺条款
- 安装前必须完成现场复尺、旧窗拆除后洞口复核和安装条件确认；洞口尺寸、垂直平整度明显不合格时，必须先修补后安装。
- 窗框安装必须横平竖直，固定点距框角不大于 `150-200mm`，中间固定点间距不大于 `500mm`。
- 窗框不得只靠发泡胶固定，必须使用满足高层外窗要求的固定片、膨胀螺栓或连接件。
- 窗框下口必须保证排水通道通畅，排水孔不得被发泡、胶或收口材料堵死；排水方向必须明确向外，不得形成倒泛水。
- 内外侧收口责任必须一次性写清：窗框与墙面、瓷砖、石材、窗台板、柜体交接处由谁收口、做到什么标准、费用是否已含，签约后不得互相甩锅。
- 如需现场加装副框、垫块、转接件、附框加固件或额外防水处理，必须在复尺后一次性书面列清；未书面列清的，默认已含在合同总价内。
- 乙方必须对每一樘窗留存安装隐蔽照片，至少包含固定点、连接件、发泡、排水孔、外侧胶缝、防坠件。

### 7. 工期、验收与付款
- 工期以图纸确认后约定天数执行；因乙方复尺、生产、运输、安装安排问题导致延期的，由乙方承担延期责任。
- 每樘窗安装完成后，开启扇连续开合不少于 `10次`，不得卡顿、异响、下垂；锁闭后窗扇不得明显晃动。
- 验收时逐樘核对型材系列、型材壁厚、型材颜色、玻璃结构、`3C` 标识、五金型号、胶条和密封胶品牌。
- 验收时必须检查玻璃是否存在划伤炸边、Low-E 面错误、中空层污染、边部脱胶等问题。
- 淋水测试连续不少于 `30分钟`，重点测试四周、拼接、转角、下口和排水孔；室内不得渗漏。
- 定金不超过合同总价 `20%`；到货核对后支付主要货款；安装和淋水验收通过后支付至 `90%`；保留 `10%` 质保尾款。

## 签约 Checklist

### 签约前
- [ ] 每一樘窗都单独报价，不是只写一个总价。
- [ ] 客厅大阳台 `108/110系列`、卧室 `75/80系列` 已写进合同。
- [ ] 型材品牌、系列、壁厚、颜色、隔热条材质已写清。
- [ ] 玻璃结构、Low-E、夹胶、中空层厚度、`3C` 要求已写清。
- [ ] 五金品牌、系列、型号、数量已写清，不是“品牌五金”。
- [ ] 胶条、密封胶、纱窗的品牌型号或明确配置已写清。
- [ ] 《封窗材料与五金清单》已并入合同。
- [ ] 拆旧、吊装、固定、打胶、收口、洞口修补责任、垃圾清运是否包含已写清。
- [ ] 高空作业、超大玻璃、异形窗、转角窗、外墙修补、二次上门等费用是否另算已写清。
- [ ] 合同里没有 `同级替代 / 品牌同档次 / 常规系统窗 / 按现场调整 / 主材正品辅材同级` 这类模糊词。

### 安装和验收
- [ ] 固定点、连接件、发泡、排水孔、外侧胶缝有隐蔽照片留档。
- [ ] 是否需要副框、转接件、附框加固件、额外防水处理已在复尺后一次写清。
- [ ] 客厅大阳台开启扇数量足够，且不与柜体、晾衣架、窗帘盒打架。
- [ ] 五金开关顺畅，无明显晃动和异响。
- [ ] 限位器、防坠件、纱窗安装到位。
- [ ] 室内外收口完整，无明显缝隙、裂缝和甩锅项。
- [ ] 淋水 `30分钟` 或雨后复检无渗漏。
