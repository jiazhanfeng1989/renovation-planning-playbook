## 下单底线
- 本合同报价统一按 `默认品牌口径 + 必填字段 + 预算` 三栏去问价，任何商家报价不写型号和数量，默认视为不可比价。
- 所有自采件都要锁定：`品牌 + 系列 + 型号 + 尺寸 + 颜色 + 数量 + 单价 + 是否含安装 + 是否含安装辅件`。
- 赠品件必须写明品牌型号；未写清的赠品默认按未包含处理。
- 厨房设备、卫浴设备、灯具面板、窗帘电机必须和水电点位、定制尺寸、吊顶开孔一起复核。
- 任何涉及安装的小件，不接受商家口头说“到时师傅会带”。

## 下单 Checklist
- [ ] 供应商订单或报价单已写清品牌、型号、尺寸、颜色、数量、单价和总价。
- [ ] 已写清是否含安装、是否含安装辅件、是否含配送上楼和二次上门。
- [ ] 已写清到货时间、到货地址、签收人、退换货和破损责任。
- [ ] 已复核与水电点位、定制尺寸、开孔、排烟、上下水和收口的关系。
- [ ] 已确认发票抬头、收款账户、售后联系人和质保口径。
