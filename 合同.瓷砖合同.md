## 本项目签约底线
- 本项目瓷砖主材由甲方采购，铺贴施工由半包工队负责；乙方不得把本应由供货方承担的 `同批次供货、补货、破损、运输、上楼、色差` 风险转嫁给甲方。
- 所有瓷砖必须写清 `品牌 + 系列 + 型号 + 规格 + 等级 + 厚度 + 表面工艺 + 色号 + 批号 + 防滑等级 + 数量 + 损耗率 + 单价 + 总价`。
- 美缝 / 环氧彩砂必须写清 `品牌 + 系列 + 型号 + 颜色 + 施工范围 + 材料总价 + 施工单价/总价 + 合计总价`，本模板默认材料和施工分开报价、合并预算。
- 同一空间必须尽量同品牌、同系列、同批次供货，不接受 `同色系 / 同款效果 / 广东砖 / 一线同厂` 这类模糊写法。
- 本合同默认报价统一按 `强辉 / 新中源 / 鹰牌` 这类质量稳定二线品牌询价；正式签约时必须从可接受品牌池中锁定 `唯一品牌 + 唯一系列 + 唯一型号 + 唯一色号/批号`。
- 本项目固定方案为：客餐厅、过道、卧室、书房、阳台 `750x1500mm` 主砖通铺；厨房墙砖 `400x800mm`；厨房地砖 `300x600mm`；主卫和公卫墙砖 `400x800mm`；主卫和公卫地砖 `300x600mm`。
- 卫生间墙面外阳角固定做 `海棠角`；墙地交界固定按 `墙压地 + 柔性防霉硅酮胶缝` 执行。
- 不接受先用低砖价签约，再通过 `加工费、搬运费、上楼费、补货费、美缝费、收边条、小料、异形切割` 补价。

## 合同模板

### 1. 报价范围
- 报价必须拆分瓷砖主材、加工切割、倒角、海棠角、搬运上楼、补货预备金、美缝或环氧彩砂材料、收边条和小料。
- 每个空间分别列瓷砖品牌、系列、型号、规格、等级、厚度、表面工艺、是否直边修边、数量、损耗率、单价和总价。
- 大规格砖、墙砖上墙、加工切割、倒角、海棠角、异形铺贴费用必须提前列明。
- 全屋通铺排版、对缝、切割、无过门石收口费用必须提前列明。
- 送货上楼、搬运、补货、退货、破损责任必须写清。
- 瓷砖排版图必须在铺贴前确认，显眼位置不能出现小窄条。
- 主砖、厨卫墙地砖、海棠角和收边效果，建议在下单或首批到货前封样签字；到货应与封样一致。

### 2. 选型与数量口径
- 客餐厅、走廊、卧室、书房使用同一款 `750x1500mm` 柔光砖或哑光砖通铺。
- 厨房墙面固定 `400x800mm` 易清洁釉面砖；厨房地面固定 `300x600mm` 防滑砖。
- 主卫和公卫墙面固定 `400x800mm`；地面固定 `300x600mm` 防滑砖。
- 大砖通铺主砖下单约 `95.7㎡`，约 `86片`。
- 厨房墙砖下单约 `25.4㎡`，约 `80片`；厨房地砖下单约 `10.8㎡`，约 `60片`。
- 主卫墙砖下单约 `31.8㎡`，约 `100片`；公卫墙砖下单约 `28.6㎡`，约 `90片`。
- 两个卫生间防滑地砖各按 `30片左右` 复核。
- 主砖损耗按 `10%`，厨房和两个卫生间墙地砖损耗按 `6%-8%` 估算。
- 上述面积和片数仅作为预算测算口径，正式下单数量以复尺、排版和双方签字确认的算量表为准。

### 3. 海棠角、加工和补货条款
- 海棠角必须在签约前写清：`是否按米数计价`、`是否已含在合同总价内`、`加工失败或爆边补货由谁承担`。
- 地漏、门洞、窗边、包管、阳角、柜体边、卫生间门口收口必须在铺贴前先做交底，不接受铺到现场再临时决定。
- 乙方应按复尺和排版图分批次组织到货；如需二次补货，必须尽量保证同品牌、同系列、同批次，并写清补货时限、运输责任和停产断货时的替代处理口径。
- 因乙方分批混料、批次失控、加工错误、运输保护不当造成的色差、崩边、数量不足或返工，由乙方承担责任。

### 4. 到货、验收与付款
- 到货时必须核对包装型号、色号、批号、破损率和片数。
- 同一空间必须尽量同批次供货，显眼区域禁止混批。
- 明显崩边、裂纹、翘曲、严重色差砖不得混入正常铺贴区。
- 乙方须保留同批次备用砖，数量不少于铺贴面积的 `3%-5%`；备用砖是否计价、保留期限、剩余整砖退货口径和归属必须写清。
- 付款节点应至少分为：下单定金、到货核对、补货完成或最终结算；尾款应保留到到货核对和批次确认完成后支付。
