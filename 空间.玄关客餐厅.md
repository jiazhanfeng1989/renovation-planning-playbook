# 玄关客餐厅
## 玄关
- 做鞋柜、换鞋凳、挂衣区、临时置物区。
- 鞋柜要留常穿鞋区、儿童鞋区、雨伞区和快递临时放置区。
- 入户通道不能被柜体压窄，换鞋凳不能影响入户门开启。
- 预留感应灯或玄关灯，晚上回家不用摸黑。

## 客厅
- 以电视 + 沙发为核心。
- 电视墙简洁处理，不做复杂造型和大面积岩板。
- 客厅对面柜体做收纳、展示和吧台/饮水区，不做厨房电器区。
- 客厅收纳做局部封闭柜，避免满墙压迫。
- 预留电视墙电源、网口、路由/机顶盒位置和客厅自动窗帘电源。

## 餐厅
- 餐桌按四口之家 + 偶尔老人/客人使用考虑。
- 餐边柜承担饮水、咖啡、小家电、杯子、零食和部分餐具收纳。
- 餐边柜不放蒸箱、烤箱、微波炉这些强厨房属性电器。
- 餐桌周边要保证通道，不要被餐边柜和冰箱挤压。


## 客餐厅柜体原则
- 柜子要做足，但不要为了收纳做满整面墙。
- 优先封闭柜，少量开放格展示。
- 儿童家庭要注意柜体固定、防攀爬、防夹手。
- 柜体颜色跟全屋现代简约风格一致，减少杂乱感。

## 必须确认
- 玄关柜、客厅柜、餐边柜必须和开关、插座、网口、弱电箱、空调风口避让清楚。
- 高柜和吊柜必须固定到墙体，儿童可接触柜体必须防倾倒。
- 客厅电视墙、全屋 WiFi、自动窗帘电机和餐边柜电源点位必须提前定稿。
- 电视墙必须预埋 `PVC 50管`，电视背后到电视柜设备区形成可穿线通道。
- 电视柜设备区必须满足开门、散热、走线和路由器检修，不能做成封死柜。
- 柜体板材、封边、铰链、滑轨和安装固定必须符合全屋定制文件要求。

## 玄关标准
- 玄关柜必须包含鞋柜、换鞋凳、挂衣区、临时置物区。
- 常穿鞋区留空，儿童鞋和成人鞋分区。
- 挂衣区必须能挂外套、书包和临时雨具，高度按成人外套和儿童书包分区。
- 玄关柜底部或高柜区预留行李箱位置，净高不低于 `70cm`，避免后期行李箱只能外放。
- 柜体固定到墙体，防止儿童攀爬倾倒。
- 入户通道不能被柜体压窄，入户门、柜门、换鞋凳不能互相打架。
- 玄关预留感应灯或低位灯，开门后有辅助照明。
## 客厅标准
- 客厅以电视墙和沙发为核心，不做复杂岩板背景墙。
- 电视墙预留电源、网口、机顶盒/路由设备隐藏位置。
- 客厅对面柜体只做收纳、展示、饮水吧台，不放蒸箱、烤箱、微波炉。
- 客厅柜体以封闭柜为主，开放格控制在少量展示区。
- 客厅自动窗帘预留电源、窗帘盒和检修空间。
- 电视墙至少预留 `4个强电插座 + 1个有线网口 + 1根PVC 50预埋管`，位置避开电视挂架。
- 50管从电视背后通到电视柜设备区，管内不穿强电线，只用于 HDMI、网线、音频线等弱电线缆走线。
- 50管两端使用成品穿线面板或可拆盖板，弯头不能做死弯，穿线前必须穿通测试。
- 电视柜设备区采用下翻门、抽拉托盘或大角度平开门，不使用难开启的小门和反弹器。
- 路由器、机顶盒、游戏机等设备格背板开孔，侧板或顶板开散热孔，高发热设备位置预留主动散热风扇电源。
- 沙发两侧预留手机充电、落地灯或边几插座，避免后期拖线板横跨地面。
- 客厅路由、机顶盒、游戏机等设备位必须有散热孔和检修空间，不能封死在柜体里。
- 客厅居中吊顶位置预留吸顶 AP 网线，避开金属灯具、中央空调出风口和检修口。
- 入户门内侧和客餐厅公共区预留监控摄像头点位，摄像头角度只覆盖公共区域。

## 餐厅标准
- 餐桌满足四口之家日常用餐和偶尔客人使用。
- 餐边柜承担饮水、咖啡、小家电、杯子、零食和部分餐具收纳。
- 餐边柜插座不少于 `3个`，饮水、小家电、备用充电分开使用。
- 餐桌周边保留餐椅拉出和日常通行空间。
- 餐边柜水吧区台面、背板和插座位置必须防潮、易清洁。
- 餐边柜小家电插座不得被层板或背板封死，后期能更换和检修。
- 餐桌正上方灯位必须和最终餐桌中心对齐，不按原始顶面灯位直接施工。
## 材料和安全要求
- 柜体板材、环保等级、封边、五金品牌必须在全屋定制下单文件中锁定。
- 柜体所有高柜、吊柜必须固定。
- 儿童可接触位置做圆角、防夹手和防攀爬处理。
- 插座避开柜体背板封死位置，后期能检修。
- 玄关、餐边柜、客厅柜的见光板、封板、收口条、灯带和拉手必须列入下单清单，避免安装时增项。
- 高柜旁边有弱电箱、配电箱、检修口的位置必须留可开启检修门。
- 电视柜设备区门板开启后不得挡住电源、网口、50管出口和路由器散热孔。
## 材料与选型
- 地面使用 `750x1500mm` 柔光或哑光主砖，和走廊、卧室、书房同砖通铺，不做过门石。
- 玄关柜、餐边柜、客厅柜使用 `ENF级板材`，柜体侧板、层板、门板不低于 `18mm`，背板不低于 `9mm`。
- 柜体封边采用 `PUR封边` 或 `激光封边`，现场开孔边做防潮封闭。
- 高频柜门使用 `百隆 CLIP top BLUMOTION / 海蒂诗 Sensys 8645i / DTC三段力阻尼铰链`。
- 柜门采用门板下延、45度斜切或 G 型拉手槽，不用高频反弹器和自动开合器。
- 开关插座采用 `西门子远景Pro / 施耐德绎尚 / 罗格朗逸远`，电视墙和餐边柜按实际点位统计数量。
- 电视墙、路由设备位使用 `Cat6 UTP 23AWG 无氧铜六类网线`。
- 墙面使用耐水腻子和乳胶漆，乳胶漆在 `立邦竹炭抗甲醛 / 多乐士森呼吸 / 三棵树鲜呼吸` 中选择。
## 需要提前定好尺寸
- 玄关柜深度、换鞋凳宽度、入户通道宽度。
- 沙发尺寸、电视尺寸、电视墙插座、网口、50管、挂架和电视柜设备区位置。
- 餐桌尺寸、餐边柜深度、餐厅通道宽度。
- 客厅自动窗帘盒和电源位置。
- 客厅吸顶 AP、入户监控、客餐厅监控点位。
