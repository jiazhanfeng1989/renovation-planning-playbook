# Renovation Planning Playbook

一个用于家庭装修准备、预算控制、材料采购、施工管理和现场验收的文档仓库。

仓库内容以中文整理，适合业主在装修前、中、后期按清单执行，也适合拿去和施工方、定制商、材料商现场对照。

## 包含内容

- 预算与开工前准备
- 施工工艺标准
- 材料采购总清单和现场打勾版
- 分空间、分专项设计与监管辅助文档
