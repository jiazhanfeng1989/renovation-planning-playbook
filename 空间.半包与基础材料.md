### 1.3 停工/复工/暂停付款触发矩阵

### 1.4 家装材料总清单和环保质量避坑
#### 必须重点管控的材料类别
- 板材类：全屋定制柜体板、门板、背板、层板、木门门芯、木地板或木饰面。
- 涂料类：墙面乳胶漆、木器漆、金属漆、界面剂、底漆和配套稀释体系。
- 胶粘剂类：瓷砖胶、背胶、建筑胶、结构胶、发泡胶、白乳胶、免钉胶。
- 密封和美缝类：防霉玻璃胶、耐候胶、美缝剂、环氧彩砂。
- 矿物和烧结类：瓷砖、岩板、石材、石英石台面、轻质砖、水泥、石膏板。
- 玻璃和淋浴隔断类：封窗玻璃、厨房推拉门玻璃、卫生间隔断玻璃、镜柜镜面、长虹玻璃。
- 管线和绝缘类：PPR 水管、BV 电线、线管、底盒、网线、保温棉。
- 防水类：防水涂料、堵漏材料、阴阳角加强材料。
- 五金类：门窗五金、柜体铰链滑轨、角阀、地漏、软管、锁体、合页。
- 成品家具和床垫类：床、床垫、沙发、成品柜、儿童家具。
#### 环保优先看的标准口径
- 板材甲醛强制限量现行执行 `GB 18580-2017`，实施日期为 `2018-05-01`；新标准 `GB 18580-2025` 已发布，将于 `2026-06-01` 实施。卧室、儿童房、书房和高柜量空间，优先要求按更严口径采购并留检测资料。
- 板材分级参考 `GB/T 39600-2021`，实施日期为 `2021-10-01`。本项目继续以 `ENF` 级作为卧室、儿童房、书房和主要定制柜体目标。
- 室内空气验收参考 `GB/T 18883-2022`，实施日期为 `2023-02-01`。材料环保不只看单品，还要看叠加后的整屋空气结果。
- 墙面涂料看 `GB 18582-2020`，实施日期为 `2020-12-01`；木器涂料看 `GB 18581-2020`，实施日期同为 `2020-12-01`。
- 胶粘剂 VOC 看 `GB 33372-2020`，实施日期为 `2020-12-01`；建筑胶粘剂有害物质看 `GB 30982-2014`，实施日期为 `2015-05-01`。
- 瓷砖、石材、岩板等矿物类材料关注 `GB 6566-2010` 建筑材料放射性核素限量，住宅室内用材需索取合格资料。
- 成品家具环保当前可参考 `GB 18584-2024`，实施日期为 `2025-07-01`，其已统一替代旧木家具和塑料家具有害物质限量要求。

#### 材料进场必须核对
- 包装标签：品牌、系列、型号、规格、颜色、等级、生产日期、执行标准。
- 批次一致性：同一空间尽量同批次，尤其瓷砖、板材、漆料和定制门板。
- 资料完整性：检测报告、合格证、送货单、防伪信息、条码照片。
- 现场一致性：到货材料与合同、报价单、签字图纸逐项核对，不一致不得上墙、上地、进柜。

### 1.7 半包最容易增项
- 水电按实结算：合同必须写清点位、米数、开槽、封槽、强弱电箱、回路、空开、漏保、网线和底盒单价，总价浮动规则写入合同。
- 拆改修补：拆墙、砌墙、门洞修补、梁柱找平、墙面基层修补必须按工程量提前列价。
- 找平和墙面基层：地面找平、墙面找平、铲墙皮、挂网、刷界面剂、阴阳角找直必须写清范围和单价。
- 防水增刷：卫生间、厨房、阳台防水高度、遍数、管根加强、门口加强、闭水复验必须写清，不接受后期按口头追加。
- 瓷砖铺贴：大砖上墙、瓷砖胶、背胶、海棠角、倒角、异形切割、地漏切割、美缝、搬运、补货必须提前列价。
- 吊顶和检修口：中央空调吊顶、窗帘盒、检修口、风口、灯槽、局部包管必须按米数或面积列价。
- 安装收口：门套收口、窗边收口、柜体收口、踢脚线收口、防霉胶、美缝修补必须写入报价。
- 垃圾清运和搬运：拆旧垃圾、施工垃圾、材料上楼、电梯保护、成品保护必须写清次数、楼层和费用。
- 材料替换：辅材品牌、系列、规格不得用“同级别”替代，替换必须业主书面确认并扣除差价。
### 2. 水管标准
- 冷热水管使用 `PPR管`，品牌在 `伟星 / 日丰 / 金牛 / 保利 / 联塑` 中选择。
- 主选型号为 `伟星 PPR S3.2 / PN2.0热水管`、`日丰 PPR S3.2 / PN2.0热水管`、`金牛 PPR S3.2 / PN2.0热水管`。
- 全屋冷热水主水路统一使用 `S3.2 / PN2.0热水管`，不使用低配冷水管替代。
- 仅少量末端短距离冷水支管经业主确认后使用 `S4 / PN1.6`，厨房、卫生间、阳台长期用水点仍按热水管标准执行。
- 厨房、卫生间、阳台主水路优先用 `25管`，末端按实际点位配置。
- 冷热水管分色或明确标识，左热右冷、上热下冷。
- 水管走顶或走墙，不大量走地，方便后期检修。
- 管卡固定牢固，热水管、冷水管保持合理间距。
- 水管穿墙、穿梁、转角位置做保护，不允许硬折和强行弯曲。
### 3. 水路测试
- 水路完工后必须做打压测试。
- 打压压力按 `0.8MPa` 执行。
- 稳压时间不少于 `30分钟`。
- 30分钟内压力下降不超过 `0.05MPa`，各接头无渗漏。
- 打压测试必须拍照留存压力表读数，由业主或监理签字确认。
- 未打压验收，不允许封槽、贴砖或做吊顶。
### 4. 电线和回路标准
- 水电定位前必须先核对入户电表容量、总开规格、原始配电箱位数和可扩展空间。
- 本项目存在中央空调、蒸箱、烤箱、洗碗机、烘干机、洗衣机、热水器等叠加负荷，必须按“同时使用场景”校核总负荷，不按单个设备分别够用来判断。
- 配电箱回路数量必须留 `15%-20%` 备用位，避免后期新增回路只能并线或外挂小箱。
- 电线使用国标单股铜芯线，品牌型号优先 `远东电缆 BV线 / 上上电缆 BV线 / 熊猫电线 BV线`。
- 照明回路使用不低于 `1.5平方毫米` 铜线。
- 普通插座回路使用不低于 `2.5平方毫米` 铜线。
- 厨房、卫生间、阳台大功率插座使用不低于 `4平方毫米` 铜线。
- 蒸箱、烤箱、洗碗机、烘干机、洗衣机等大功率设备独立或分区专用回路。
- 中央空调外机按设备功率配置独立回路和空开，外机电源线不低于设备厂家要求，常规按 `6平方毫米BV线` 复核。
- 冰箱单独回路，使用不低于 `2.5平方毫米BV线`，避免长期离家断其他插座时冰箱断电。
- 厨房大功率电器按功能分回路，蒸箱、烤箱、洗碗机不共用普通插排。
- 空调、地暖设备、厨房大功率电器按设备功率配置独立回路和空开。
- 强电和弱电分管分槽，平行间距不小于 `30cm`，交叉处做屏蔽处理。
- 线管内电线不得有接头，接线必须在底盒或检修位置完成。
- 同一线管内穿线不得过满，方便散热和后期检修。
- 配电箱回路必须标识清楚，厨房、卫生间、阳台、中央空调、地暖和大功率电器单独或分区配置漏保/空开。
- 卫生间、厨房、阳台湿区插座必须配置漏电保护，插座带防溅盒或防溅措施。
- 线管转弯不得死弯，强电管内穿线后能抽动，便于后期维修。
- 开关插座底盒必须横平竖直，盒内预留线长度满足安装检修。
### 4.1 电气测试
- 水电完工后必须做通断测试、绝缘测试和漏保测试。
- 每个插座必须检查火线、零线、地线接线正确，不允许缺地线、零火反接。
- 绝缘电阻测试按国标和监理要求执行，测试记录拍照留存。
- 绝缘测试必须逐回路记录，仪表型号、回路编号、测试时间和读数留档。
- 漏电保护器必须按测试按钮逐个测试，动作正常后才能验收。
- 配电箱回路标签必须和实际房间、设备一致，不能只写“插座1、插座2”。
### 5. 弱电和网络标准
- 网线使用 `六类非屏蔽网线 Cat6 UTP 23AWG 无氧铜`，按 305 米箱线进场。
- 网线品牌型号优先 `普天天诚 UTP-13-6-4P / 安普康 AMPCOM AMC655305 / 秋叶原 CHOSEAL QS2668`。
- 不使用铜包铝、铜包铁、成品跳线剪断后穿管。
- 书房、两个儿童房、电视墙必须预留有线网口。
- 弱电箱必须预留路由器、光猫、交换机电源和散热空间。
- 全屋 WiFi 点位在水电前确定，避免后期信号死角。
- 客厅预留 `1个吸顶 AP` 点位，位置放在客餐厅相对居中且避开中央空调出风口、灯具和金属遮挡处。
- 主卧、两个儿童房、书房预留 `86面板 AP` 网口，点位高度按普通网口面板统一，不能被衣柜、书桌、床头柜遮挡。
- 弱电箱内预留光猫、主路由、交换机、AC 控制器电源和散热空间。
- 监控摄像头点位采用网线 PoE 供电优先；入户门内侧、客餐厅公共区、阳台预留网线，卧室、卫生间不预留摄像头。
### 5.1 开关插座标准
- 开关插座优先 `西门子远景Pro / 施耐德绎尚 AvatarOn / 罗格朗逸远` 三个系列。
- 所有插座载流件要求锡磷青铜，面板要求阻燃 PC 材质。
- 普通五孔优先斜五孔，避免两孔和三孔互相打架。
- 厨房、卫生间、阳台、洗衣区、扫地机器人插座使用带防溅盒或防溅措施。
- 空调、烤箱、蒸箱、洗碗机、烘干机、洗衣机按设备功率配置 `16A` 或专用插座。
- 开关插座品牌、系列、颜色、五孔数量、16A数量、带开关插座数量必须写入合同。
### 5.2 角阀、玻璃胶和安装辅材标准
- 角阀由业主提前购买或指定品牌型号，不使用装修公司杂牌赠品。
- 角阀选择全通大流量角阀，材质采用 `全铜主体镀铬` 或 `304不锈钢主体`，接口规格为家装常用 `G1/2`。
- 角阀品牌在 `潜水艇 / 九牧 / 摩恩 / 科勒 / 箭牌` 中选择。
- 本项目按不少于 `14个角阀` 预备：厨房水槽冷热 `2个`、洗碗机 `1个`、净水器 `1个`、两个浴室柜冷热 `4个`、两个马桶 `2个`、两个马桶喷枪 `2个`、洗衣机 `1个`、扫地机器人基站 `1个`；现场增加用水点时同步增加。
- 厨房、卫生间、阳台收口使用厨卫防霉玻璃胶，品牌在 `道康宁 / 西卡 / 瓦克 / 百得 / 之江` 中选择。
- 水槽、台盆、马桶底边、淋浴房、台面挡水条使用中性防霉硅酮胶，不使用工人自带杂牌酸性玻璃胶。
- 玻璃胶施工前基层必须干燥、无灰、无油，打胶后胶缝连续饱满，不开裂、不发黑、不漏水。
### 5.3 热水系统配合标准
- 施工前必须明确全屋热水来源，是燃气热水器、壁挂炉生活热水还是其他方案，不能等橱柜和吊顶完成后再反推位置。
- 两个卫生间 + 厨房存在同时用热水场景，热水器容量、升数、零冷水方式和最远端等待时间必须提前评估。
- 采用零冷水时，必须确认回水管方案、耗气量、噪音、回水泵位置和后期维护，不默认零冷水一定更优。
- 热水器、角阀、止回阀、排烟管和检修空间必须在柜体、吊顶和封窗方案里同步避让。
### 6. 防水材料标准
- 防水材料品牌在 `东方雨虹 / 德高 / 西卡 / 立邦 / 三棵树` 中选择。
- 主选型号为 `东方雨虹 雨虹200柔韧型防水涂料`、`德高 K11柔韧型防水涂料`、`西卡 家装柔性防水涂料`。
- 卫生间墙地面使用柔性防水涂料。
- 厨房、阳台地面和有水区域使用正规防水涂料。
- 防水材料必须提供产品合格证和批次信息。
- 防水施工前基层必须清理干净，阴阳角、管根、地漏、门口先做加强处理。
### 7. 防水施工和闭水测试
- 卫生间地面满刷防水。
- 淋浴区墙面防水高度不低于 `1.8米`。
- 卫生间其他墙面防水高度不低于 `30cm`，洗手盆背墙不低于 `1.2米`。
- 阳台地面满刷防水，墙面上翻不低于 `30cm`。
- 防水至少涂刷 `2遍`，交叉涂刷，不能漏刷、堆积、起皮。
- 防水干透后做闭水试验。
- 卫生间闭水时间不低于 `48小时`。
- 阳台闭水或淋水测试不低于 `24小时`，按物业条件执行。
- 闭水期间检查楼下顶面、门口、管根、地漏周边，无渗漏后再贴砖。
- 闭水前必须拍照记录水位高度和开始时间，结束时复查水位、门口、墙根、楼下顶面。
- 闭水合格后不得随意破坏防水层；后续打孔、开槽、移位造成破坏必须重新补做并复验。
- 卫生间门口、淋浴区挡水、地漏和管根必须作为重点验收点。
### 8. 书房新增一路地暖配合标准
- 开发商已有地暖，硬装公司负责配合书房新增一路地暖的地面高度、回填找平和成品保护。
- 书房新增一路地暖由专业商家施工，硬装公司不得自行接管施工。
- 书房新增一路地暖管使用 `PE-RT II型` 或 `PEX-a阻氧管`。
- 地暖管品牌在 `乔治费歇尔 / 瑞好 / 伟星 / 日丰 / 保利` 中选择。
- 新增一路必须写清接入位置、阀门、过滤、回路标识和控制方式。
- 书房新增一路铺设完成后必须打压，压力按 `0.8MPa` 执行，保压不低于 `24小时`。
- 书房新增一路回填期间保持带压施工。
- 回填层不得使用尖锐石子，避免损伤地暖管。
- 全屋瓷砖完成面高度必须提前和门、柜体、卫生间门槛石统一核算。
### 9. 瓷砖和铺贴辅材标准
- 全屋通铺瓷砖，不做木地板。
- 客餐厅、卧室、书房优先柔光砖或哑光砖，防滑、耐脏、易清洁。
- 厨房、卫生间、阳台地砖必须防滑。
- 瓷砖胶品牌在 `德高 / 马贝 / 西卡 / 东方雨虹` 中选择。
- 大规格砖、墙砖上墙、吸水率低的砖，必须使用匹配等级瓷砖胶或背胶。
- 水泥、沙子、找平、美缝、阳角条、收边条的费用和责任必须写清。
- 瓷砖铺贴前必须排版，客餐厅通铺要对缝，门口、过道和显眼位置避免小窄条。
- 地漏周边必须做坡度，卫生间和阳台不得积水。
- 地面找平必须在铺砖前验收，使用 `2米靠尺` 检查平整度，明显高低差先整改再铺贴。
### 10. 瓷砖验收
- 瓷砖表面平整、缝隙均匀、无明显高低差。
- 墙砖、地砖空鼓率必须控制，单块砖边角和中间重点检查。
- 单块砖空鼓面积超过明显范围必须返工。
- 卫生间、阳台地面泼水后能流向地漏，不积水、不倒坡。
- 瓷砖阳角顺直，收口干净。
- 瓷砖阳角采用海棠角时，倒角边必须顺直，不崩边、不割手；儿童经常经过位置必须做钝化处理。
- 美缝前必须清缝，缝内无水泥、灰尘和积水。
### 10.1 资料归档和审批单
- 材料进场必须形成《材料进场验收单》，包含品牌、系列、型号、批次、数量、抽检照片、防伪或喷码、监理和施工方签字。
- 材料替换必须形成《材料替换审批单》，写清原约定、替换原因、替换后参数、差价、工期影响、质保影响和签字。
- 增项必须形成《增项审批单》，写清原合同范围、增加原因、工程量、单价、总价、责任归属和是否影响工期。
- 图纸变更必须形成《图纸变更单》，写清变更位置、前后做法、影响项、返工责任和复验要求。
- 隐蔽工程照片必须带房间、位置参照物、时间、尺寸定位和对应节点编号，重要测试同步留视频。
- 竣工资料除照片外，还必须交付尺寸定位图或竣工红线图，标注水电、地暖、空调、阀门、检修口和网口位置。
### 11. 吊顶标准
- 吊顶不做复杂造型，只为中央空调、风口、灯光、窗帘盒和局部遮管服务。
- 石膏板品牌在 `龙牌 / 泰山 / 可耐福` 中选择。
- 轻钢龙骨品牌在 `龙牌 / 泰山 / 可耐福` 中选择。
- 厨房、卫生间、阳台吊顶使用防潮材料，优先铝扣板或防潮石膏板方案。
- 中央空调内机、检修口、风口、回风口位置必须先确定再封板。
- 重型灯具、风口、电器不能直接吊在石膏板上，必须有独立加固。
- 石膏板接缝必须留缝、嵌缝、贴防裂带。
### 11.1 卫生间包管隔音标准
- 卫生间下水立管和横支管做隔音包管。
- 管道先用阻尼隔音毡或橡塑隔音棉紧密包裹，外层用扎带或铝箔胶带固定，不能留明显缝隙。
- 隔音层外侧采用红砖或轻体砖砌筑包管，表面抹灰找平后再做防水和贴砖。
- 包管位置必须预留检修口，检修口对应阀门、检查口或存水弯位置。
- 包管完成后敲击检查不空鼓，贴砖完成后无明显流水噪音放大。
### 12. 墙面和油工标准
- 腻子使用耐水腻子，品牌在 `美巢 / 立邦 / 德高 / 圣戈班` 中选择。
- 乳胶漆品牌在 `立邦 / 多乐士 / 三棵树 / 芬琳` 中选择。
- 腻子主选 `美巢耐水腻子 / 立邦耐水腻子 / 圣戈班耐水腻子`。
- 乳胶漆主选 `立邦竹炭抗甲醛系列 / 多乐士森呼吸系列 / 三棵树鲜呼吸系列`。
- 墙面基层必须铲除空鼓、起砂、开裂位置后再处理。
- 阴阳角必须找直，门洞、柜体交接处重点处理。
- 腻子不少于 `2遍`，每遍干透后再施工下一遍。
- 乳胶漆按 `一底两面` 施工。
- 乳胶漆色号、型号、环保等级和施工面积必须写入合同。
### 12.1 踢脚线标准
- 全屋明装踢脚线采用铝合金踢脚线，颜色选低饱和灰、黑钛或同墙地面接近色。
- 踢脚线高度采用 `40-60mm`，推荐 `50mm`；型材壁厚不低于 `1.2mm`。
- 安装采用卡扣式或专用底座固定，卡扣间距控制在 `35-60cm`，阴阳角使用专用配件收口。
- 踢脚线与墙面、地面贴合顺直，转角拼接严密，不翘边、不松动、不割脚。
- 柜体落地位置、门套位置和踢脚线位置必须提前统一，避免柜体安装后踢脚线断裂或外露。
### 13. 成品保护和现场管理
- 入户门、电梯、公共走廊、窗户、地面、已安装设备必须做成品保护。
- 中央空调内机、地暖管、封窗、瓷砖铺贴完成后必须保护。
- 现场材料进场必须核对品牌、型号、规格、数量。
- 隐蔽工程拍照留档：水管走向、电线走向、防水、地暖、中央空调管线。
- 垃圾清运、材料搬运、施工安全责任写入合同。
- 施工现场禁止使用业主已进场主材当脚手板、垫板或临时工作台。
- 每个阶段验收问题必须形成整改清单，整改完成并复验后才能进入下一工序。
- 竣工交付必须提供水电走向照片、配电回路表、材料合格证、检测记录和质保联系方式。
### 14. 付款节点
- 定金不超过半包合同总价 `20%`。
- 水电材料进场核对品牌型号后支付第一阶段款。
- 水电、防水、地暖配合、中央空调管线等隐蔽工程验收合格后支付第二阶段款。
- 瓦工、吊顶、油工分别验收合格后支付对应阶段款。
- 竣工验收、整改完成、资料交付后支付主要尾款。
- 保留 `5%-10%` 尾款，入住前复查无渗漏、开裂、空鼓和安装遗留问题后支付。
