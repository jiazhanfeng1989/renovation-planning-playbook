# 瓷砖
## 业主指定合同标准
### 分工边界
- 本项目瓷砖主材由业主采购，铺贴施工由工队负责。
- 业主负责确定瓷砖品牌、系列、型号、规格、色号、批号、防滑等级、数量和备用砖。
- 工队负责基层检查、找平、排版落地、铺贴施工、辅材使用、成品保护和阶段验收配合。
- 工队不得以“砖是业主买的”为理由降低基层处理、排版、铺贴、找坡、空鼓控制和成品保护标准。
- 业主自购瓷砖到场后，工队必须先做开箱核对、抽检和平铺预排，不得未核对就直接铺贴。
- 因业主采购型号错误、规格错误、数量不足导致停工或返工的，由业主承担对应责任；因工队排版错误、切割错误、铺贴错误、保护不当导致返工或损耗超出的，由工队承担责任。
### 1. 报价范围
- 报价必须拆分瓷砖主材、瓷砖胶、背胶、水泥砂浆、找平、美缝、阳角条、收边条、铺贴人工。
- 每个空间分别列瓷砖品牌、系列、型号、规格、数量、损耗率、单价和总价。
- 大规格砖、墙砖上墙、加工切割、倒角、海棠角、异形铺贴费用必须提前列明。
- 全屋通铺排版、对缝、切割、无过门石收口费用必须提前列明。
- 送货上楼、搬运、补货、退货、破损责任必须写清。
- 瓷砖排版图必须在铺贴前确认，显眼位置不能出现小窄条。
### 2. 瓷砖选择标准
- 客餐厅、走廊、卧室、书房使用同一款 `750x1500mm` 柔光砖或哑光砖通铺。
- 主砖品牌在 `马可波罗 / 东鹏 / 蒙娜丽莎` 中选择。
- 主砖色系采用低饱和暖灰、浅灰、米灰，不选强纹理和亮面砖。
- 主砖不选 `800x800mm`，避免大面积空间缝隙过多、视觉零碎。
- 主砖不默认选 `900x1800mm`，避免材料、搬运、铺贴和损耗成本过高。
- 厨房地面本项目固定采用 `300x600mm` 防滑砖，颜色与全屋主砖统一到同色系，不进 `750x1500mm` 大砖。
- 卫生间地面、淋浴区、阳台地面使用防滑砖，表面不能过亮过滑。
- 卫生间墙砖本项目固定采用 `400x800mm` 易清洁大砖，颜色和全屋主砖同色系。
- 厨房墙面本项目固定采用 `400x800mm` 易清洁釉面大砖，不使用复杂花砖、深凹凸纹理砖和小马赛克。
- 卫生间地砖按本项目最终方案执行：
- `普通点位地漏`：固定 `300x600mm` 防滑砖，不上 `750x1500mm` 大砖。
- 油烟机、灶具后方墙砖优先亮面或柔光釉面砖，减少油污附着，方便擦洗。
- 全屋颜色控制在低饱和中性色，和现代简约风格统一。
- 同一空间瓷砖必须同品牌、同系列、同批次，避免明显色差。
- 商家必须提供瓷砖包装、型号、色号、批号照片。
- 主砖必须确认耐磨、防污、防滑、吸水率和放射性检测信息，不能只看展厅样板。
- 瓷砖、岩板、石材等矿物类材料必须关注 `GB 6566-2010` 建筑材料放射性核素限量，商家应能提供对应合格资料。
- 厨房、卫生间、阳台地砖湿水后必须有明显摩擦感，优先选择防滑等级不低于 `R10` 或商家书面标注的同等防滑性能。
- 卫生间淋浴区不使用亮面砖、柔抛过滑砖和表面凹凸过深难清洁的砖。
### 2.1 这套房的最终建议搭配
- 客餐厅、过道、卧室、书房、阳台：统一 `750x1500mm` 主砖通铺。
- 厨房墙面：固定 `400x800mm` 易清洁釉面砖。
- 厨房地面：固定 `300x600mm` 防滑砖，颜色与全屋主砖同色系。
- 主卫和公卫墙面：固定 `400x800mm` 大砖。
- 主卫和公卫地面：固定 `300x600mm` 防滑砖，不做超大砖切坡。
- 卫生间墙面外阳角：固定做 `海棠角`。
- 卫生间墙地交界：固定做 `墙压地`，并在墙地阴角留 `柔性防霉硅酮胶`，不做硬性美缝顶死。
### 2.1A 海棠角和墙压地做法
- `海棠角`：本项目必须做。
- 原因：整体更利落，比阳角条更像完整石材面，更符合你要的效果。
- 但海棠角不是“切45度就完事”，必须控制崩边、大小头、接缝和割手风险。
- `墙压地`：本项目默认最优解。
- 原因：观感更整，墙砖压住地砖边，日常看起来更完整，也更容易把墙面水顺下来。
- 但无论墙压地还是地压墙，真正关键都不是“谁压谁”，而是 `墙地交界必须留变化缝，并用柔性胶处理`。
- 这里不建议用硬美缝把墙地阴角封死，因为变化位需要留一定形变空间。
### 2.1B 海棠角哪些位置必须做，哪些位置不建议做
#### 建议必须做海棠角的位置
- 两个卫生间墙面外阳角
- 卫生间门洞瓷砖包边显眼外角
- 卫生间包管外阳角
- 厨房墙砖外阳角和门洞显眼角
- 窗边、垭口边、壁龛外口这类明显展示面
- 客餐厅背景墙、端景墙、玄关墙砖或岩板外露边
#### 可以做海棠角，但要看现场条件的位置
- 阳台窗边、垭口边
- 厨房台面挡水上口附近墙砖转角
- 大砖上墙后的外包柱、外包梁边
#### 不建议强行做海棠角的位置
- 地漏四周和排水坡度复杂位置
- 卫生间地面外露尖角、容易踢碰的位置
- 门后、窄过道、儿童高频磕碰位的锐利低位角
- 柜体、五金、玻璃、门套安装后会被遮住的隐藏角
- 后期要经常拆装检修盖板、检修门的边口
- 瓷砖本身太薄、砖纹不适合、加工后容易爆边的特殊砖
#### 为什么这些位置不建议硬做
- 海棠角本质上比金属收边更脆，机械保护能力更弱。
- 外露低位、脚边、扫地机器人常撞位、儿童常碰位，后期更容易崩角。
- 地漏、坡度、下水收口是功能位，优先级必须是排水和耐久，不是先追刀锋感。
### 2.1C 品牌和价格路线
- 本项目瓷砖主材不追一线品牌溢价，但也不买来源不明的杂牌砖。
- 优先看 `佛山成熟二线 / 工厂系稳定品牌 / 本地靠谱总代仓库货`。
- 选砖时优先顺序是：
- `砖坯和平整度稳定`
- `表面质感和防滑达标`
- `同批次供货稳定`
- `补货规则清楚`
- `能提供包装型号、批号、检测资料`
- 不把钱花在品牌广告溢价上，但也不为了便宜去赌杂牌和尾货混批。
### 2.2 业主采购主材注意事项
- 先锁主砖，再锁厨卫阳台砖，避免全屋颜色和收口关系最后打架。
- 采购时必须同时拿到：包装型号照片、色号照片、批号照片、商家报价单、损耗率说明和补货规则。
- 同一空间必须尽量同批次采购，尤其是客餐厅、走廊、卧室、书房通铺主砖。
- 下单时必须把 `是否含送货上楼、搬运、破损补货、退换货、补货批次一致责任` 写清。
- 主砖采购量不能只按实测面积下单，必须按通铺排版后的面积加损耗计算。
- 本项目全屋通铺、750x1500 大砖、无过门石、对缝铺贴，主砖损耗建议按 `8%-12%` 预留；厨卫阳台墙地砖建议按 `5%-8%` 预留。
- 瓷砖到场时必须先挑出明显崩边、裂纹、翘曲、严重色差砖，不得混入正常铺贴区。
- 瓷砖到场后必须保留同批次备用砖，数量不少于铺贴面积的 `3%-5%`，显眼区域维修优先使用备用砖，不等后期补货碰运气。
- 采购时不要只看展厅灯光效果，要看自然光、侧光下的纹理、色差、反光和显脏程度。
- 厨房墙砖优先易清洁釉面砖；卫生间和阳台地砖优先防滑，不为了统一效果牺牲安全和排水。
- 海棠角必须在下单前写清：`是否按米计价`、`是否已含总价`、`加工失败或爆边补货由谁承担`。
### 2.3 本项目瓷砖数量和预算估算口径
#### 先说明
- 这一段是为了方便你问商家，不是最终下单单。
- 最终下单必须以 `复尺 + 排版图 + 门洞收口 + 地漏排版 + 墙面净尺寸` 为准。
- 但如果商家给你的面积、片数和这段偏差特别大，你就要追问它为什么多。
#### 当前估算口径
- 除厨房和两个卫生间外，其余地面都按大砖通铺。
- 客餐厅、过道、卧室、书房、两个阳台地面合并按约 `87㎡` 计算，全部使用 `750x1500mm` 主砖。
- 厨房墙面约 `24㎡`，地面约 `10㎡`。
- 主卫墙面约 `30㎡`，地面约 `5㎡`。
- 公卫墙面约 `27㎡`，地面约 `5㎡`。
- 主砖损耗按 `10%`，厨房和两个卫生间墙地砖损耗按 `6%-8%`。
#### 换算结果
- 大砖通铺主砖 `750x1500mm`：下单约 `95.7㎡`，约 `86片`。
- 厨房墙砖 `400x800mm`：下单约 `25.4㎡`，约 `80片`。
- 厨房地砖 `300x600mm`：下单约 `10.8㎡`，约 `60片`。
- 主卫墙砖 `400x800mm`：下单约 `31.8㎡`，约 `100片`。
- 主卫地砖 `300x600mm`：下单约 `5.4㎡`，约 `30片`。
- 公卫墙砖 `400x800mm`：下单约 `28.6㎡`，约 `90片`。
- 公卫地砖 `300x600mm`：下单约 `5.4㎡`，约 `30片`。
#### 合理报价口径
- 这次默认按 `佛山成熟二线 / 工厂系高性价比` 口径问价，不按一线大牌溢价算。
- 大砖通铺主砖先按 `90元/片` 左右问价。
- 厨房墙砖先按 `22元/片` 左右问价。
- 厨房地砖先按 `14元/片` 左右问价。
- 两个卫生间墙砖先按 `22元/片` 左右问价。
- 两个卫生间防滑地砖 `300x600mm` 先按 `16元/片` 左右问价。
- 再额外单独问：`加工切割 / 上楼搬运 / 补货规则 / 美缝价格 / 收边条和阳角条 / 备用砖保留`。
#### 预算底线判断
- 只看砖本身主材，这套方案合理口径大约在 `1.5万左右`。
- 加上 `加工、搬运、补货预备、美缝、小料` 后，整套瓷砖主材合理口径大约在 `2.45万左右`。
- 这里便宜下来的，只是 `瓷砖品牌溢价`，不是便宜在辅料和施工上。
- 如果商家只报砖价很低，但后面加工、美缝、搬运、上楼单独很高，最终总价还是会超。
### 3. 全屋通铺标准
- 客餐厅、过道、卧室、书房连续通铺，不做过门石。
- 门洞位置瓷砖直接贯通，门口不单独切一条门槛石。
- 铺贴前必须出全屋排版图，从客餐厅主视觉面开始排版。
- 客厅、餐厅、过道、卧室门口保持对缝和通缝。
- 门口、走廊、电视墙、沙发区、餐桌区不能出现明显小窄条。
- 墙边、柜体下方、床下等不显眼位置吸收裁切损耗。
- 卫生间因防水、坡度和高差需要单独收口，地面使用防滑砖并独立找坡，门口使用极窄同色收边条或同色瓷砖收口，不做传统过门石。
- 阳台地面可接近全屋主砖颜色，但必须满足防滑、耐晒和排水坡度。
- 通铺会增加损耗和人工，合同里按全屋排版后的数量报价。
- 全屋通铺必须和书房补地暖完成面、阳台门口高差、卫生间挡水和扫地机器人越障能力一起复核。
### 4. 铺贴辅材标准
- 瓷砖胶品牌在 `德高 / 马贝 / 西卡 / 东方雨虹` 中选择。
- 低吸水率瓷砖、大规格砖、墙砖上墙必须使用匹配瓷砖胶。
- 大规格砖和低吸水率砖上墙时必须配合背胶或厂家指定界面处理。
- 背胶品牌在 `德高 / 马贝 / 西卡 / 东方雨虹` 中选择。
- 美缝剂或环氧彩砂品牌、颜色、施工范围必须写入合同。
- 卫生间、厨房、阳台地面美缝材料必须防霉、耐水、易清洁。
### 5. 铺贴工艺
- 铺贴前基层必须平整、坚实、无空鼓、无起砂、无浮灰。
- 工队在铺贴前必须先完成基层复核，并明确哪些位置需要找平、修补、补直角、补垂直，不得边铺边发现问题边加价。
- 铺贴前完成排版，客餐厅、过道、门口、电视墙等显眼位置优先整砖。
- 工队必须按签字排版图施工，不得现场擅自改排版、随意改砖缝、随意把小窄条甩到显眼位置。
- 大规格砖使用瓷砖胶薄贴或厂家认可工艺，不用厚水泥砂浆硬贴。
- 墙砖上墙必须检查基层垂直度和平整度。
- 地砖铺贴必须控制通缝、对缝、平整度和高低差。
- 厨房、卫生间、阳台地面必须向地漏找坡。
- 卫生间地漏周边采用回字形或斜切做法，坡度方向必须明确，不能为追求大砖通铺牺牲排水。
- 地漏、门洞、窗边、包管、阳角、踢脚线交界、柜体边、地暖高差和卫生间门口收口必须在铺贴前先做交底，不接受工队铺到现场再临时决定。
- 铺贴完成后做好成品保护，未达到强度前不允许重物碾压。
- 地面找平完成后必须检查完成面高度，和入户门、房门、柜体、地暖、卫生间挡水收口统一。
- 大规格砖搬运、上楼、切割、铺贴必须使用合适工具，崩边、裂纹、明显色差砖不得上墙上地。
- 墙砖上墙前必须确认墙面垂直度和基层强度，基层不合格先整改再铺贴。
- 地漏周边必须先确认地漏型号和排水方式，再切割瓷砖。
- 海棠角必须先试拼再大面积切割，不接受边切边赌尺寸。
- 海棠角完成后外露边不能过尖，避免后期磕碰崩角和割手。
- 卫生间墙地阴角按 `墙压地` 做法施工，阴角位置留柔性缝，不做刚性硬碰硬。
### 5.3 海棠角和墙压地工艺标准
- 所有外阳角海棠角在下单、排版和施工前必须先交底清楚，哪些位置做海棠角，哪些位置如地漏边、隐藏区可不做，要提前定。
- 海棠角优先用于：卫生间墙面外阳角、门洞包边显眼角、包管外角、窗边显眼外角。
- 海棠角不要求切成极薄刀口，保留适度钝边，重点是 `顺直、细腻、不崩边、不割手`。
- 海棠角两边砖纹、色差、缝宽要顺，不接受一边厚一边薄、角度乱飘。
- 低位易碰撞区如果坚持做海棠角，必须接受后期崩角风险高于金属收边条。
- 墙压地时，地砖先按坡度和完成面找平落地，墙砖下口压住地砖边。
- 墙地阴角必须预留变化缝，优先使用 `防霉中性硅酮胶` 或同等级柔性密封材料处理，不用普通白水泥或硬质美缝剂顶死。
- 卫生间湿区、淋浴区、地漏周边、门口收边必须先看排水和坡度，再谈海棠角和观感。
- 湿区大砖上墙或大规格墙砖施工时，粘结必须满足湿区覆盖率要求，不能空鼓偷工。
### 5.1 工队施工闭坑重点
- 不接受工队口头说“这砖不好铺所以要加钱”，所有大砖上墙、海棠角、异形切割、地漏切割、对缝通铺费用必须在开工前写清。
- 不接受工队为了省事把显眼位置整砖改成边角小条，把问题藏到门口、过道、沙发区、电视墙前。
- 不接受工队用业主瓷砖做临时垫板、搅拌台、踩踏通道或切割废料试机。
- 不接受工队边铺边随意调缝，导致同一空间缝宽忽大忽小。
- 不接受工队为了赶工在基层未干、找平未验收、防水未复验时直接铺砖。
- 不接受工队为了省料省工，在低吸水率砖和大规格砖上跳过背胶、错用胶种或随意改工艺。
- 不接受工队把空鼓、崩边、色差砖铺在柜体外露区、门口、视线中心区。
- 不接受卫生间、阳台因赶排版效果放弃排水坡度和地漏周边排水。
### 5.2 工队必须承担的责任
- 负责铺贴前基层验收和问题上报。
- 负责按签字排版图施工。
- 负责瓷砖胶、背胶、美缝、收口等辅材按合同使用。
- 负责地漏找坡、空鼓控制、通缝对缝、阳角收口和成品保护。
- 负责施工损耗控制在正常范围内，不得因乱切、错切、误铺造成异常损耗。
- 负责把施工中发现的规格偏差、批次色差、缺砖和补货风险及时通知业主，不得先铺后说。
### 6. 防滑和安全
- 卫生间、公卫、阳台地砖必须做防滑优先选择。
- 公卫儿童使用频率高，地砖表面湿水后仍要有明显摩擦感。
- 淋浴区地面不能使用亮面砖。
- 卫生间地面湿区防滑等级按不低于 `R10` 执行，淋浴区优先选择更高防滑表面。
- 阳台地砖兼顾防滑、耐晒、耐污和排水。
- 瓷砖阳角采用海棠角或金属阳角条，儿童经常经过位置避免锐角。
### 7. 美缝和收口
- 美缝施工前必须清缝，缝内无水泥、灰尘、积水。
- 美缝颜色和瓷砖颜色接近，整体简洁耐看。
- 厨房、卫生间、阳台优先使用防霉耐水材料。
- 厨房油烟机后方、灶台两侧、卫生间淋浴区使用防霉、耐水、耐油污美缝材料。
- 门洞、地漏、墙地交界、柜体收口位置必须处理整齐。
- 全屋不做过门石，门洞位置采用同砖通铺或极窄同色收口。
- 地漏周边采用回字形或斜切排水做法，保证水流向地漏。
- 扫地机器人常走路径上的门口和阳台收口，优先控制高差和锐边，避免后期补坡或卡轮。
- 墙地阴角、墙墙阴角、不同材料交界变化位，不默认全部做硬美缝；湿区优先按柔性防霉胶处理。
### 8. 验收标准
- 全屋瓷砖整体空鼓率控制在 `5%以内`。
- 单块砖空鼓面积不超过该砖面积 `5%`。
- 单块砖中心位置不得空鼓。
- 墙面阳角、水电开孔周边、门口、地漏周边不得空鼓。
- 表面平整度用 `2米靠尺` 检查，明显高低差必须整改。
- 卫生间、阳台泼水后流向地漏，不积水、不倒坡。
- 瓷砖表面无明显崩边、裂纹、划伤、污染和色差。
- 相邻砖高低差不得明显硌脚，显眼位置高低差必须整改。
- 砖缝宽度必须按排版图和瓷砖规格控制，缝宽均匀，不能忽宽忽窄。
- 卫生间、阳台排水坡度必须现场泼水检查，水流不绕行、不滞留在墙边和门口。
- 空鼓检查必须在铺贴强度达到后进行，发现空鼓先标记、拍照、列整改清单。
### 9. 付款节点
- 瓷砖到场核对品牌、型号、色号、批号后支付材料款。
- 铺贴前确认排版图。
- 铺贴完成并通过空鼓、平整度、坡度验收后支付施工款。
- 美缝完成并验收后支付美缝款。
- 保留 `5%` 尾款，入住前复查无明显空鼓、开裂、脱落后支付。
- 瓷砖到场必须预留同批次备用砖，数量不少于合同铺贴面积的 `3%-5%`，用于后期维修。
- 补货色号、批号不一致造成色差的，不得用于显眼位置。
- 工队施工款必须和 `基层验收 + 排版确认 + 铺贴验收 + 整改闭环` 绑定，不接受只按工期或口头进度付款。
