# 全屋定制合同模板

打印和签约时，正文及连续装订页逐页签字或盖章，空白处划线作废。

## 签约信息
- 甲方：本房屋业主
- 乙方：全屋定制签约商家工商全称
- 项目地址：本项目房屋
- 联系人：甲方本人；乙方销售负责人、复尺负责人、安装负责人、售后负责人
- 联系电话：签约时填写
- 预估签约报价：`11.3万` 合理口径；高配口径约 `16-23.9万`
- 合同总价：按各空间定制报价汇总填写大写和小写金额
- 收款账户：乙方对公账户全称、开户行、账号
- 签约日期：签约时填写
- 备注：除上述签约信息、总价和时间节点外，其余柜体范围、材料、五金、图纸、安装、验收和付款条款按本合同固定执行。

## 适用范围
- 适用于柜体、柜门、书桌、台面、拉篮、抽屉、灯带、安装和售后。
- 本项目以 `半包 + 自购主材 + 单独签全屋定制` 为前提，合同必须能单独约束定制商的复尺、下单、供货、安装和售后责任。

## 本项目签约底线
- 合同必须逐项写清 `品牌 + 系列 + 基材 + 环保等级 + 厚度 + 饰面 + 颜色 + 封边工艺 + 五金品牌型号 + 数量`。
- 干区柜体、厨房柜体、阳台柜体、浴室柜柜体、柜门、背板、书桌板、台面必须分开写，不能只写“ENF板材”或“同级板材”。
- 柜体侧板、层板、门板不低于 `18mm`，背板不低于 `9mm`；长跨层板、悬空书桌、高门板、超高柜必须单独写结构加固方案。
- 封边采用 `PUR封边` 或 `激光封边`，开孔边、切割边和潮湿区裸露边必须做封闭防潮处理。
- 五金只接受 `百隆 / 海蒂诗 / DTC` 中的具体系列和型号，不接受 `品牌五金 / 套餐五金 / 同档次五金`。
- 厨房、阳台、卫生间柜体优先 `ENF级多层板 / ENF级防潮板`；普通干区优先 `ENF级实木颗粒板` 或已锁定的等效环保板材。
- 厨房和卫生间涉及台面、水槽、台盆、上下水、电器和阀门避让的，必须在下单图纸里标清，不允许安装时现场随意改。

## 合同模板

### 1. 合同主体与项目概况
- 甲方为本房屋业主，乙方为全屋定制供货及安装单位。
- 本合同总价以《签字版报价清单》《签字版复尺图纸》《材料与五金清单》《台面及开孔清单》为准。
- 未写入报价、图纸、材料清单和五金清单的项目，视为乙方已自行复核并纳入本合同，不得在下单后以 `常规增项 / 安装时再算 / 套餐外费用` 方式补收。

### 2. 报价与计价条款
- 报价必须按空间拆分：玄关、客餐厅、厨房、阳台、书房、主卧、两个儿童房、卫生间。
- 每个空间必须列明柜体尺寸、计价方式、单价、数量、总价和计价规则。
- 柜体板、门板、背板、书桌板、台面、抽屉、拉篮、玻璃门、见光板、封板、收口、灯带、变压器、安装费、搬运费、上楼费必须分项写清。
- 投影面积和展开面积不得混用；同一柜体采用哪一种计价方式必须写死，封板、见光板、顶封板、侧封板、收口条、拉直器、加固件不得重复计费。
- 异形柜、加厚板、超高柜、特殊开孔、台下盆、拉篮、抽屉、玻璃门、灯带、感应器、拉直器、加固件、台面异形切割必须提前列单价。
- 乙方完成现场复尺并经甲方确认签字图纸后，除甲方主动变更外，合同总价和各项单价不得再因漏量、漏项、复尺失误、图纸错误而上调。

### 3. 材料与五金条款
- 合同必须逐项写清板材品牌、系列、基材类型、环保等级、厚度、颜色、饰面和封边工艺。
- 柜门必须写清材质、厚度、颜色、封边方式，以及是否需要拉直器或加强筋。
- 背板必须写清厚度和材质，不接受“标配背板”“同级背板”。
- 书桌板必须写清环保等级、厚度、跨度、支撑方式和收边方式；悬空书桌必须写清固定方式和承重方案。
- 板材进场必须能对应到合同清单、包装标识、批次信息和检测资料；甲方在安装前有权按包装、标签和清单核验，不符可拒装。
- 铰链、滑轨、拉篮、反弹器、上翻支撑、抽屉侧板、挂衣杆、裤架、感应灯配套件必须写清品牌、系列、型号、数量。
- 高频使用柜门、抽屉和拉篮不得使用杂牌五金，不得以 `套餐五金 / 常规配置 / 品牌代工款` 替代。

### 4. 复尺、图纸与下单条款
- 下单前必须完成现场复尺，乙方必须对墙地完成面、吊顶完成面、踢脚线、门套、窗套、插座、网口、上下水、阀门、电器尺寸、排烟管、燃气表和检修口进行复核。
- 乙方必须提供平面图、立面图、内部结构图、五金清单、板材清单、台面及开孔图。
- 图纸必须标注长、宽、高、深度、门板开启方式、抽屉数量、拉篮数量、见光面、收口、踢脚避让、灯带、变压器、散热孔、穿线孔、检修口、上下水开孔和电器开孔。
- 电视柜设备区必须标清开门方式、背板开孔、散热孔、电源、网口、PVC 50 管出口和检修空间。
- 厨房柜体必须标清水槽、洗碗机、净水器、灶具、蒸箱、烤箱、微波炉、燃气表、燃气阀门、烟管、止逆阀和插座避让关系。
- 卫生间浴室柜必须标清镜柜电源、台盆、下水、角阀、插座和检修空间；不得封死阀门、下水和插座。
- 甲方书面确认签字图纸后方可下单；因乙方复尺、制图、下单错误导致返工、延期、增项、材料报废，或造成瓷砖、水电、吊顶、台面等已完成项目拆改恢复的，由乙方承担。

### 5. 柜体结构与专项做法条款
- 衣柜必须标明长衣区、短衣区、抽屉区、被褥区、行李箱区的内部净尺寸，不能只画外轮廓。
- 儿童房、玄关、书柜、高柜、吊柜必须有防倾倒固定方案，固定点位和连接件类型应与墙体条件匹配。
- 电视柜设备区必须可散热、可走线、可检修，不能做成封死柜。
- 书桌、悬空台面、长跨层板、高门板、薄门板、深色门板、肤感门板等易变形部位，乙方必须在图纸和报价中明确防变形方案；未写明的，视为乙方已包含满足正常使用要求的加固方案。
- 厨房石英石台面或其他台面必须写清品牌、系列、厚度、挡水、接缝位置、开孔加固、台下盆加固和防霉密封做法。
- 厨房、阳台、卫生间柜体开孔边、切割边、靠墙收口边必须做防潮封闭，水槽柜、洗碗机位、台盆柜重点加强。
- 燃气表、燃气阀门、冷热水阀、净水器滤芯、洗碗机进排水、地漏检修口、镜柜电源、变压器位置必须可见或可通过可拆结构检修，不得被固定柜体封死。
- 浴室柜相关的 `镜柜 / 灯带 / 插座 / 吹风机位 / 纸巾位 / 侧边收口` 必须在图纸、报价和安装责任中提前写清，不接受安装时再补项。

### 6. 安装、验收与付款
- 安装前核对板材、门板、背板、五金、台面、拉篮、灯带、变压器与签字图纸一致。
- 柜体安装必须横平竖直，门缝均匀，收口顺直，见光面完整，无明显崩边、划伤、色差异常和污染。
- 高柜、吊柜、书柜、衣柜、儿童房柜体必须可靠固定到墙体；轻体墙、空心墙等特殊墙体必须先确认加固方案。
- 柜体侧板、层板、门板、背板厚度应按合同约定抽检；厚度不符视为降配。
- 插座、阀门、上下水、检修口、散热孔、网口和穿线孔不得被封死或无法使用；如被封死，视为未验收合格。
- 定金不超过合同总价 `20%`；复尺和签字图确认后支付第二笔款；材料到场核对后支付第三笔款；安装完成并初验合格后支付主要尾款；保留 `5%-10%` 质保尾款。
- 柜体质保不低于 `5年`，五金质保不低于 `3年`，台面质保不低于 `3年`。
