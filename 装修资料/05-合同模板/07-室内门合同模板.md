# 室内门合同模板

打印和签约时，正文及连续装订页逐页签字或盖章，空白处划线作废。本合同按 `室内门成品供货 + 门套 + 五金 + 踢脚线 + 安装收口` 执行。

## 签约信息
- 甲方：本房屋业主
- 乙方：签约室内门商家工商全称
- 项目地址：本项目房屋
- 联系人：甲方本人；乙方销售、复尺、安装、售后负责人
- 联系电话：签约时填写
- 预估签约报价：`2.2万`
- 合同总价：按各樘门、门套、五金、踢脚线和安装合计填写大写和小写金额
- 收款账户：乙方对公账户全称、开户行、账号
- 签约日期：签约时填写

## 选型建议
- 本项目室内门按 `卧室/书房木门 + 厨房推拉玻璃门 + 卫生间铝合金玻璃门 + 配套门套线 + 五金 + 踢脚线 + 安装` 口径报价。
- 卧室、书房、儿童房优先选 `45mm` 厚静音实木复合门，不建议选 `40mm` 及以下薄门。
- 门芯建议选 `桥洞力学板 / 实芯复合填充 / 稳定防潮复合芯材`，不接受蜂窝纸芯、稀疏木龙骨空心芯。
- 厨房门当前方案固定为 `推拉玻璃门`，重点控制轨道顺滑、缓冲、玻璃安全和收口。
- 卫生间门当前方案固定为 `铝合金玻璃门`，重点控制防潮、防锈、玻璃安全、锁具和收口。

## 合同模板

### 1. 适用范围
- 本合同适用于卧室、书房、厨房、卫生间等室内门系统。
- 乙方负责复尺、下单、供货、运输、安装、调试、基础收口和售后。
- 乙方负责和半包施工、瓷砖、地面完成面、墙面完成面、全屋定制、封窗收口关系做复核。

### 2. 报价范围
- 报价必须写清：每樘门的位置、数量、门扇尺寸、开启方向、门套、门套线、五金、门吸、门底密封、踢脚线、安装、搬运、收口。
- 不接受只写 `静音门 / 实木复合门 / 门套另算 / 安装另算`。
- 垭口、门套加宽、单包双包、门套线、踢脚线、五金升级、锁具、门吸、门底密封、安装上楼和二次上门费用必须在签约前一次写清。
- 除甲方主动变更外，结算增幅不得超过签约总价 `5%`。

### 3. 材料和品牌建议
- 门品牌建议在 `TATA / 梦天 / 霍尔茨 / 鑫迪 / 美心 / 欧派木门 / 尚品本色 / 金迪` 中锁定唯一品牌和系列。
- 五金品牌建议在 `DTC / 顶固 / 汇泰龙 / 海福乐 / 海蒂诗` 中锁定具体型号。
- 卧室、书房、儿童房木门建议锁定：`门扇厚度不低于45mm + 磁吸静音锁 + 三边静音条 + 门底密封`。
- 厨房推拉玻璃门建议写清：`8mm钢化玻璃优先 + 单轨或双轨形式 + 缓冲器 + 拉手 + 是否带地轨/吊轨`。
- 卫生间铝合金玻璃门建议写清：`磨砂钢化玻璃 + 铝材系列 + 开启方式 + 门锁形式 + 应急开启`。
- 踢脚线如由门商一并供应，必须和门套、门色、安装范围一起写进合同，不接受后期只送门不管踢脚线收口。

### 4. 门锁配置建议
- 门锁不要只写“静音锁”或“磁吸锁”，必须写清 `品牌 + 系列 + 型号 + 颜色`。
- 卧室、书房、儿童房建议统一用 `磁吸静音锁体 + 分体或一体把手`。
- 卫生间门锁必须写清是否带 `应急开启`。
- 厨房推拉玻璃门和卫生间铝合金玻璃门的锁具、拉手、缓冲、限位方式必须单列。

### 5. 供货和安装要求
- 乙方必须在墙地完成面基本确定后复尺，不得按毛坯洞口直接下单。
- 乙方必须和地砖完成面、卫生间挡水、书房补地暖完成面、柜体侧板收口、踢脚线高度一起复核。
- 门扇安装后应保证开合顺畅、门缝均匀、无明显下坠、反弹和擦挂。
- 厨房推拉玻璃门要重点检查轨道顺滑、缓冲器、玻璃安全、收口与门洞垂直度。
- 卫生间铝合金玻璃门要重点检查防潮、防锈、锁具、应急开启、玻璃厚度和门口收口。

### 6. 验收、付款与售后
- 逐樘核对门扇数量、尺寸、开启方向、五金数量和踢脚线安装范围。
- 门扇安装后应无明显磕碰、划伤、翘曲、色差异常和污染。
- 门锁、合页、门吸、门底密封开合正常，无异响和松动。
- 门套和墙面、地面、踢脚线收口顺直，无明显大缝和歪斜。
- 定金或首期预付款累计不超过合同总价 `20%`；复尺、下单图和报价清单确认后累计付款不超过 `50%`；材料到场核对后累计付款不超过 `80%`；安装完成并验收合格后支付至 `95%`；保留 `5%` 质保尾款。
- 门扇、门套安装质保不低于 `2年`；五金按厂家质保执行；门扇下坠、门锁故障、合页松动、门套开裂、门吸松脱、门底密封失效等，乙方应在 `48小时` 内响应。

## 最容易踩坑的点
- 厨房推拉玻璃门只写“玻璃门”，不写玻璃厚度、缓冲、轨道形式。
- 卫生间铝合金玻璃门只写“铝合金门”，不写磨砂、锁具、应急开启。
- 只写“静音门”，不写门芯结构、门扇厚度、门锁型号。
- 先报低门价，后续再把门套加宽、垭口、锁具升级、静音条、踢脚线、安装费、上楼费逐项加回去。

- [ ] 门扇、门套、门套线、五金、门吸、门底密封、踢脚线、安装是否包含已写清。
- [ ] 门品牌、系列、门芯结构、门扇厚度已写清，不是只写“实木复合门”。
- [ ] 卧室和书房木门已锁 `45mm` 及以上门扇厚度。
- [ ] 锁体、锁具、合页、门吸、门底密封品牌型号已写清。
- [ ] 卫生间门锁带 `应急开启` 已写清。
- [ ] 厨房推拉玻璃门和卫生间铝合金玻璃门已单独拆开写清。
- [ ] 厨房玻璃门玻璃厚度、轨道、缓冲；卫生间玻璃门磨砂、锁具、应急开启都写清。
- [ ] 垭口、门套加宽、单包双包、收口责任已写清。
- [ ] 付款节点按复尺、到货、安装验收走，尾款保留 `5%`。
- [ ] 合同中没有 `同级 / 标配 / 安装时再定 / 门套另算` 这类模糊词。
