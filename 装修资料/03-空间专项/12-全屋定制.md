# 全屋定制
## 最终方向
- 全屋定制以环保、耐用、收纳效率、儿童安全为核心。
- 重点柜体包括玄关柜、餐边柜、电视/客厅收纳柜、橱柜、电器高柜、阳台家政柜、书柜、衣柜、儿童房柜体、浴室柜。
- 合同必须按“板材品牌 + 环保等级 + 厚度 + 封边工艺 + 五金品牌型号 + 柜体结构 + 安装验收”签。
- 不接受只写“环保板材”“进口五金”“品牌五金”“同款同质”等模糊说法。
- 各空间柜体、柜门、书桌材料和配色，统一执行本项目已确定的全屋定制材料与配色规则。
### 1. 报价范围
- 报价必须按空间拆分：玄关、客餐厅、厨房、阳台、书房、主卧、两个儿童房、卫生间。
- 每个空间分别列柜体长度、投影面积、展开面积、单价、总价。
- 报价必须列明柜体板、门板、背板、台面、封边、铰链、滑轨、拉篮、拉手、灯带、安装、收口。
- 非标柜、异形柜、见光板、加厚板、抽屉、拉篮、灯带、玻璃门、上楼搬运费用必须提前列明。
- 增项必须先报价，业主书面确认后制作。
- 报价必须写清投影面积和展开面积的计算规则，封板、见光板、顶线、侧封、收口条不得重复计费。
- 抽屉、拉篮、灯带、感应器、台面开孔、台下盆、异形切割必须逐项写单价和数量。
- 下单后因商家漏量、漏报、图纸错误产生的增项和返工，由商家承担。
### 2. 材料和五金执行原则
- 柜体、柜门、书桌板、背板、封边、五金品牌型号、颜色和潮湿区基材，统一执行 `D01 全屋定制材料与配色规则`。
- 本文件不重复展开材料选型细节，签约时直接把 `D01` 作为合同附件，按附件执行。
- 合同必须写清 `品牌 + 系列 + 基材类型 + 环保等级 + 厚度 + 封边工艺 + 五金品牌型号 + 颜色`。
- 厨房、阳台、浴室柜必须与普通干区柜体分开写基材，不接受全屋笼统写成“ENF板材”。
- 板材、门板、背板、封边条、五金进场时必须能对应到合同清单、包装标识和检测资料。
- 安装完成后保留通风时间，卧室、儿童房、书房优先做空气检测，不合格不入住。
### 3. 各空间功能要求
- 玄关柜包含鞋柜、换鞋凳、挂衣区、常穿鞋区、临时置物区。
- 玄关柜挂衣区必须能挂成人外套、儿童书包和临时雨具。
- 玄关柜或客厅储物柜预留行李箱区，净高不低于 `70cm`，深度不低于 `55cm`。
- 餐边柜承担饮水、小家电、杯子、零食和餐具收纳，不放蒸箱、烤箱、微波炉。
- 电视柜包含设备区，设备区放路由器、机顶盒、游戏机或影音设备，必须可开门、可散热、可检修。
- 电视柜设备区采用下翻门、抽拉托盘或大角度平开门，禁止使用难开启小门和高频反弹器。
- 电视柜设备区背板开孔，侧板或顶板开散热孔；高发热设备预留主动散热风扇电源。
- 橱柜包含水槽柜、洗碗机位、灶台柜、锅具抽屉、调料拉篮、电器高柜。
- 阳台家政柜包含洗烘区、扫地机器人基站位、电动拖把位、清洁工具区和纸品耗材区。
- 书房柜以封闭书柜和资料柜为主，开放格控制比例。
- 主卧衣柜包含长衣、短衣、抽屉、被褥、行李箱和换季区。
- 儿童房衣柜和玄关储物必须预留大件收纳格，放行李箱、换季被褥和大件玩具。
- 儿童房柜体包含衣物、书籍、玩具、文具分区。
- 浴室柜必须防潮、易清洁，并预留镜柜电源。
### 4. 复尺和图纸
- 下单前必须完成现场复尺。
- 商家必须提供平面图、立面图、柜体内部结构图、五金清单、板材清单。
- 每个柜体必须标注长、宽、高、深度、门板开启方式、抽屉数量、拉篮数量。
- 水电点位、插座、网口、上下水、扫地机器人、洗碗机、蒸箱、烤箱、微波炉必须和柜体图对齐。
- 图纸确认后再生产，生产后变更费用由责任方承担。
- 图纸必须标注见光面、封板、收口、灯带、变压器、检修孔、踢脚线、台面开孔和避让位置。
- 图纸必须标注每个抽屉、拉篮、铰链、滑轨、挂衣杆、层板、灯带的数量和型号。
- 图纸必须标注电视柜设备区开门方式、散热孔、背板开孔、PVC 50管出口、电源插座和网口。
- 图纸必须标注行李箱区、换季被褥区和大件储物区尺寸。
- 图纸签字版作为生产依据，商家不得用未确认版本下单。
- 生产前必须完成水电复核，确认插座、阀门、网口、上下水和检修口不会被柜体封死。
### 5. 安装工艺
- 安装前核对板材、门板、五金、台面、拉篮、灯带和图纸。
- 柜体安装必须横平竖直，门缝均匀。
- 柜体与墙面、地面、顶面收口整齐。
- 台面开孔位置准确，水槽、灶具、净水器、洗碗机、蒸箱、烤箱、微波炉满足安装要求。
- 柜体背后有插座、阀门、检修口的位置必须留检修孔。
- 安装完成后清洁现场，保护地面、墙面、瓷砖、门窗。
- 高柜、吊柜、书柜、衣柜必须使用可靠连接件固定到墙体，安装完成后现场晃动检查。
- 吊柜安装必须确认墙体承重条件，轻体墙或空心墙必须加固，不得直接硬装。
- 台面、水槽、灶具、洗碗机、蒸箱、烤箱、微波炉安装后必须逐项试装和开合检查。
- 灯带、感应灯、变压器、开关安装后必须通电测试，变压器位置可检修。
### 6. 质保和售后
- 柜体质保不低于 `5年`。
- 五金质保不低于 `3年`。
- 台面质保不低于 `3年`。
- 板材开裂、封边开胶、柜门变形、铰链下坠、滑轨故障必须明确售后责任。
- 售后响应时间不超过 `48小时`。
### 7. 付款节点
- 定金不超过合同总价 `20%`。
- 复尺、图纸、材料清单、五金清单确认后支付第二笔。
- 材料进场核对品牌、型号、数量后支付第三笔。
- 安装完成并验收合格后支付主要尾款。
- 保留 `5%-10%` 质保尾款，使用一段时间无明显问题后支付。
## 签约和验收检查
### 签约前检查
- [ ] `D01 全屋定制材料与配色规则` 已作为合同附件或报价附件引用。
- [ ] 环保等级、唯一品牌、唯一系列、唯一基材已写清。
- [ ] 干区和潮湿区柜体基材已明确区分。
- [ ] 柜体、门板、背板厚度和封边工艺已写清。
- [ ] 铰链、滑轨、拉篮等五金品牌、系列、型号、数量已写清。
- [ ] 报价按柜体、门板、背板、台面、五金、抽屉、见光板、收口、安装、搬运拆分清楚。
- [ ] 图纸已标注尺寸、结构、开孔、避让、散热和检修位置。
- [ ] 付款节点按验收走，质保尾款保留 `5%-10%`。
- [ ] 不接受“环保板、ENF同级、进口五金、品牌五金、套餐总价、常规五金、现场再开孔”。
### 验收时检查
- [ ] 板材品牌、系列、环保等级、厚度与合同一致。
- [ ] 商家提供 ENF 级检测报告和板材批次信息。
- [ ] 柜体侧板、层板、门板不低于 18mm，背板不低于 9mm。
- [ ] 封边为 PUR 或激光封边，无翘边、开胶、明显胶线。
- [ ] 铰链、滑轨、拉篮品牌型号与合同一致。
- [ ] 高频柜门铰链为可调三维十字底座阻尼铰链，数量和开合角度与图纸一致。
- [ ] 柜门开合顺畅，门缝均匀，无异响、下坠、明显晃动。
- [ ] 抽屉全拉出顺畅，关闭有阻尼缓冲。
- [ ] 抽屉斜角拉手或 45 度斜切拉手顺滑不割手。
- [ ] 高频柜门无反弹器、无自动开合器，免拉手边缘不割手。
- [ ] 电视柜设备区开门顺畅，散热孔、背板开孔、50管出口、电源和网口可检修。
- [ ] 玄关、衣柜或储物柜预留行李箱和大件储物空间。
- [ ] 高柜、书柜、衣柜、吊柜固定牢固。
- [ ] 儿童房和玄关柜体防倾倒、防夹手、防尖角处理到位。
- [ ] 厨房、阳台、卫生间柜体防潮收口到位。
- [ ] 插座、阀门、上下水、检修口没有被柜体封死。
- [ ] 台面、水槽、灶具、洗碗机、电器高柜开孔尺寸准确。
- [ ] 现场安装完成后无明显磕碰、划伤、污染和收口粗糙。
