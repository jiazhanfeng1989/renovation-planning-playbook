## 合同模板

### 1. 适用范围
- 本合同适用于卧室、书房、厨房、卫生间等室内门系统。
- 乙方负责复尺、下单、供货、运输、安装、调试、基础收口和售后。
- 乙方负责和半包施工、瓷砖、地面完成面、墙面完成面、全屋定制、封窗收口关系做复核。

### 2. 报价范围
- 报价必须写清：每樘门的位置、数量、洞口完成尺寸、墙厚、门扇尺寸、开启方向、门套、门套线、单包或双包、五金、门吸、门底密封、踢脚线、安装、搬运、收口。
- 不接受只写 `静音门 / 实木复合门 / 门套另算 / 安装另算`。
- 垭口、门套加宽、单包双包、非标高、非标宽、门套线、踢脚线、五金升级、锁具、门吸、门底密封、安装上楼和二次上门费用必须在签约前一次写清触发条件和单价。
- 除甲方主动变更、现场尺寸变化、墙地未干、含水率超标或他方施工影响外，结算增幅不得超过签约总价 `5%`。

### 3. 材料和品牌建议
- 本合同《默认产品配置》、逐樘报价清单、复尺单和签字版下单图具有同等效力；除甲方书面同意外，乙方不得以“同级替代”方式更换门扇、门套、玻璃、锁具、合页、门吸和门底密封。
- 木门品牌锁定 `TATA 至臻系列磁吸降噪门`。
- 木门五金锁定 `海福乐磁吸静音锁 + DTC 5寸304合页 + DTC地吸 + 海福乐自动门底密封`。
- 卧室、书房、儿童房木门建议锁定：`门扇厚度不低于45mm + 石墨匀质板门芯 + 磁吸静音锁 + 三边静音条 + 自动门底密封`。
- 厨房推拉玻璃门锁定：`新豪轩 16极窄吊轨推拉门 + 6063-T5铝合金 + 型材壁厚不低于1.4mm + 8mm带3C标识超白钢化玻璃 + 悍高缓冲吊轮 + 悍高防摆器 + 304拉丝一字拉手`。
- 卫生间铝合金玻璃门锁定：`新豪轩 35窄边平开卫浴门 + 6063-T5铝合金 + 型材壁厚不低于1.4mm + 8mm带3C标识磨砂钢化玻璃 + 悍高卫浴锁应急开启 + 防潮收口`。
- 踢脚线如由门商一并供应，必须和门套、门色、安装范围一起写进合同，不接受后期只送门不管踢脚线收口。

### 4. 门锁配置建议
- 门锁不要只写“静音锁”或“磁吸锁”，必须写清 `品牌 + 系列 + 型号 + 颜色`。
- 卧室、书房、儿童房建议统一用 `磁吸静音锁体 + 分体或一体把手`。
- 卫生间门锁必须写清是否带 `应急开启`。
- 厨房推拉玻璃门和卫生间铝合金玻璃门的锁具、拉手、缓冲、限位方式必须单列。
- 合页常规按 `3只/樘` 计，超高门、超宽门或重门扇必须单列增加合页数量和型号。
- 门扇颜色、木纹方向、玻璃效果和五金颜色，建议通过色卡、样件或封样页确认。

### 5. 供货和安装要求
- 乙方必须在墙地完成面基本确定后复尺，不得按毛坯洞口直接下单。
- 乙方必须和地砖完成面、卫生间挡水、书房补地暖完成面、柜体侧板收口、踢脚线高度一起复核。
- 若墙地未干、湿作业未结束、洞口未收方、挡水未完成或完成面仍可能变化，乙方有权书面要求暂缓复尺或安装，并约定顺延工期和二次上门费用口径。
- 门扇安装后应保证开合顺畅、门缝均匀、无明显下坠、反弹和擦挂。
- 厨房推拉玻璃门要重点检查轨道顺滑、缓冲器、玻璃安全、收口与门洞垂直度。
- 卫生间铝合金玻璃门要重点检查防潮、防锈、锁具、应急开启、玻璃厚度和门口收口。
- 厨房门洞净宽、净高、墙厚、地面完成面高差、门套压边、踢脚线断点和门边柜体关系必须在复尺单和下单图上单独标清。
- 卫生间门下口、挡水、门槛石、门套基层防潮、门口高差和开门避让关系必须在复尺单和下单图上单独标清。

### 6. 验收、付款与售后
- 逐樘核对门扇数量、尺寸、开启方向、门套展开宽度、五金数量和踢脚线安装范围。
- 门扇安装后应无明显磕碰、划伤、翘曲、色差异常和污染。
- 门锁、合页、门吸、门底密封开合正常，无异响和松动。
- 门套和墙面、地面、踢脚线收口顺直，无明显大缝和歪斜。
- 定金或首期预付款累计不超过合同总价 `20%`；复尺、下单图和报价清单确认后累计付款不超过 `50%`；材料到场核对后累计付款不超过 `80%`；安装完成并验收合格后支付至 `95%`；保留 `5%` 质保尾款。
- 门扇、门套安装质保不低于 `2年`；五金按厂家质保执行；门扇下坠、门锁故障、合页松动、门套开裂、门吸松脱、门底密封失效等，乙方应在 `48小时` 内明确回复，并在合同中写清紧急问题到场时限。
