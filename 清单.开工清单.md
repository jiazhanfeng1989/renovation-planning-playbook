# 02 装修前 Checklist
## 原则

本文件只记录开工前业主需要准备和确认的事项。

## 不准备不能开工
### 1. 物业和现场条件
- [ ] 确认封窗、换窗许可，外立面颜色和窗型限制。
- [ ] 确认开发商已有地暖情况、书房新增一路地暖许可、接入位置和分集水器要求。
- [ ] 确认装修时间、押金、垃圾清运、电梯保护、材料进场路线。
- [ ] 拿到物业装修手册或书面要求。
- [ ] 向物业或开发商索取户型实测尺寸图、原始地暖铺设图、原始水电图、结构承重图。
- [ ] 确认入户电表容量、空开规格、是否支持后期增容，核对中央空调、烤箱、蒸箱、洗碗机、烘干机同时使用时的总负荷余量。
- [ ] 确认燃气表位置、燃气热水器安装条件、烟道/排烟条件和燃气公司改管规则。
- [ ] 完成全屋现场复尺，记录厨房、卫生间、阳台、门洞、窗洞、梁位、下水、地漏、燃气管、烟道、强弱电箱位置。

### 2. 平面布局
- [ ] 确认房间功能：主卧、两个儿童房、纯书房。
- [ ] 确认客厅：电视墙、沙发、餐桌、餐边柜位置。
- [ ] 确认厨房：水槽、灶具、油烟机、洗碗机、独立微波炉、独立蒸箱、独立烤箱位置。
- [ ] 确认阳台：洗衣机、烘干机、家政柜、扫地机器人基站、清洁工具位置。
- [ ] 确认两个卫生间干湿分离方式、马桶坑距、浴室柜和淋浴区尺寸。
- [ ] 确认书房放两张儿童升降桌，作为两个孩子长期学习区；儿童房书桌按临时学习和阅读使用。
- [ ] 确认全屋监控预留点位，摄像头只放入户、客餐厅公共区和阳台，不进入卧室、卫生间和书房正对书桌位置。
- [ ] 确认扫地机器人主要通行路径，重点复核阳台、卫生间门口和推拉门收口高差，尽量控制常走路径门槛高度。
- [ ] 确认卧室、儿童房、书房安静需求，室内门隔音等级、门底密封和门套收口做法提前纳入采购标准。

### 3. 开工前图纸
- [ ] 平面布置图。
- [ ] 水电点位图：插座、开关、网口、灯位、上下水、排水。
- [ ] 中央空调图：内机、外机、出风口、回风口、检修口、冷凝水。
- [ ] 综合天花图：中央空调风口、回风口、检修口、窗帘盒、灯位、吊顶收口统一出图。
- [ ] 书房新增一路地暖图：铺设区域、接入位置、分集水器、完成面高度。
- [ ] 综合完成面标高表：客餐厅、卧室、书房、阳台、卫生间门口、挡水和机器人常走路径的目标高差。
- [ ] 封窗图：固定扇、开启扇、通风位置、玻璃分格、五金和纱窗位置。
- [ ] 瓷砖排版图：全屋通铺起铺点、门洞收口、地漏切割、显眼位置避小条。
- [ ] 全屋定制初步尺寸图：橱柜、玄关柜、衣柜、阳台柜、餐边柜、书房柜。
- [ ] 厨房拆改/门洞专项图：是否扩门洞、可拆改边界、推拉门尺寸、烟道和燃气避让。
- [ ] 全屋网络和监控图：客厅吸顶 AP、各房间 86 面板 AP、摄像头网线或电源点位、弱电箱设备位置。

### 4. 水电前必须定尺寸的电器和设备
- [ ] 洗碗机：宽、高、深、进水、排水、插座位置决定橱柜开孔。
- [ ] 蒸箱：宽、高、深、功率、散热要求、插座位置决定电器高柜。
- [ ] 烤箱：宽、高、深、功率、散热要求、插座位置决定电器高柜。
- [ ] 微波炉：宽、高、深、散热空间决定开放格或台面外固定位置。
- [ ] 油烟机和灶具：烟机尺寸、排烟管直径、止逆阀口径、灶具开孔决定吊柜、烟道和台面。
- [ ] 水槽和厨房龙头：水槽尺寸、安装方式、龙头孔决定台面开孔和水槽柜结构。
- [ ] 厨下 RO 净水器：主机尺寸、滤芯更换方向、净水龙头开孔、插座和排水决定水槽柜内部布局。
- [ ] 洗衣机和烘干机：宽、高、深、叠放支架、插座、进水、排水决定阳台洗烘柜。
- [ ] 扫地机器人基站：宽、高、深、自动上下水、插座、排水和检修空间决定阳台家政柜。
- [ ] 浴霸：主机尺寸、排风管、止逆阀、吊顶开孔、开关控制决定卫生间吊顶。
- [ ] 中央空调内机：内机尺寸、检修口、风口、回风口、冷凝水和吊顶高度决定吊顶方案。
- [ ] 两张儿童升降桌：宽、深、升降范围、电源、网口、台灯和座椅活动空间决定书房布局。

### 5. 后期购买但水电前必须预留
- [ ] 冰箱：宽、高、深、散热距离、插座位置。
- [ ] 电视：尺寸范围、电视墙插座、网口、预埋管和挂架位置。
- [ ] 电视墙 50 管：电视背后到电视柜内预埋 `PVC 50管`，两端预留检修口，避开承重墙深开槽。
- [ ] 燃气报警器：电源位置、安装高度和燃气类型。
- [ ] 前置过滤器：安装位置、管径、检修空间和排污条件。
- [ ] 电动窗帘：电机位置、窗帘盒尺寸、插座和轨道长度。
- [ ] 路由器、交换机、AP 或 Mesh：弱电箱电源、散热、网线数量和安装位置。
- [ ] 全屋 WiFi：客厅预留吸顶 AP 网线和电源条件，卧室、书房、儿童房预留 86 面板 AP 网口。
- [ ] 监控摄像头：入户门内侧、客餐厅公共区、阳台预留网线或电源，点位避开隐私区域。
- [ ] 热水系统：燃气热水器容量、是否零冷水、安装位置、回水管条件、厨房和双卫同时用水能力。
- [ ] 室内门：门洞尺寸、门扇厚度、门芯结构、合页、门锁、门底密封和静音条要求。
- [ ] 电动晾衣架：如后期可能增加，需确认阳台顶部电源、承重点位和与封窗/吊顶的避让。
- [ ] 新风替代方案：既然先不做新风，需确认可持续通风路径、纱窗、限位、卫生间排风和厨房补风方式。
- [ ] 吹风机、电动牙刷、剃须刀：镜柜或台盆区插座位置。
- [ ] 电动拖把或清洁工具：阳台充电插座、收纳高度和排水条件。

### 6. 关键商家方案
- [ ] 装修公司：半包施工范围、辅材清单、工期、现场负责人和施工节点。
- [ ] 第三方监理：上门节点、报告形式和复验方式。
- [ ] 封窗商家：窗型图、开启扇位置、吊装和施工周期。
- [ ] 中央空调商家：设备位置、吊顶高度、检修口和冷凝水方案。
- [ ] 书房新增一路地暖商家：接入位置、铺设范围、完成面高度。
- [ ] 全屋定制商家：橱柜、玄关柜、衣柜、阳台柜、餐边柜、书房柜初步尺寸。
- [ ] 水电商家：空调、冰箱、厨房大功率电器、卫生间喷枪、监控、AP 点位和强电回路划分。
- [ ] 热水器商家或燃气公司：热水器容量、零冷水、回水方式、排烟、燃气管改位和用气安全方案。

### 7. 半包签约前必须准备
- [ ] 装修公司提供“半包包含项目清单”：拆改、水电、防水、瓦工、吊顶、油工、安装收口、垃圾清运、成品保护。
- [ ] 装修公司提供“辅材品牌型号清单”：水管、电线、线管、底盒、网线、防水、瓷砖胶、背胶、水泥、沙子、腻子、乳胶漆、龙骨、石膏板、玻璃胶。
- [ ] 业主准备“自购主材清单”：瓷砖、封窗、中央空调、地暖、室内门、开关插座、灯具、卫浴、地漏、角阀、软管、喷枪、网络设备。
- [ ] 水电报价写清点位、米数、开槽、封槽、强弱电箱、回路、空开、漏保、网线、底盒单价和总价上限。
- [ ] 配电箱回路表和空开漏保规格提前审核，确认总开容量、回路数量和预留回路充足，不靠现场临时拼接。
- [ ] 找平、防水、瓷砖铺贴、吊顶、包管、垃圾清运、材料搬运、成品保护全部列明单价和范围。
- [ ] 合同写明漏项责任：施工方漏报、漏量、漏写造成的必要项目，由施工方承担或按合同增项上限执行。

## 最好提前准备好
### 1. 预算和报价对比
- [ ] 硬装预算拆分表：半包基础施工费 + 业主自购硬装主材和设备费。
- [ ] 完全入住预算控制表。
- [ ] 每个大项准备 `2-3家` 商家报价用于横向比较。
- [ ] 明确先入住项目和后置升级项目。

### 2. 主材和设备备选
- [ ] 封窗型材、玻璃、五金备选。
- [ ] 中央空调品牌和系列备选。
- [ ] 瓷砖品牌、型号、色号和防滑砖备选。
- [ ] 全屋定制板材、门板、封边、五金备选。
- [ ] 地漏、角阀、软管、马桶喷枪、淋浴房五金、止逆阀备选。
- [ ] 开关插座、灯具、浴霸、网络设备备选。
- [ ] 防霉玻璃胶、铝合金踢脚线、卫生间包管隔音材料备选。
- [ ] 床垫、沙发、软包床、儿童成品家具的环保和异味控制备选。

### 3. 采购里程碑
- [ ] 洗碗机、蒸箱、烤箱、微波炉在水电定位前锁型号。
- [ ] 油烟机止逆阀、浴霸、燃气报警器在吊顶封板前到场。
- [ ] 洗衣机、烘干机、扫地机器人基站在阳台柜和水电复核前锁型号。
- [ ] 马桶、浴室柜、淋浴房在卫生间铺贴前复核尺寸和安装条件。
- [ ] 室内门在贴砖完成面和门洞尺寸稳定后复尺下单。
- [ ] 窗帘电机在窗帘盒尺寸确认前锁型号。

### 4. 现场记录
- [ ] 全屋原始照片和视频。
- [ ] 水电、燃气、烟道、地漏、下水、梁位、门窗洞口照片。
- [ ] 物业规定、商家方案、复尺尺寸统一保存。
- [ ] 每个房间单独建立照片和尺寸记录。
- [ ] 板材、瓷砖、涂料、胶粘剂、五金、卫浴和门窗到货时，拍包装标签、批次、防伪码、合格证和检测报告。

### 5. 施工准备
- [ ] 确认开工日期、施工时间、电梯预约、材料进场路线。
- [ ] 准备临时钥匙、门禁、施工用水电和现场保护要求。
- [ ] 提前和邻居、物业沟通噪音施工时间。
- [ ] 准备卷尺、记号笔、文件夹、标签纸、插座点位贴纸。
- [ ] 确认施工期间家人不在现场居住。

### 6. 可后期定稿
- [ ] 乳胶漆色号。
- [ ] 灯具款式。
- [ ] 美缝颜色。
- [ ] 窗帘布料。
- [ ] 活动家具、床垫、沙发、餐桌椅。
